# Резервное копирование и восстановление данных мониторинга VictoriaMetrics

## Обзор функций

Функция резервного копирования и восстановления данных мониторинга VictoriaMetrics позволяет выполнять регулярное резервное копирование данных мониторинга и восстанавливать данные при необходимости, что обеспечивает безопасность и доступность данных мониторинга.

## Примеры использования

- Регулярное резервное копирование данных мониторинга для предотвращения потери данных
- Миграция данных во время миграции системы
- Восстановление после катастроф
- Восстановление данных тестовой среды

## Необходимые условия

- Плагин ACP Monitoring с VictoriaMetrics установлен в кластере
- Убедитесь, что есть достаточное дисковое пространство для резервных копий
- Иметь доступ к пути хранения VictoriaMetrics

## Процедуры

### 1. Подтвердите путь хранения

Данные мониторинга VictoriaMetrics хранятся в указанном пути контейнера, который указывается параметром `-storageDataPath`, по умолчанию равным `/vm-data`.

Пример конфигурации:

```yaml
spec:
  template:
    spec:
      containers:
        - args:
            - '-storageDataPath=/vm-data'
```

**Примечание**: Имя вычислительного компонента в плагине ACP Monitoring с VictoriaMetrics — `vmstorage-cluster`, а его тип — `StatefulSet`.

### 2. Выполните резервное копирование данных

Используйте инструмент vmbackup для выполнения резервного копирования данных; пожалуйста, обратитесь к [официальной документации vmbackup](https://docs.victoriametrics.com/vmbackup.html) для получения подробных инструкций.

### 3. Выполните восстановление данных

Используйте инструмент vmrestore для восстановления данных резервной копии; пожалуйста, обратитесь к [официальной документации vmrestore](https://docs.victoriametrics.com/vmrestore.html) для получения подробных инструкций.

## Результат операции

После завершения резервного копирования вы получите полный файл резервной копии данных мониторинга. После выполнения операции восстановления ваши данные мониторинга будут восстановлены в состояние на момент резервного копирования.
