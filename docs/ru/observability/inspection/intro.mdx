# Введение

## Введение в модуль

Чтобы помочь предприятиям сократить затраты на ручные проверки, базовая функция проверки на платформе разработана на основе опыта выполнения ручных проверок для корпоративных клиентов. Это позволяет корпоративным клиентам в реальном времени понимать статус работы всех бизнес-ресурсов на платформе, своевременно выявлять аномалии и снижать бизнес-риски.

- Поддерживает онлайн-выполнение задач инспекции, включая проверки рисков ресурсов всех кластеров, узлов, подов и сертификатов на платформе, а также проверки использования обычных ресурсов, позволяя отслеживать прогресс проверки в реальном времени;
- По завершении проверки результаты визуально отображаются, включая информацию о рисках ресурсов и использовании;
- Поддерживает возможность скачивания отчетов о проверке в формате PDF или Excel;
- Для обеспечения безопасности данных клиентов, только пользователи с соответствующими правами доступа могут использовать функционал проверки.

## Преимущества модуля

- **Всеобъемлющее покрытие**: Поддерживает проверки всех критически важных ресурсов на платформе, гарантируя, что ничего не будет упущено.
- **Мониторинг в реальном времени**: Пользователи могут просматривать прогресс проверки в реальном времени, оставаясь в курсе статуса ресурсов.
- **Визуальная презентация**: Результаты проверки отображаются через интуитивно понятный визуальный интерфейс, что облегчает быструю идентификацию проблем.
- **Гибкая отчетность**: Поддерживает возможность скачивания отчетов о проверке в форматах PDF или Excel для удовлетворения потребностей различных пользователей.

## Сценарии использования модуля

- **Ежедневные операции**: Регулярное выполнение задач проверки для обеспечения нормальной работы и безопасности платформенных ресурсов.
- **Диагностика неисправностей**: Быстрое нахождение рисков ресурсов или аномалий использования через функционал проверки при возникновении проблем.
- **Аудиты на соответствие требованиям**: Скачивание отчетов о проверке для аудитов на соответствие и внутренних проверок, чтобы гарантировать соответствие платформы соответствующим стандартам и регламентам.
- **Оптимизация ресурсов**: Анализ информации о использовании ресурсов для выявления избыточности или нехватки ресурсов и оптимизация распределения ресурсов.

## Ограничения использования

- Некоторые позиции проверки на платформе зависят от установки компонентов мониторинга на кластерах. Пожалуйста, убедитесь, что каждый кластер имеет либо установленный плагин ACP Monitoring с Prometheus, либо установленный плагин ACP Monitoring с VictoriaMetrics.
- Платформенная проверка поддерживает отправку результатов проверки по электронным сообщениям. Пожалуйста, убедитесь, что конфигурация сервера уведомлений по электронной почте завершена заранее.

С помощью функционала проверки контейнерной платформы пользователи могут более эффективно управлять и поддерживать контейнерную среду, повышая стабильность и безопасность системы.
