# Статус здоровья компонентов

Страница состояния здоровья платформы предоставляет статистические данные о состоянии здоровья функций, установленных на платформе. Когда у вашей учетной записи есть разрешения на управление или аудит, связанные с платформой, вы также можете просмотреть подробные данные о здоровье конкретных функций, включая: список кластеров, в которых функция не установлена, статус здоровья кластеров с установленной функцией, а также данные обнаружения для компонентов в кластерах, связанных с функцией. Это поможет вам быстро выявить проблемы и повысить операционную эффективность платформы.

## Процедуры выполнения

1. Перейдите на страницу просмотра установленных продуктов или в центр платформы (управление платформой, управление проектами, операционный центр).

2. Нажмите кнопку вопросительного знака в правом верхнем углу навигационной панели > **Статус здоровья платформы**.

3. Проверьте карточку функции; карточка функции отображает информацию о состоянии здоровья функции. Если в компонентах функции есть аномалии, это будет отражено на карточке как `fault`.

4. Нажмите на значение здоровья/неисправности на карточке функции, чтобы развернуть страницу подробного состояния здоровья на правой стороне страницы, где вы можете просмотреть подробную информацию о проблемах с неисправными компонентами.
