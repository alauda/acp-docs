# Статус состояния компонентов

Страница состояния здоровья платформы представляет собой статистические данные о состоянии здоровья функций, которые были установлены на платформе. Когда у вашей учетной записи есть права управления или аудита, связанные с платформой, вы также можете просмотреть подробные данные о состоянии здоровья для конкретных функций, включая: список кластеров, в которых функция не установлена, статус здоровья кластеров с установленной функцией и данные о компонентах в кластерах, связанных с функцией. Это поможет вам быстро выявлять проблемы и повышать операционную эффективность платформы.

## Процедуры для работы

1. Перейдите на страницу просмотра установленных продуктов или в центр платформы (управление платформой, управление проектом, операционный центр).

2. Нажмите кнопку вопросительного знака в верхнем правом углу навигационной панели > **Статус состояния платформы**.

3. Проверьте карточку функции; карточка функции отображает информацию о состоянии здоровья функции. Если есть отклонения в компонентах функции, это будет отражено на карточке как `неисправность`.

4. Нажмите на значение здоровья/неисправности на карточке функции, чтобы развернуть подробную страницу состояния здоровья с правой стороны страницы, где вы можете просмотреть подробную информацию о проблемах с неисправными компонентами.
