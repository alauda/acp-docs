# Интеграция репозиториев кода через панель управления Argo CD

Используйте встроенную панель управления Argo CD для интеграции репозиториев кода и назначения репозиториев, позволяя разработчикам управлять приложениями GitOps на протяжении всего их жизненного цикла через визуальный интерфейс.

## Сценарии использования

- Упрощение процесса создания приложений **GitOps**, выбирая связанный репозиторий через веб-консоль во время их создания.
- При создании **Приложения** через встроенную панель управления Argo CD вы можете выбрать использование связанного репозитория.

## Предварительные требования

- [Установка ACP GitOps](../install_gitops/install_gitops_plugin.mdx) и переключатель **Встроенного UI Argo CD** должен быть включен.
- Доступ к URL-адресу **Встроенного UI Argo CD** вместе с именем пользователя и паролем.
  - Администраторы могут напрямую получить доступ к URL через страницу деталей плагина GitOps кластера.

## Процедура

Следуйте этим шагам, чтобы использовать возможности:

1. Подключить репозиторий кода

   - Войдите в **Argo CD**, используя URL доступа.

   - Щелкните на **Настройки** в левой панели навигации.

   - Щелкните на карточку **РЕПО**.

   - Нажмите **ПОДКЛЮЧИТЬ РЕПО** в верхнем левом углу страницы.

   - Выберите способ подключения репозитория и заполните соответствующие параметры по мере необходимости.

   - Нажмите **ПОДКЛЮЧИТЬ**.

2. Ассоциировать проект

   - Щелкните на **Настройки** в левой панели навигации.

   - Щелкните на карточку **Проекты**.

   - Нажмите на проект, в котором необходимо создать приложение GitOps.

   Примечание: Argo CD автоматически синхронизирует проекты в кластере, поэтому нет необходимости создавать их вручную.

   - Нажмите **ИЗМЕНИТЬ** в разделе **ИСТОЧНИКИ РЕПОЗИТОРИЕВ**.

   - Нажмите **ДОБАВИТЬ ИСТОЧНИК**, введите URL репозитория из шага **Подключить репозиторий** и ассоциируйте его с проектом.

   - Нажмите **СОХРАНИТЬ**.

## Результат операции

- Вернитесь к веб-консоли и перейдите в **Container Platform** > **GitOps Applications**. На странице **Создать** вы увидите связанные репозитории.
