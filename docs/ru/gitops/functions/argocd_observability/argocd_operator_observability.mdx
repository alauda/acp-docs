# Мониторинг компонентов Argo CD

## Обзор

Мониторинговая панель веб-консоли предлагает визуальный подход к мониторингу компонентов Argo CD. Она проактивно наблюдает за ресурсами и эксплуатационными состояниями компонентов Argo CD, с целью обеспечения их здоровья и доступности. Здесь под эксплуатационными состояниями подразумеваются условия работы и показатели производительности компонентов в окружении Kubernetes (K8s). Тщательно отслеживая эти аспекты, мы можем быстро обнаруживать и устранять любые потенциальные проблемы, поддерживая бесперебойную работу Argo CD в кластере K8s.

## Предварительные требования

- [Установка ACP GitOps](/gitops/install_gitops/install_gitops_plugin.mdx)
- [Установка плагинов мониторинга](/observability/monitor/install_monitor.mdx)

## Просмотр панели мониторинга компонентов Argo CD

1. Войдите в систему и перейдите в раздел **Управление платформой**, выберите кластер `global`.

2. Нажмите на **Центр операций** > **Мониторинг** > **Панель мониторинга**.

3. Нажмите кнопку **Переключить** и выберите **контейнерная платформа**, чтобы просмотреть панель инструментов **ArgoCD**.

4. Нажмите на панель инструментов **ArgoCD**, чтобы просмотреть информацию о мониторинге компонентов **Argo CD**.
