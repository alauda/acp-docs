# Установка ACP GitOps

## Предварительные условия

1. **Скачайте** установочный пакет плагина кластера **ACP GitOps**, соответствующий архитектуре вашей платформы.
2. **Загрузите** установочный пакет **ACP GitOps** с помощью механизма Загрузки пакетов.
3. **Установите** плагин кластера **ACP GitOps** в кластер `global` с использованием механизма плагинов кластера.

:::info
Загрузка пакетов:
**Управление платформой** > **Маркетплейс** > страница **Загрузка пакетов**.
Нажмите **Документация**, чтобы получить инструкции о том, как опубликовать плагин кластера в кластере `global`. Для получения дополнительной информации, пожалуйста, ознакомьтесь с [CLI](/ui/cli_tools/index.mdx).
:::

## Установка плагина кластера ACP GitOps

### Ограничения и ограничения

- Поддерживает установку только в кластере `global`.
- После установки плагина экземпляр ArgoCD в `argocd-operator` будет ограничен в операциях.

### Процедура

1. Войдите в систему, перейдите на страницу **Управление платформой**.
2. Нажмите **Маркетплейс** > **Плагины кластера**, чтобы перейти на страницу списка **Плагинов кластера**.
3. Найдите плагин кластера **GitOps**, нажмите **Установить**, и перейдите на страницу **Установка плагина GitOps**.
4. Рекомендуется использовать предложенную конфигурацию по умолчанию; просто нажмите **Установить**, чтобы завершить установку плагина кластера **ACP GitOps**.

Описание параметров приведены ниже:

Этот интерфейс включает в себя такие функции, как мониторинг, управление репозиториями и настройки, и может использоваться для управления и мониторинга созданных приложений.                                                                                                                                                                                                                                                                                                                                                                 |

| **Адрес доступа**    | Рекомендуется: Этот адрес динамически генерируется на основе адреса платформы для доступа к панели инструментов Argo CD. Ручной ввод не требуется.                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Учетная запись**           | Учетная запись, используемая для входа и доступа к нативному интерфейсу Argo CD.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **Пароль**          | После активации доступа к нативному интерфейсу Argo CD вы можете выполнить следующую команду в инструменте CLI глобального кластера, чтобы получить его. <br />[Получить информацию о доступе к Argo CD](../how_to/argocd_info.mdx#argocd-password-cli)                                                                                                                                                                                                                                                                                                                                                                |

### Проверка

1. На странице **Управление платформой**, в разделе **Кластер** в левом меню отобразится запись **Конфигурация**. Вы можете использовать возможности управления конфигурацией кластера.
2. Перейдите на **Контейнерную платформу**, в левом меню будет отображаться запись **GitOps Applications**, где вы можете создать приложение GitOps, чтобы сразу испробовать [Создание приложения Argo CD через веб-консоль](../how_to/create_gitops_app.mdx).
