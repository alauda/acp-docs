# Введение

ACP GitOps — это решение GitOps, основанное на Kubernetes и построенное на Argo CD. Оно следит за манифестами конфигурации (приложениями, определениями инфраструктуры и т.д.) в репозиториях Git и автоматически синхронизирует их с целевыми кластерами Kubernetes, реализуя непрерывную доставку на основе Git. Кодируя весь процесс доставки в системе контроля версий Git, оно повышает скорость развертывания, надежность и безопасность, одновременно обеспечивая точное распределение приложений по нескольким кластерам.

Решение нативно интегрирует оператор Argo CD для автоматизации операций жизненного цикла развертывания, включая Provisioning, обновления и откаты.

## Почему Argo CD?

Argo CD является лидирующим решением GitOps с открытым исходным кодом в индустрии благодаря своим уникальным преимуществам:

| **Технические преимущества**                                                                                                                                                             | **Операционные выгоды**                                                                                      |
| :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------ |
| **Декларативный GitOps-движок** <ul><li>Согласование состояния через CRD (Application, ApplicationSet)</li><li>Поддержка нескольких источников (Helm, Kustomize, сырые YAML)</li></ul> | **Ускоренное развертывание** <ul><li>Развертывание на 70% быстрее благодаря автоматизации на основе Git</li></ul> |
| **Архитектура, оптимизированная для Kubernetes** <ul><li>Глубокая интеграция с API-сервером Kubernetes ≈ Нативная поддержка изоляции пространств имен и RBAC</li></ul>                    | **Готовность к корпоративным требованиям** <ul><li>Встроенные многоарендные модели и возможности аудита</li></ul>       |
| **Управление многими кластерами** <ul><li>Централизованная контрольная панель для гибридных/мультиоблачных развертываний</li><li>Конфигурация, специфичная для кластеров, через ApplicationSets</li></ul> | **Операционная эффективность** <ul><li>Снижение ошибок развертывания на 60% благодаря декларативному соблюдению</li></ul> |

## Преимущества <a id="alauda-gitops-advantage" />

> Помимо присущих преимуществ GitOps, ACP GitOps предлагает следующие расширенные выгоды:

- **Оператор Argo CD корпоративного уровня**

  - Предоставляет полный набор функциональных возможностей нативного оператора Argo CD, охватывающий развертывание приложений, обновления, откаты и все основные функции Argo CD.

- **Служба безопасности оператора Argo CD**

  - Предлагает специализированную техническую поддержку для оператора Argo CD, включая реагирование на неполадки, патчи для уязвимостей в безопасности и общую стабильность системы.

- **Управление многоокруженческой дистрибуцией визуальных приложений GitOps**

  - Использует возможности управления несколькими кластерами платформы и дифференцированных конфигураций для достижения стильной, визуальной GitOps-управляемости приложениями и управления конфигурацией кластеров, упрощая точное распределение по мультиоблачным и многоокруженческим настройкам.

- **Управление операциями и обслуживанием визуальных приложений GitOps**

  - Предоставляет доступ к реальным логам и событиям ресурсов нагрузки Kubernetes в рамках приложений GitOps. В случае аномалий в приложениях GitOps пользователи могут быстро анализировать и решать проблемы, используя информацию о аномалиях Argo CD и реальные логи нагрузки, не покидая текущий интерфейс.

- **Управление конфигурацией кластера визуальных приложений GitOps**

  - Управляет конфигурациями кластеров с помощью GitOps, достигая единого управления и распределения конфигураций кластеров в визуальном формате.

- **Замкнутое управление приложениями GitOps, интегрированное со всеми продуктами платформы**

  - Интегрирует возможности DevOps платформы для непрерывной сборки, управления артефактами и серого развертывания микросервисов, чтобы реализовать полностью автоматизированное управление приложениями GitOps, формируя замкнутый цикл непрерывной интеграции и сотрудничества по непрерывной доставке.
  - Сочетает с продуктами платформы, такими как CrossPlane, MySQL и Портал для разработчиков, чтобы достичь комплексного процесса, охватывающего инициализацию инфраструктуры, инициализацию бизнес-приложений (включая код, конвейер, инициализацию приложений GitOps и т.д.) и разработку и развертывание кода приложений.
