# Tool

## Введение

**Tool** — это утилита, используемая для генерации или обработки ресурсов Kubernetes `Manifests`.

## Поддерживаемые инструменты

Argo CD поддерживает несколько подходов к определению манифестов Kubernetes:

- **Kustomize Applications** [Kustomize](./concept_argocd_tool_kustomize.mdx)
- **Helm Charts** [Helm](./concept_argocd_tool_helm.mdx)
- **Каталог**: Манифесты, содержащие файлы `YAML`/`JSON`/`Jsonnet`, включая `Jsonnet` [Directory](./concept_argocd_tool_directory.mdx)
- **Плагины управления конфигурацией настраиваемого типа**: Любой настраиваемый инструмент, сконфигурированный в качестве плагина управления конфигурацией

## Рабочий процесс разработки

Argo CD позволяет напрямую загружать локальные `manifests`, но это предназначено **только для целей разработки**. Для переопределения требуется наличие у пользователей разрешений (обычно администраторов) для загрузки локальных `manifests`. Поддерживаются все вышеупомянутые инструменты развертывания Kubernetes. Чтобы загрузить локальное приложение:

```bash
$ argocd app sync APPNAME --local /path/to/dir/
```

## Сравнение функций

| **Функция**                  | **Helm**                                      | **Kustomize**                      | **Каталог (чистый YAML)**        |
| :--------------------------- | :-------------------------------------------- | :--------------------------------- | :-------------------------------- |
| **Метод конфигурации**       | Шаблонирование (динамическая генерация)      | Декларативное (патчи и наложения) | Статические YAML файлы           |
| **Повторное использование**   | Высокое (через Charts)                        | Среднее (через базу/наложение)    | Низкое                            |
| **Поддержка множественных окружений** | Высокая (через values.yaml)           | Высокая (через наложения)         | Низкая                            |
| **Постепенная доставка**     | Высокая (поддержка сложной логики)           | Средняя (поддержка простых патчей) | Низкая                            |
| **Кривая обучения**          | Высокая (синтаксис шаблонов)                  | Низкая (на основе YAML)           | Низкая                            |
| **Интеграция с Argo CD**    | Поддерживается                                | Нативная поддержка                | Поддерживается                    |
| **Сценарии использования**   | Сложные приложения, множественные окружения, распределение | Множественные окружения, повторное использование конфигурации | Малые проекты, быстрое прототипирование |
