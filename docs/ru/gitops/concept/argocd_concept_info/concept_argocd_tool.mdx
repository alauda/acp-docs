# Инструмент

## Введение

**Инструмент** относится к утилите, используемой для генерации или обработки `Manifests` ресурсов Kubernetes.

## Поддерживаемые инструменты

Argo CD поддерживает несколько подходов к определению манифестов Kubernetes:

- **Kustomize Applications** [Kustomize](./concept_argocd_tool_kustomize.mdx)
- **Helm Charts** [Helm](./concept_argocd_tool_helm.mdx)
- **Каталог**: Манифесты, содержащие файлы `YAML`/`JSON`/`Jsonnet`, включая `Jsonnet` [Directory](./concept_argocd_tool_directory.mdx)
- **Плагины управления конфигурацией**: Любой пользовательский инструмент, настроенный в качестве плагина управления конфигурацией

## Рабочий процесс разработки

Argo CD позволяет напрямую загружать локальные `manifests`, но это предназначено **только для целей разработки**. Переопределение требует наличия у пользователей прав (обычно администраторов) для загрузки локальных `manifests`. Он поддерживает все вышеупомянутые инструменты развертывания Kubernetes. Для загрузки локального приложения:

```bash
$ argocd app sync APPNAME --local /path/to/dir/
```

## Сравнение функций

| **Функция**                  | **Helm**                                      | **Kustomize**                      | **Каталог (чистый YAML)**       |
| :--------------------------- | :-------------------------------------------- | :--------------------------------- | :------------------------------- |
| **Метод конфигурации**       | Шаблонирование (динамическая генерация)      | Декларативный (патчи и оверлеи)  | Статические файлы YAML           |
| **Повторное использование**   | Высокое (через Charts)                        | Среднее (через базу/оверлей)      | Низкое                           |
| **Поддержка нескольких сред** | Высокая (через values.yaml)                   | Высокая (через оверлеи)           | Низкое                           |
| **Постепенная доставка**      | Высокая (поддержка сложной логики)            | Средняя (поддержка простых патчей) | Низкое                           |
| **Кривое обучение**           | Высокое (синтаксис шаблонов)                  | Низкое (на основе YAML)           | Низкое                           |
| **Интеграция с Argo CD**      | Поддерживается                                | Нативная поддержка                | Поддерживается                   |
| **Сценарии использования**    | Сложные приложения, многосредовая, распределение | Многосредовая, повторное использование конфигураций | Малые проекты, быстрое прототипирование |
