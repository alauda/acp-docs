# Введение

Платформа контейнеров Alauda (<Term name="productShort" />) — это решение уровня предприятия, основанное на облачных технологиях, разработанное для революции цифровой трансформации. Построенная на прочной основе Kubernetes, <Term name="productShort" /> предоставляет организациям возможность справляться с сложностями современной ИТ-инфраструктуры с беспрецедентной гибкостью, безопасностью и эффективностью.

## Основное ценностное предложение

<Term name="productShort" /> устраняет разрыв между инновационными технологиями и операционной эффективностью, предоставляя предприятиям комплексную платформу, которая упрощает управление контейнеризованными приложениями в разнообразных и динамичных средах.

### 3. Повышенная продуктивность разработчиков

- Архитектура многоарендной платформы
- Поддержка интегрированного инструментария DevOps

### 4. Гибкие операции инфраструктуры

- Единая управляющая плоскость для различных сред
