# О платформе Alauda DevOps

DevOps — это усовершенствованная платформa облачных приложений на основе контейнеров, которая позволяет осуществлять единое планирование контейнерных рабочих нагрузок с тонкой настройкой многоарендных разрешений. Созданная на базе нативной архитектуры Kubernetes, она безупречно управляет цепочкой инструментов DevOps, предлагая гибкую совместимость с существующими инструментами и учитывая потенциальные изменения инструментов в будущем.
