# О платформе Alauda DevOps

DevOps является сложной контейнерной облачной платформой приложений, которая позволяет объединенно планировать контейнерные рабочие нагрузки с детальными многоарендными разрешениями. Построенная на базе родной архитектуры Kubernetes, она безшовно управляет цепочкой инструментов DevOps, предлагая гибкую совместимость с вашими существующими инструментами, а также учитывая возможные изменения инструментов в будущем.
