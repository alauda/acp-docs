# Взаимосвязь с возможностями платформы

Оповещение является одной из основных возможностей платформы, которая предлагает широкий спектр функций оповещения для шести категорий ресурсов, включая Сервис Приложений. Центр оповещений Сервиса Приложений является подмножеством возможностей оповещения платформы. Для получения подробной информации о создании стратегий оповещения и настройке тихих оповещений, пожалуйста, обратитесь к [Управлению Оповещениями](/observability/monitor/functions/manage_alert.mdx).

::: info

Из-за соображений разрешений Управление Оповещениями в представлении Сервисов Данных и Управление Оповещениями в представлении Управления Платформой изолированы друг от друга на уровне данных. Это означает, что если пользователь создает политику оповещения в представлении Сервисов Данных, управление этой политикой (например, обновление, просмотр и удаление) должно по-прежнему выполняться в представлении Сервисов Данных и наоборот.

:::
