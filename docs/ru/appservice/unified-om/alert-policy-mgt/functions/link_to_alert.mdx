# Отношения с возможностями платформы

Управление уведомлениями является одной из основных возможностей платформы, которое предлагает широкий спектр функций по отправке уведомлений для шести категорий ресурсов, включая сервис приложений. Центр уведомлений сервиса приложений является подмножеством возможностей уведомлений платформы. Для получения подробной информации о создании стратегий уведомлений и настройке тихих уведомлений, пожалуйста, обратитесь к [Управление уведомлениями](/observability/monitor/functions/manage_alert.mdx).

::: info

С учетом вопросов разрешений, управление уведомлениями в режиме сервиса приложений и управление уведомлениями в режиме управления платформой изолированы друг от друга на уровне данных. Это означает, что если пользователь создает политику уведомлений в режиме сервиса приложений, управление этой политикой (например, обновление, просмотр и удаление) по-прежнему должно выполняться в режиме сервиса приложений и наоборот.

:::
