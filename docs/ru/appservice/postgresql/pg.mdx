Поддержка PostgreSQL в Alauda представляет собой облачное решение для управления PostgreSQL, основанное на операторе PostgreSQL от Zalando. Оно обеспечивает полное управление жизненным циклом кластеров PostgreSQL через определения пользовательских ресурсов (CRD), поддерживая основные операционные возможности, такие как автоматическое переключение на резервный сервер, поэтапные обновления и восстановление из резервной копии, позволяя предприятиям без усилий запускать базы данных PostgreSQL производственного уровня на Kubernetes.
