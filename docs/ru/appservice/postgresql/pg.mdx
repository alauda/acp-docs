Поддержка PostgreSQL в Alauda представляет собой облачное решение для управления PostgreSQL, построенное на основе Zalando Postgres Operator. Оно достигает полного управления жизненным циклом кластеров PostgreSQL через определения пользовательских ресурсов (CRD), поддерживая основные операционные возможности, такие как автоматический отказ, поэтапные обновления и восстановление резервных копий, что позволяет предприятиям без усилий запускать базы данных PostgreSQL производственного уровня на Kubernetes.
