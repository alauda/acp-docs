Alauda Database Service для MySQL-MGR - это решение для высокодоступной базы данных, основанное на технологии группового репликации MySQL, работающей на Kubernetes. Оно предоставляет возможность автоматизированного развертывания и управления кластерами групповой репликации MySQL.
