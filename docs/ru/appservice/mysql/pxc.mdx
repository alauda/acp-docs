Alauda Database Service для MySQL-PXC — это решение для базы данных с высокой доступностью, использующее синхронную репликацию на основе Percona XtraDB Cluster, работающего на Kubernetes. Оно предоставляет возможность автоматического развертывания и управления объектами внутри Percona XtraDB Cluster, а также может использоваться для запуска новых экземпляров Percona XtraDB Cluster или масштабирования существующих экземпляров.
