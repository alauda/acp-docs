Служба базы данных Alauda для MySQL-PXC — это решение для высокодоступной базы данных, использующее синхронную репликацию на основе Percona XtraDB Cluster и работающая в Kubernetes. Она предоставляет возможность автоматического развертывания и управления объектами в рамках Percona XtraDB Cluster, а также может быть использована для запуска новых экземпляров Percona XtraDB Cluster или масштабирования существующих экземпляров.
