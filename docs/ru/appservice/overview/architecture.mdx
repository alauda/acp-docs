# Архитектура

![](../../../en/appservice/overview/assets/appservice_architecture_en.png)

Платформа Alauda Application Services представляет собой мощное решение для управления сервисами с открытым исходным кодом, разработанное на основе Kubernetes (K8S), которое идеально подходит для бесперебойных операций Alauda Application Services. Она служит централизованным хабом, эффективно оркеструющим разнообразные популярные компоненты сервиса, включая MySQL, Redis, Kafka и RabbitMQ. Вот как это работает:

- Компоненты сервиса: Платформа поддерживает ключевые сервисы, такие как MySQL, Redis, Kafka и RabbitMQ, обеспечивая стабильную и надежную работу всех компонентов.

- Общие службы: Она предоставляет шаблоны параметров, инспекцию, управление оповещениями, услуги резервного копирования и т. д., предлагая пользователям комплексное решение для управления сервисами.

- Контроллер оператора: Контроллер оператора является расширением Kubernetes, предназначенным для упрощения развертывания и управления приложениями. Он расширяет API Kubernetes через определения пользовательских ресурсов (CRD), чтобы управлять жизненным циклом приложений.

- Контейнерная инфраструктура: Платформа Alauda Application Services построена на Kubernetes и использует основную инфраструктуру, которую он предоставляет, чтобы предложить основные возможности, такие как постоянство данных и внешний доступ.
