# Архитектура

![](../../../en/appservice/overview/assets/appservice_architecture_en.png)

Платформа Alauda Application Services представляет собой мощное решение для управления сервисами с открытым исходным кодом, построенное на Kubernetes (K8S), предназначенное для бесшовной работы Alauda Application Services. Она служит централизованным узлом, эффективно оркестрируя разнообразные популярные компоненты сервисов, включая MySQL, Redis, Kafka и RabbitMQ. Вот как это работает:

- Сервисные компоненты: Платформа поддерживает ключевые сервисы, такие как MySQL, Redis, Kafka и RabbitMQ, обеспечивая стабильную и надежную работу всех компонентов.

- Общие сервисы: Она предоставляет шаблоны параметров, инспекцию, управление оповещениями, услуги резервного копирования и т.д., предлагая пользователям универсальное решение для управления сервисами.

- Операторский контроллер: Операторский контроллер является расширением Kubernetes, предназначенным для упрощения развертывания и управления приложениями. Он расширяет API Kubernetes через определение пользовательских ресурсов (CRD) для управления жизненным циклом приложений.

- Инфраструктура контейнеров: Платформа Alauda Application Services построена на Kubernetes и использует основную инфраструктуру, которую он предоставляет, для обеспечения основных возможностей, таких как сохранение данных и внешний доступ.
