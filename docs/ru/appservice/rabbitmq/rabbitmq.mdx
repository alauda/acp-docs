Alauda Streaming Service для RabbitMQ — это открытый брокер сообщений, который реализует Протокол расширенной очереди сообщений (AMQP) и работает на Kubernetes. Он служит коммуникационным мостом между приложениями, позволяя различным системам асинхронно отправлять и получать сообщения. Он предоставляет надежную основу для построения разъединенных, масштабируемых и устойчивых распределенных систем.
