# Обновление

Основы Alauda Application Services предназначены для автоматического обновления во время обновлений платформы, не требуя ручного вмешательства. Система бесшовно обрабатывает все необходимые миграции и обновления в фоновом режиме.

Рамки Alauda Application Services RDS будут выполнять обновления на основе настроенной стратегии обновления:

- **Автоматически**: Автообновления запускаются немедленно при обнаружении новых версий компонентов.
- **Вручную**: Требуется ручное одобрение перед началом процесса обновления.

:::warning
При обновлении ACP с версии v3 до v4 рекомендуется немедленно обновить как глобальный кластер, так и рабочие кластеры до новой версии, чтобы обеспечить правильное отображение представления Alauda Application Service.\
Если только глобальный кластер обновлен до v4, в то время как рабочие кластеры остаются на v3, пожалуйста, свяжитесь с технической командой в этот период, чтобы получить шаги по устранению проблем с отображением представления в не обновленных рабочих кластерах.
:::
