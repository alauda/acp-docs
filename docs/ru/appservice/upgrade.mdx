# Обновление

Основные услуги данных платформы Alauda Container Platform предназначены для автоматического обновления во время обновлений платформы, не требуя ручного вмешательства. Система обрабатывает все необходимые миграции и обновления бесшовно в фоновом режиме.

Фреймворк RDS платформы Alauda Container Platform будет выполнять обновления на основе настроенной стратегии обновления:

- **Автоматическое**: Автообновления запускаются немедленно при обнаружении новых версий компонентов.
- **Ручное**: Требуется ручное одобрение перед началом процесса обновления.

:::warning
При обновлении ACP с версии v3 до v4 рекомендуется незамедлительно обновить как глобальный кластер, так и кластеры рабочих нагрузок до новой версии, чтобы обеспечить правильное отображение представления услуг данных.\
Если только глобальный кластер будет обновлен до v4, в то время как кластеры рабочих нагрузок останутся на v3, пожалуйста, свяжитесь с технической командой в этот период, чтобы получить шаги по смягчению проблем с отображением представления в не обновленных кластерах рабочих нагрузок.
:::
