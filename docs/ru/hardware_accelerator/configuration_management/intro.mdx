# Введение

## Введение в управление конфигурацией

Управление конфигурацией — это централизованный портал документации для настройки возможностей ускорения GPU в средах kubernetes. Этот живой документ предоставляет администраторам единые рекомендации по настройке физических GPU (pGPU), виртуальных GPU (vGPU) и конфигураций Multi-Process Service (MPS) в рамках гибридной инфраструктуры.
