# Введение

## Введение в управление конфигурацией

Управление конфигурацией — это централизованный портал документации для настройки возможностей ускорения GPU в средах ACP. Этот актуализируемый документ предоставляет администраторам единые рекомендации по настройке физических графических процессоров (pGPU), виртуальных графических процессоров (vGPU) и конфигураций службы многопроцессного обслуживания (MPS) в рамках гибридной инфраструктуры.
