# Введение

## Введение в управление конфигурацией

Управление конфигурацией – это централизованный портал документации для настройки возможностей ускорения GPU в средах ACP. Этот живой документ предоставляет администраторам единую информацию по настройке физических GPU (pGPU), виртуальных GPU (vGPU) и конфигураций Мультипроцессорного Сервиса (MPS) в гибридной инфраструктуре.
