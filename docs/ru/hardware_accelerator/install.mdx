# Установка

## Установка инструментария аппаратного ускорителя Kubernetes

### Предварительные условия

Перед установкой убедитесь, что выполнены следующие требования:

1. **Кластер Kubernetes**: Версия ≥1.25 с включенной функцией `DevicePlugins`.
2. **Драйверы NVIDIA**: Установлены на всех узлах с GPU. Проверьте с помощью `nvidia-smi`.
3. **Контейнерная среда выполнения**: Настроена с NVIDIA Container Toolkit (≥1.7.0) для поддержки GPU.

---

### Установка через веб-консоль

1. **Перейдите к плагинам кластера**:
   - Перейдите в **Управление платформой → Каталог → Плагин кластера**
   - Найдите "gpu" и нажмите **Установить**

2. **Переключатели функций**:
   Включите/отключите расширенные возможности во время установки:
   | Опция    | Функциональность                                   | Рекомендуемый сценарий                |
   | -------- | -------------------------------------------------- | ------------------------------------- |
   | **PGPU** | Физическая изоляция GPU с учетом NUMA             | Обучение AI/высокопроизводительные вычисления |
   | **vGPU** | Виртуализация GPU через GPU-менеджер              | Совместное использование в многопользовательской среде/квоты ресурсов |
   | **MPS**  | Мультипроцессорная служба для совместного использования вычислений/памяти | Инференс с низкой задержкой/параллельные задачи |
