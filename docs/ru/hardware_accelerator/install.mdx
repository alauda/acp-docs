# Установка

## Установка набора инструментов аппаратного ускорителя Kubernetes

### Предварительные требования

Перед установкой убедитесь, что выполнены следующие требования:

1. **Kubernetes-кластер**: версия ≥1.25 с включенной функцией `DevicePlugins`.
2. **Драйверы NVIDIA**: установлены на всех узлах с GPU. Проверьте с помощью `nvidia-smi`.
3. **Контейнерная среда выполнения**: настроена с NVIDIA Container Toolkit (≥1.7.0) для поддержки GPU.

---

### Установка через веб-консоль

1. **Перейдите к плагинам кластера**:
   - Перейдите в раздел **Управление платформой → Каталог → Плагин кластера**
   - Найдите "gpu" и нажмите **Установить**

2. **Переключатели функций**:
   Включите/выключите расширенные возможности во время установки:
   | Опция    | Функциональность                                  | Рекомендуемый сценарий                |
   | -------- | ------------------------------------------------- | -------------------------------------- |
   | **PGPU** | Физическая изоляция GPU с учетом NUMA            | Обучение ИИ/высокопроизводительные вычисления |
   | **vGPU** | Виртуальный разрез GPU через GPU-Manager          | Общение между несколькими арендаторами/квоты ресурсов   |
   | **MPS**  | Мультипроцессорная служба для совместного использования вычислений/памяти | Низкие задержки при выводе/параллельные задачи   |
