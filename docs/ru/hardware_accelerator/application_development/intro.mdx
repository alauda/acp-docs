# Введение

## Введение в разработку приложений

Модуль разработки приложений направляет пользователей на настройку аппаратных ускорителей от нескольких производителей (например, GPU от AMD/Intel, ПЛИС) через унифицированный интерфейс, что позволяет организовывать и оптимизировать гетерогенные вычислительные ресурсы в контейнеризованных средах для улучшения высокопроизводительных рабочих нагрузок, таких как обучение ИИ и обработка изображений.
