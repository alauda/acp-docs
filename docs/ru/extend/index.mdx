# Расширение

Платформа предоставляет комплексную систему расширений, которая позволяет пользователям улучшать функциональность своих кластеров Kubernetes. Эта система разработана так, чтобы быть гибкой и удобной для пользователя, что позволяет легко добавлять новые функции и возможности в кластеры.

Эта система состоит из двух основных типов расширений:

- Операторы: Операторы созданы на основе фреймворка Operator Lifecycle Manager (OLM) v0, обеспечивая специализированные операционные возможности для платформы. Эти расширения позволяют автоматически управлять сложными приложениями и службами в вашем кластере.

- Плагины кластера: Платформа предлагает собственную систему плагинов кластера, специально разработанную для плагинов типа Chart. Эта система обеспечивает улучшенное впечатление от установки и управления по сравнению с обычными методами, предлагая удобный интерфейс для работы с расширениями на основе Chart.

С поддержкой многочисленных Операторов и плагинов кластера пользователи могут значительно расширить возможности платформы для удовлетворения конкретных операционных требований и сценариев использования.
