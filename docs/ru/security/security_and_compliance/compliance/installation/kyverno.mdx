# Соответствие Alauda с Kyverno

Соответствие Alauda с Kyverno — это служба платформы, которая интегрирует Kyverno для управления политиками соответствия на платформе ACP.

## Шаги установки

1. Перейдите в **Управление платформой**
2. В левой навигационной панели щелкните **Маркетплейс** > **Плагины кластера**
3. Найдите **Соответствие Alauda с Kyverno** и нажмите, чтобы просмотреть его детали
4. Нажмите **Установить**, чтобы развернуть плагин

## Шаги деинсталляции

1. Выполните шаги 1-3 из процесса установки, чтобы найти плагин
2. Нажмите **Деинсталляция**, чтобы удалить плагин
