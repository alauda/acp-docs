# Установка

ACP API Refiner — это сервис платформы, который фильтрует данные ответов API Kubernetes. Он предоставляет возможности фильтрации по проекту, кластеру и пространству имен, а также поддерживает исключение, включение и десенсибилизацию полей в ответах API.

## Шаги установки

1. Перейдите в раздел **Управление платформой**
2. В левой боковой панели навигации нажмите **Маркетплейс** > **Плагины кластера**
3. Выберите кластер **global** в верхней панели навигации
4. Найдите **ACP API Refiner** и нажмите, чтобы просмотреть его детали
5. Нажмите **Установить**, чтобы развернуть плагин

## Шаги деинсталляции

1. Следуйте шагам 1-4 из процесса установки, чтобы найти плагин
2. Нажмите **Деинсталлировать**, чтобы удалить плагин

## Конфигурация по умолчанию

### Отфильтрованные ресурсы

Следующие ресурсы отфильтрованы по умолчанию:

| Ресурс        | Версия API                     |
| -------------- | ------------------------------- |
| пространства имен | v1                              |
| проекты       | auth.alauda.io/v1               |
| кластермодули | cluster.alauda.io/v1alpha2      |
| кластеры      | clusterregistry.k8s.io/v1alpha1 |

### Десенсибилизация полей

По умолчанию следующее поле подлежит десенсибилизации:

- `metadata.annotations.cpaas.io/creator`
