# Установка

ACP API Refiner — это сервис платформы, который фильтрует данные ответов API Kubernetes. Он предоставляет возможности фильтрации по проектам, кластерам и пространствам имен, а также поддерживает исключение, включение и десенсибилизацию полей в ответах API.

## Шаги установки

1. Перейдите в **Управление платформой**
2. В левой навигационной панели нажмите **Маркетплейс** > **Плагины кластера**
3. Выберите кластер **global** в верхней навигационной панели
4. Найдите **ACP API Refiner** и нажмите, чтобы просмотреть его детали
5. Нажмите **Установить**, чтобы развернуть плагин

## Шаги деинсталляции

1. Следуйте шагам 1-4 из процесса установки, чтобы найти плагин
2. Нажмите **Удалить**, чтобы удалить плагин

## Настройки по умолчанию

### Отфильтрованные ресурсы

Следующие ресурсы по умолчанию отфильтрованы:

| Ресурс         | Версия API                     |
| -------------- | ------------------------------- |
| пространства   | v1                              |
| проекты        | auth.alauda.io/v1             |
| кластерные модули | cluster.alauda.io/v1alpha2  |
| кластеры       | clusterregistry.k8s.io/v1alpha1 |

### Десенсибилизация полей

По умолчанию десенсибилизировано следующее поле:

- `metadata.annotations.cpaas.io/creator`
