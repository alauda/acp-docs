# Введение

Платформа предлагает всесторонние политики безопасности пользователей для повышения безопасности входа и защиты от злонамеренных атак.

## Обзор

Платформа поддерживает следующие политики безопасности:

- Управление безопасностью паролей
- Отключение учетной записи пользователя
- Блокировка учетной записи пользователя
- Уведомления для пользователей
- Контроль доступа

## Настройка политики безопасности

### Шаги

1. В левой навигационной панели нажмите **Управление ролями пользователей** > **Политика безопасности пользователей**
2. Нажмите **Обновить** в правом верхнем углу
3. Настройте политики безопасности по мере необходимости
4. Нажмите **Обновить**, чтобы сохранить изменения

:::warning

Примечания по конфигурации политики:

- Установите флажок перед политикой, чтобы включить ее
- Снимите флажок, чтобы отключить политику
- Отключенные политики сохраняют свои данные конфигурации
- Предыдущие настройки восстанавливаются при повторном включении политики

:::

## Доступные политики

| Политика                               | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| --------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Политика аутентификации пользователей** | Включает двойную аутентификацию для входа по паролю:<br />- Пользователи получают коды проверки через указанные методы уведомления<br />- Поддерживает различные уведомительные серверы (например, Сервер корпоративных коммуникационных инструментов)                                                                                                                                                                                                                                                                                                                                    |
| **Политика безопасности паролей**           | Управляет требованиями к паролям:<br /><br />Первый вход:<br />- Заставляет изменить пароль при первом входе в платформу<br /><br />Регулярные обновления:<br />- Требует изменения пароля через указанный период времени (например, 90 дней)<br />- Запрещает вход до изменения пароля                                                                                                                                                                                                                                                                           |
| **Политика отключения пользователей**     | Автоматически отключает неактивные учетные записи:<br />- Срабатывает после указанного периода без входа                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Политика блокировки пользователей**      | Защищает от атак методом перебора:<br /><br />Условия блокировки:<br />- Срабатывает после указанного количества неудачных попыток входа в течение 24 часов<br /><br />Длительность блокировки:<br />- Учетная запись остается заблокированной на указанные минуты<br />- Автоматически разблокируется после истечения периода блокировки                                                                                                                                                                                                                       |
| **Политика уведомлений**                  | Управляет уведомлениями для пользователей:<br />- Отправляет первоначальный пароль по электронной почте после создания пользователя                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Контроль доступа**                       | Управляет сессиями пользователей и доступом:<br /><br />Управление сессиями:<br />- Авторазлогивает неактивные сессии после указанного времени<br />- Ограничивает максимальное количество одновременных пользователей онлайн<br /><br />Контроль браузера:<br />- Заканчивает сессию, когда все вкладки продукта закрыты<br />- Запрещает множественные входы с одного клиента<br /><br />:::note<br /><br />Важные заметки:<br />- Контроль доступа затрагивает только новые входы после обновления политики<br />- Восстановление вкладки браузера может не привести к окончанию сессии<br />- Разрешен только последний вход с клиента при запрете повторного входа<br />::: |
