# Введение

Платформа предоставляет комплексные политики безопасности пользователя для повышения безопасности входа и защиты от вредоносных атак.

## Обзор

Платформа поддерживает следующие политики безопасности:

- Управление безопасностью паролей
- Деактивация учетных записей пользователей
- Блокировка учетных записей пользователей
- Уведомления пользователей
- Контроль доступа

## Настройка политики безопасности

### Шаги

1. В левой навигационной панели нажмите **Управление ролями пользователей** > **Политика безопасности пользователей**
2. Нажмите **Обновить** в правом верхнем углу
3. Настройте политики безопасности по мере необходимости
4. Нажмите **Обновить**, чтобы сохранить изменения

:::warning

Примечания к настройке политик:

- Установите флажок перед политикой, чтобы включить ее
- Снимите флажок, чтобы отключить политику
- Отключенные политики сохраняют свои данные конфигурации
- Предыдущие настройки восстанавливаются при повторном включении политики

:::

## Доступные политики

| Политика                       | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Политика аутентификации пользователей** | Включает двойную аутентификацию для входа с использованием пароля:<br />- Пользователи получают проверочные коды через указанные методы уведомления<br />- Поддерживает различные серверы уведомлений (например, сервер корпоративных коммуникационных инструментов)                                                                                                                                                                                                                                                                                                        |
| **Политика безопасности паролей** | Управляет требованиями к паролям:<br /><br />Первый вход:<br />- Заставляет изменить пароль при первом входе на платформу<br /><br />Регулярные обновления:<br />- Требует смены пароля после указанного периода (например, 90 дней)<br />- Предотвращает вход, пока пароль не обновлен                                                                                                                                                                                                                                                                                          |
| **Политика деактивации пользователей**   | Автоматически деактивирует неактивные учетные записи:<br />- Активируется после указанного периода без входа                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Политика блокировки пользователей**     | Защищает от атак методом подбора пароля:<br /><br />Условия блокировки:<br />- Активируется после указанного количества неудачных попыток входа в течение 24 часов<br /><br />Продолжительность блокировки:<br />- Учетная запись остается заблокированной в течение указанного времени<br />- Автоматически разблокируется по истечении периода блокировки                                                                                                                                                                     |
| **Политика уведомлений**        | Управляет уведомлениями пользователей:<br />- Отправляет начальный пароль по электронной почте после создания учетной записи пользователя                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Контроль доступа**            | Управляет сессиями пользователей и доступом:<br /><br />Управление сессиями:<br />- Автоматически выходит из неактивных сессий после указанного времени<br />- Ограничивает максимальное количество пользователей онлайн одновременно<br /><br />Контроль браузера:<br />- Завершает сессию, когда все вкладки продукта закрыты<br />- Предотвращает множественные входы с одного клиента<br /><br />:::note<br /><br />Важные заметки:<br />- Контроль доступа касается только новых входов после обновления политики<br />- Восстановление вкладок браузера может не привести к завершению сессии<br />- Разрешается только последний вход с одного клиента при предотвращении повторного входа<br />::: |
