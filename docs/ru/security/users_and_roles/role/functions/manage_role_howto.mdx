# Управление пользовательскими ролями

Этот руководство описывает, как управлять пользовательскими ролями на платформе, включая:

- Обновление основной информации и разрешений
- Копирование существующих ролей для создания новых
- Удаление пользовательских ролей

## Обновление основной информации

Вы можете обновить отображаемое имя и описание пользовательских ролей на платформе.

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Роли**
2. Нажмите на имя ***роли для обновления***
3. Нажмите **Действия** > **Обновить** в правом верхнем углу
4. Обновите:
   - Отображаемое имя
   - Описание
5. Нажмите **Обновить**

## Обновление разрешений роли

Вы можете обновить информацию о разрешениях пользовательских ролей, включая:

- Добавление новых операционных разрешений для ресурсов платформы
- Удаление существующих разрешений
- Изменение разрешений для собственных ресурсов

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Роли**
2. Нажмите на имя ***роли для обновления***
3. Нажмите **Действия** > **Обновить разрешения роли** в правом верхнем углу области разрешений
4. Внесите изменения на странице **Обновление разрешений роли**
5. Нажмите **Подтвердить**

## Копирование существующей роли

Вы можете создать новую роль, скопировав существующую (системную или пользовательскую). Новая роль унаследует всю информацию о разрешениях от исходной роли, которую вы затем можете изменить в зависимости от ваших потребностей.

:::warning

Разрешения новой роли не могут превышать разрешения роли, к которой принадлежит создатель.

:::

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Роли**
2. Нажмите на имя ***роли для копирования***
3. Нажмите **Действия** > **Копировать как новую роль** в правом верхнем углу
4. На странице **Копировать как новую роль** настройте:
   - Имя
   - Отображаемое имя
   - Описание
   - Тип
5. Нажмите **Создать**

## Удаление пользовательской роли

Вы можете удалить пользовательские роли, которые больше не используются.

:::warning

Когда вы удаляете пользовательскую роль:

- Связи роли с пользователями будут удалены
- Пользователи, назначенные этой роли, потеряют все права, предоставленные ролью
- Роль будет удалена из списков ролей пользователей

:::

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Роли**
2. Нажмите на имя ***роли для удаления***
3. Нажмите **Действия** > **Удалить** в правом верхнем углу
4. Введите имя роли для подтверждения удаления
5. Нажмите **Удалить**
