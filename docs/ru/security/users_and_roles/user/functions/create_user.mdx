# Создание пользователя

Пользователи с ролями администратора платформы могут создавать локальных пользователей и назначать им роли через интерфейс платформы.

## Шаги

1. В левой панели навигации нажмите **Пользователи** > **Управление пользователями**
2. Нажмите **Создать пользователя**
3. Настройте следующие параметры:

| Параметр             | Описание                                                                                                                                                                                                                                                                                                                                                     |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Тип пароля**       | Выберите метод генерации пароля:<br /><br />**Случайный**: система генерирует безопасный случайный пароль<br />**Пользовательский**: пользователь вводит пароль вручную                                                                                                                                                                                      |
| **Пароль**           | Введите или создайте пароль в зависимости от выбранного типа.<br /><br />Требования к паролю:<br />- Длина: 8-32 символов<br />- Должен содержать буквы и цифры<br />- Должен содержать специальные символы (`~!@#$%^&*() -_=+?`)<br /><br />Функции поля пароля:<br />- Нажмите на значок глаза, чтобы отобразить/скрыть пароль<br />- Нажмите на значок копирования, чтобы скопировать пароль |
| **Электронная почта**| Адрес электронной почты пользователя:<br />- Должен быть уникальным<br />- Може использоваться как имя пользователя для входа<br />- Ассоциирован с именем пользователя                                                                                                                                                                                                                                          |
| **Срок действия**    | Установите срок действия учетной записи пользователя:<br /><br />Опции:<br />- **Постоянный**: без ограничения по времени<br />- **Пользовательский**: установите время начала и окончания, используя выпадающий список «Диапазон времени»                                                                                                                                                               |
| **Роли**             | Назначьте одну или несколько ролей пользователю                                                                                                                                                                                                                                                                                                              |
| **Продолжить создание** | Переключатель для управления поведением после создания:<br />- **Включено**: перенаправляет на страницу создания нового пользователя<br />- **Отключено**: показывает страницу с деталями пользователя                                                                                                                                                          |

4. Нажмите **Создать**

:::note

После успешного создания пользователя:

- Если "Продолжить создание" включено, вас перенаправят для создания другого пользователя
- Если отключено, вы увидите страницу с деталями созданного пользователя

:::
