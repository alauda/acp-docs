# Создание пользователя

Пользователи с правами администратора платформы могут создавать локальных пользователей и назначать им роли через интерфейс платформы.

## Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Управление пользователями**
2. Нажмите **Создать пользователя**
3. Настройте следующие параметры:

| Параметр              | Описание                                                                                                                                                                                                                                                                                                                                                     |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Тип пароля**       | Выберите метод генерации пароля:<br /><br />**Случайный**: Система генерирует безопасный случайный пароль<br />**Настраиваемый**: Пользователь вводит пароль вручную                                                                                                                                                                                         |
| **Пароль**           | Введите или сгенерируйте пароль в соответствии с выбранным типом.<br /><br />Требования к паролю:<br />- Длина: 8-32 символов<br />- Должен содержать буквы и цифры<br />- Должен содержать специальные символы (`~!@#$%^&*()-_=+?`)<br /><br />Функции поля пароля:<br />- Нажмите на иконку глаза, чтобы показать/скрыть пароль<br />- Нажмите на иконку копирования, чтобы скопировать пароль |
| **Электронная почта**| Адрес электронной почты пользователя:<br />- Должен быть уникальным<br />- Може быть использован в качестве имени пользователя для входа<br />- Ассоциирован с именем пользователя                                                                                                                                                                                                                                         |
| **Срок действия**    | Установите срок действия учетной записи пользователя:<br /><br />Варианты:<br />- **Постоянный**: Без ограничения по времени<br />- **Настраиваемый**: Установите время начала и окончания с помощью выпадающего списка диапазона времени                                                                                                                                                                 |
| **Роли**             | Назначьте одну или несколько ролей пользователю                                                                                                                                                                                                                                                                                                            |
| **Продолжить создание** | Переключатель для управления поведением после создания:<br />- **Включено**: Перенаправляет на страницу создания нового пользователя<br />- **Выкл**: Показывает страницу деталей пользователя                                                                                                                                                                   |

4. Нажмите **Создать**

:::note

После успешного создания пользователя:

- Если "Продолжить создание" включен, вы будете перенаправлены на создание другого пользователя
- Если отключен, вы увидите страницу с деталями созданного пользователя

:::
