# Управление ролями пользователей

Администраторы платформы могут управлять ролями для других пользователей (не для своей учетной записи), чтобы предоставить или отозвать права доступа.

## Добавление ролей

### Шаги

1. В левой панели навигации нажмите **Пользователи** > **Управление пользователями**
2. Нажмите на имя пользователя целевого пользователя
3. Прокрутите до секции **Список ролей**
4. Нажмите **Добавить роль**
5. В диалоговом окне назначения роли:
   - Выберите роль из выпадающего списка **Имя роли**
   - Выберите область разрешений роли (кластер, проект или пространство имен)
   - Нажмите **Добавить**

:::note

Важные примечания:

- Вы можете добавить несколько ролей к пользователю
- Каждая роль может быть добавлена только один раз для каждого пользователя
- Уже назначенные роли появляются в выпадающем списке, но их нельзя выбрать
- Роль **Администратор кластера** не может быть назначена для глобального кластера

:::

## Удаление ролей

### Шаги

1. В левой панели навигации нажмите **Пользователи** > **Управление пользователями**
2. Нажмите на имя пользователя целевого пользователя
3. Прокрутите до секции **Список ролей**
4. Нажмите **Удалить** рядом с ролью, которую вы хотите удалить
5. Подтвердите удаление

:::warning

Разрешения для управления ролями:

- Только администраторы платформы могут управлять ролями для других пользователей
- Пользователи не могут изменять роли для своей собственной учетной записи
