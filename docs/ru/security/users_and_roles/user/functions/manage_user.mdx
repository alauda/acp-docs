# Управление пользователями

Платформа предоставляет гибкие возможности управления пользователями, поддерживая как индивидуальное управление пользователями, так и пакетные операции для повышения эффективности в определенных сценариях (например, для удаленных или местных команд).

:::warning

Важные ограничения:

- Системные учетные записи не могут быть управляемы (роль администратора платформы, локальный источник)
- В настоящее время вошедшие пользователи не могут управлять своими собственными учетными записями
- Для изменений в личных учетных записях (имя отображения, пароль) используйте страницу личной информации

:::

## Сброс пароля локального пользователя

Пользователи с правами управления платформой могут сбрасывать пароли для других локальных пользователей.

### Шаги

1. В левом навигационном меню нажмите **Пользователи** > **Управление пользователями**
2. Нажмите значок рядом с записью целевого пользователя
3. Нажмите **Сбросить пароль**
4. В диалоговом окне выберите тип пароля:
   - **Случайный**: система генерирует безопасный случайный пароль
   - **Пользовательский**: введите новый пароль вручную

:::note

Требования к паролям:

- Длина: 8-32 символов
- Должен содержать буквы и цифры
- Должен содержать специальные символы (`~!@#$%^&*() -_=+?`)

Функции поля пароля:

- Нажмите значок глаза, чтобы показать/скрыть пароль
- Нажмите значок копирования, чтобы скопировать пароль

:::

5. Нажмите **Сбросить**

## Обновление даты истечения срока действия пользователя

Вы можете обновить даты истечения срока действия для пользователей со статусом **нормальный**, **отключен** или **заблокирован**. Пользователи, превышающие свою дату истечения, будут автоматически отключены.

### Шаги

1. В левом навигационном меню нажмите **Пользователи** > **Управление пользователями**
2. Нажмите **Обновить дату истечения** рядом с целевым пользователем
3. В диалоговом окне выберите вариант даты истечения:
   - **Постоянный**: без ограничения по времени
   - **Пользовательский**: установите начальное и конечное время с помощью выпадающего списка Дата и время
4. Нажмите **Обновить**

## Активировать пользователя

Вы можете активировать пользователей со статусом **отключен** или **заблокирован**.

:::note

Поведение при активации:

- Если пользователь в пределах даты истечения: дата истечения остается без изменений
- Если пользователь истек: дата истечения становится **Постоянной**

:::

### Шаги

1. В левом навигационном меню нажмите **Пользователи** > **Управление пользователями**
2. Нажмите **Активировать** рядом с целевым пользователем
3. Нажмите **Активировать** в диалоговом окне подтверждения
4. Статус пользователя изменится на **нормальный**

## Отключить пользователя

Вы можете отключить пользователей со статусом **нормальный** или **заблокирован** в пределах их даты истечения. Отключенные пользователи не могут войти в систему, но могут быть повторно активированы.

### Шаги

1. В левом навигационном меню нажмите **Пользователи** > **Управление пользователями**
2. Нажмите на значок рядом с целевым пользователем
3. Нажмите **Отключить** и подтвердите

## Добавить пользователя в локальную группу пользователей

Вы можете добавить пользователей с **Источник** как **Локальный** или `LDAP` в одну или несколько локальных групп пользователей.

:::warning

Поведение роли группы:

- Пользователи автоматически унаследуют роли от своих групп
- Роли группы видны только на странице деталей группы (вкладка Настроить роли)
- Списки ролей отдельных пользователей показывают только непосредственно назначенные роли

:::

### Шаги

1. В левом навигационном меню нажмите **Пользователи** > **Управление пользователями**
2. Нажмите на значок рядом с целевым пользователем
3. Нажмите **Добавить в группу пользователей**
4. Выберите одну или несколько локальных групп пользователей
5. Нажмите **Добавить**

## Удалить пользователя

Администраторы платформы могут удалить любого пользователя, кроме текущей вошедшей учетной записи, включая:

- Пользователей, сконфигурированных в IDP
- Пользователей с источником `-`
- Локальных пользователей

### Шаги

1. В левом навигационном меню нажмите **Пользователи** > **Управление пользователями**
2. Нажмите на значок рядом с целевым пользователем
3. Нажмите **Удалить**
4. Нажмите **Подтвердить**

## Пакетные операции

Вы можете выполнять пакетные операции для:

- Обновления сроков действия
- Активации пользователей
- Отключения пользователей
- Удаления пользователей

### Шаги

1. В левом навигационном меню нажмите **Пользователи** > **Управление пользователями**
2. Выберите одного или нескольких пользователей, используя флажки
3. Нажмите **Пакетные операции** и выберите действие:
   - **Обновить срок действия**
   - **Активировать**
   - **Отключить**
   - **Удалить**

:::note

Детали пакетной операции:

- **Обновить срок действия**: установить постоянный или пользовательский диапазон времени
- **Активировать**: подтвердите активацию в диалоговом окне
- **Отключить**: подтвердите деактивацию в диалоговом окне
- **Удалить**: введите текущий пароль учетной записи и подтвердите

:::
