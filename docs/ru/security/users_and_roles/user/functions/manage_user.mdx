# Управление пользователями

Платформа предоставляет гибкие возможности для управления пользователями, поддерживая как индивидуальное управление пользователями, так и пакетные операции для повышения эффективности в определённых сценариях (например, для команд на месте или удалённых команд).

:::warning

Важные ограничения:

- Аккаунты, созданные системой, не могут быть управляемыми (роль администратора платформы, локальный источник)
- В настоящее время вошедшие пользователи не могут управлять своими собственными аккаунтами
- Для изменения личного аккаунта (отображаемое имя, пароль) используйте страницу личной информации

:::

## Сброс пароля локального пользователя

Пользователи с правами управления платформой могут сбрасывать пароли для других локальных пользователей.

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Управление пользователями**
2. Нажмите на иконку рядом с записью целевого пользователя
3. Нажмите **Сбросить пароль**
4. В диалоговом окне выберите тип пароля:
   - **Случайный**: Система генерирует безопасный случайный пароль
   - **Пользовательский**: Введите новый пароль вручную

:::note

Требования к паролю:

- Длина: 8-32 символа
- Должен содержать буквы и цифры
- Должен содержать специальные символы (`~!@#$%^&*() -_=+?`)

Особенности поля пароля:

- Нажмите на иконку глаза, чтобы показать/скрыть пароль
- Нажмите на иконку копирования, чтобы скопировать пароль

:::

5. Нажмите **Сбросить**

## Обновить дату истечения срока действия пользователя

Вы можете обновить даты истечения для пользователей в статусе **нормально**, **отключён** или **заблокирован**. Пользователи, превышающие свою дату истечения, будут автоматически отключены.

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Управление пользователями**
2. Нажмите **Обновить дату истечения** рядом с целевым пользователем
3. В диалоговом окне выберите вариант даты истечения:
   - **Постоянно**: Без ограничения по времени
   - **Пользовательский**: Установите время начала и окончания с помощью выпадающего списка «Временной диапазон»
4. Нажмите **Обновить**

## Активировать пользователя

Вы можете активировать пользователей в статусе **отключён** или **заблокирован**.

:::note

Поведение активации:

- Если пользователь находится в пределах даты истечения: дата истечения остается без изменений
- Если пользователь истек: дата истечения становится **Постоянной**

:::

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Управление пользователями**
2. Нажмите **Активировать** рядом с целевым пользователем
3. Нажмите **Активировать** в диалоговом окне подтверждения
4. Статус пользователя изменится на **нормально**

## Отключить пользователя

Вы можете отключать пользователей в статусе **нормально** или **заблокирован** в пределах их даты истечения. Отключённые пользователи не могут войти в систему, но могут быть повторно активированы.

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Управление пользователями**
2. Нажмите на иконку рядом с целевым пользователем
3. Нажмите **Отключить** и подтвердите

## Добавить пользователя в локальную группу пользователей

Вы можете добавить пользователей с **Источник** как **Локальный** или `LDAP` в одну или несколько локальных групп пользователей.

:::warning

Поведение роли группы:

- Пользователи автоматически наследуют роли от своих групп
- Роли группы видны только на странице сведений о группе (вкладка «Настроить роли»)
- Списки ролей индивидуальных пользователей показывают только непосредственно назначенные роли

:::

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Управление пользователями**
2. Нажмите на иконку рядом с целевым пользователем
3. Нажмите **Добавить в группу пользователей**
4. Выберите одну или несколько локальных групп пользователей
5. Нажмите **Добавить**

## Удалить пользователя

Администраторы платформы могут удалять любого пользователя, кроме в настоящее время вошедшего аккаунта, включая:

- Пользователи с настройками IDP
- Пользователи с источником `-`
- Локальные пользователи

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Управление пользователями**
2. Нажмите на иконку рядом с целевым пользователем
3. Нажмите **Удалить**
4. Нажмите **Подтвердить**

## Пакетные операции

Вы можете выполнять пакетные операции для:

- Обновления сроков действия
- Активации пользователей
- Отключения пользователей
- Удаления пользователей

### Шаги

1. В левой навигационной панели нажмите **Пользователи** > **Управление пользователями**
2. Выберите одного или нескольких пользователей с помощью флажков
3. Нажмите **Пакетные операции** и выберите действие:
   - **Обновить срок действия**
   - **Активировать**
   - **Деактивировать**
   - **Удалить**

:::note

Подробности пакетной операции:

- **Обновить срок действия**: Установить постоянный или пользовательский временной диапазон
- **Активировать**: Подтвердите активацию в диалоговом окне
- **Деактивировать**: Подтвердите деактивацию в диалоговом окне
- **Удалить**: Введите пароль текущего аккаунта и подтвердите

:::
