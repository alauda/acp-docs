# Введение

## Обзор

Платформа интегрируется с сервисом аутентификации идентичности Dex, что позволяет использовать заранее реализованные подключатели Dex для аутентификации учетных записей платформы через конфигурацию IDP. Для получения дополнительной информации смотрите [официальную документацию Dex](https://github.com/dexidp/dex).

## Поддерживаемые методы интеграции

### Интеграция с LDAP

Если ваше предприятие использует **LDAP** (Протокол легкого доступа к каталогу) для управления пользователями, вы можете настроить LDAP на платформе для подключения к LDAP-серверу вашего предприятия.

Преимущества интеграции с LDAP:

- Обеспечивает связь между платформой и LDAP-сервером
- Позволяет пользователям предприятия входить в систему с учетными данными LDAP
- Автоматически синхронизирует учетные записи пользователей предприятия с платформой

### Интеграция с OIDC

Платформа поддерживает интеграцию с сервисами IDP, использующими протокол OpenID Connect (OIDC) для аутентификации пользователей третьих сторон.

Преимущества интеграции с OIDC:

- Позволяет пользователям входить в систему с учетными записями третьих сторон
- Поддерживает сервисы IDP для предприятий
- Обеспечивает безопасную аутентификацию через протокол OIDC

:::note

Для аутентификации с использованием других подключателей, не упомянутых выше, обратитесь в техническую поддержку.
