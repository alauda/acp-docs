# Введение

## Обзор

Платформа интегрируется с сервисом аутентификации идентификации Dex, что позволяет использовать заранее реализованные коннекторы Dex для аутентификации учетных записей платформы через конфигурацию IDP. Для получения дополнительной информации см. в [официальной документации Dex](https://github.com/dexidp/dex).

## Поддерживаемые методы интеграции

### Интеграция LDAP

Если ваше предприятие использует **LDAP** (Протокол легкого доступа к каталогам) для управления пользователями, вы можете настроить LDAP на платформе для подключения к LDAP-серверу вашего предприятия.

Преимущества интеграции LDAP:

- Обеспечивает связь между платформой и LDAP-сервером
- Позволяет пользователям предприятия входить с учетными данными LDAP
- Автоматически синхронизирует учетные записи пользователей предприятия с платформой

### Интеграция OIDC

Платформа поддерживает интеграцию с сервисами IDP с использованием протокола OpenID Connect (OIDC) для аутентификации пользователей третьих сторон.

Преимущества интеграции OIDC:

- Позволяет пользователям входить с учетными записями третьих сторон
- Поддерживает сервисы IDP предприятий
- Обеспечивает безопасную аутентификацию через протокол OIDC

:::note

Для аутентификации с использованием других коннекторов, не упомянутых выше, пожалуйста, свяжитесь с технической поддержкой.
