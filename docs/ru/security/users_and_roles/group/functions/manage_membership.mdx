# Управление членством в локальных группах пользователей

Только пользователи с разрешениями на управление платформой могут управлять членством в локальных группах пользователей.

## Предварительные условия

:::warning

Перед управлением членством в группах обратите внимание на следующие ограничения:

- Только пользователи с разрешениями на управление платформой могут управлять группами и их участниками
- Системные учетные записи и учетные записи, которые в данный момент вошли в систему, не могут быть управляемы (добавлены в или удалены из групп)
- Каждая локальная группа пользователей может иметь максимум 5000 участников
- Когда группа достигает предела в 5000 участников, дальнейшие импорты не допускаются

:::

## Импорт участников

Вы можете импортировать пользователей с платформы в локальные группы пользователей для единого управления разрешениями.

:::tip

Пользователи, импортированные в группу, автоматически унаследуют все разрешения на выполнение операций, назначенные этой группе.

:::

### Шаги

1. В левой боковой панели навигации нажмите **Пользователи** > **Управление группами пользователей**
2. Нажмите на имя локальной группы пользователей, в которую вы хотите добавить участников
3. На вкладке **Управление членами группы** нажмите **Импорт участника**
4. Выберите одного или несколько пользователей с платформы, отметив флажки рядом с их именами пользователей/отображаемыми именами
5. Нажмите **Импорт**

:::note

- Вы можете выбрать только пользователей, которые в данный момент не являются членами группы
- Используйте кнопку **Импортировать все**, чтобы импортировать всех пользователей из списка сразу

:::

## Удаление участников

Когда вы удаляете пользователя из группы, все операционные разрешения, предоставленные этому пользователю через группу, будут автоматически отозваны.

### Шаги

1. В левой боковой панели навигации нажмите **Пользователи** > **Управление группами пользователей**
2. Нажмите на имя локальной группы пользователей, из которой вы хотите удалить участников
3. На вкладке **Управление членами группы** вы можете удалить участников двумя способами:
   - Нажмите **Удалить** рядом с именем участника и подтвердите
   - Выберите одного или нескольких участников с помощью флажков, затем нажмите **Пакетное удаление** и подтвердите
