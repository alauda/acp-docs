# Управление членством в локальных группах пользователей

Только пользователи с правами управления платформой могут управлять членством в локальных группах пользователей.

## Предварительные условия

:::warning

Перед тем как управлять членством в группах, обратите внимание на следующие ограничения:

- Только пользователи с правами управления платформой могут управлять группами и их участниками
- Системные учетные записи и учетные записи, которые в данный момент вошли в систему, не могут управляться (импортироваться в группы или удаляться из них)
- Каждая локальная группа пользователей может иметь максимум 5000 участников
- Когда группа достигает предела в 5000 участников, дальнейший импорт не разрешен

:::

## Импорт участников

Вы можете импортировать пользователей из платформы в локальные группы пользователей для единого управления правами доступа.

:::tip

Пользователи, импортированные в группу, автоматически унаследуют все операционные права, назначенные этой группе.

:::

### Шаги

1. В левой панели навигации нажмите **Пользователи** > **Управление группами пользователей**
2. Нажмите на название локальной группы пользователей, в которую вы хотите добавить участников
3. На вкладке **Управление участниками группы** нажмите **Импортировать участника**
4. Выберите одного или нескольких пользователей из платформы, отметив галочки рядом с их именами пользователей/отображаемыми именами
5. Нажмите **Импортировать**

:::note

- Вы можете выбрать только тех пользователей, которые в данный момент не являются участниками группы
- Используйте кнопку **Импортировать всех**, чтобы импортировать всех пользователей в списке за раз

:::

## Удаление участников

Когда вы удаляете пользователя из группы, все операционные права, предоставленные этому пользователю через группу, будут автоматически отозваны.

### Шаги

1. В левой панели навигации нажмите **Пользователи** > **Управление группами пользователей**
2. Нажмите на название локальной группы пользователей, из которой вы хотите удалить участников
3. На вкладке **Управление участниками группы** вы можете удалить участников двумя способами:
   - Нажмите **Удалить** рядом с именем участника и подтвердите
   - Выберите одного или нескольких участников, используя флажки, затем нажмите **Пакетное удаление** и подтвердите
