# Создание локальной группы пользователей

Локальные группы пользователей позволяют вам реализовать контроль доступа на основе ролей для нескольких пользователей из любого источника.

## Создание группы пользователей

### Шаги

1. В левом боковом меню нажмите **Пользователи** > **Управление группами пользователей**
2. Нажмите **Создать группу пользователей**
3. Введите следующую информацию:
   - **Имя**: Имя группы пользователей
   - **Описание**: Описание назначения группы
4. Нажмите **Создать**

## Управление группами пользователей

Вы можете управлять группами пользователей, нажав на значок на странице списка или выбрав **Операции** в правом верхнем углу на странице с деталями.

| Операция                     | Описание                                                                                                                                                                                                                   |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Обновить группу пользователей** | Обновить информацию о группе на основе источника группы: <br />- Для групп с **Источник** как `Local`: Можно обновить как имя, так и описание <br />- Для групп с **Источник** как `IDP name`: Можно обновить только описание |
| **Удалить локальную группу пользователей** | Удалить группы пользователей с **Источник** как `Local`                                                                                                                                                                     |

:::warning

Когда вы удаляете группу:

- Все члены группы будут удалены
- Все роли, назначенные группе, будут удалены
- Это действие нельзя отменить

:::
