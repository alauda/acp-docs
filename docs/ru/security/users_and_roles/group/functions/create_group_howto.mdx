# Создание локальной группы пользователей

Локальные группы пользователей позволяют реализовать управление доступом на основе ролей для нескольких пользователей из любых источников.

## Создание группы пользователей

### Шаги

1. В левом боковом меню нажмите **Пользователи** > **Управление группами пользователей**
2. Нажмите **Создать группу пользователей**
3. Введите следующую информацию:
   - **Имя**: Название группы пользователей
   - **Описание**: Описание цели группы
4. Нажмите **Создать**

## Управление группами пользователей

Вы можете управлять группами пользователей, нажав на иконку на странице списка или нажав **Операции** в правом верхнем углу на странице сведений.

| Операция                    | Описание                                                                                                                                                                                                      |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Обновить группу пользователей** | Обновить информацию о группе на основе источника группы:  <br />- Для групп с **Источник** как `Local`: Можно обновить и имя, и описание  <br />- Для групп с **Источник** как `IDP name`: Можно обновить только описание |
| **Удалить локальную группу пользователей** | Удалить группы пользователей с **Источник** как `Local`                                                                                                                                                          |

:::warning

При удалении группы:

- Все члены группы будут удалены
- Все роли, назначенные группе, будут удалены
- Это действие нельзя отменить

:::
