# Управление ролями групп пользователей

Пользователи с разрешениями на управление платформой могут управлять ролями как для локальных групп пользователей, так и для групп пользователей, синхронизированных с провайдером идентификации (IDP).

## Добавить роль в группу

### Шаги

1. В левом навигационном меню нажмите **Пользователи** > **Управление группами пользователей**
2. Нажмите на название целевой группы пользователей
3. На вкладке **Настроить роль** нажмите **Добавить роль**
4. Нажмите, чтобы добавить роль

:::note

Правила назначения ролей:

- Вы можете добавить несколько ролей в группу
- Каждую роль можно добавить только один раз в одну и ту же группу

:::

5. Выберите имя роли из выпадающего списка
6. Выберите область разрешений роли (кластер, проект или пространство имен)
7. Нажмите **Добавить**

## Удалить роль из группы

:::warning

Когда вы удаляете роль из группы:

- Все разрешения, предоставленные этой ролью членам группы, будут отзываться
- Это действие не может быть отменено

:::

### Шаги

1. В левом навигационном меню нажмите **Пользователи** > **Управление группами пользователей**
2. Нажмите на название целевой группы пользователей
3. На вкладке **Настроить роль** нажмите **Удалить** рядом с ролью
4. Нажмите **Подтвердить**, чтобы удалить роль
