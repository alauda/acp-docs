# Установка

ACP Telemetry — это сервис платформы, который собирает телеметрические данные из кластеров для онлайн-операций и технического обслуживания. Он собирает системные метрики и загружает их в Alauda Cloud для мониторинга и анализа.

## Предварительные требования

Перед установкой убедитесь, что:

- Платформа ACP имеет действующую лицензию
- Глобальный кластер имеет доступ к интернету

## Шаги установки

1. Перейдите в **Управление платформой**
2. В левой навигационной панели нажмите **Магазин приложений** > **Плагины кластера**
3. Выберите **глобальный** кластер в верхней навигационной панели
4. Найдите **ACP Telemetry** и нажмите, чтобы увидеть его детали
5. Нажмите **Установить**, чтобы развернуть плагин

## Включение онлайн-операций

1. В левой навигационной панели нажмите **Системные настройки** > **Техническое обслуживание платформы**
2. Нажмите кнопку **Включить** для **Онлайн-операций**

## Шаги удаления

1. Следуйте шагам 1-4 из процесса установки, чтобы найти плагин
2. Нажмите **Удалить**, чтобы удалить плагин
