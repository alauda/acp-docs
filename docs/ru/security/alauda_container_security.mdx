# Безопасность контейнеров Alauda

Безопасность контейнеров Alauda — это комплексное решение для обеспечения безопасности, разработанное для Kubernetes и контейнеризованных сред. Оно предоставляет централизованное управление, автоматизированное сканирование уязвимостей, соблюдение политик и проверки соответствия, чтобы помочь организациям защитить свою контейнерную инфраструктуру в нескольких кластерах.

Безопасность контейнеров Alauda использует распределенную, основанную на контейнерах архитектуру, состоящую из Центральных Сервисов (для управления, API и UI) и Защищенных Кластерных Сервисов (для мониторинга, соблюдения политик и сбора данных). Она интегрируется с CI/CD конвейерами, SIEM, системами логирования и поддерживает встроенный сканер уязвимостей Scanner V4.
