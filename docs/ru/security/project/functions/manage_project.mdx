# Управление проектом

В этом руководстве объясняется, как обновить основную информацию и квоты проекта для указанного проекта или удалить проект.

## Обновление основной информации о проекте

Обновление основной информации для указанного проекта, такой как отображаемое имя и описание.

### Процедура

1. В представлении **Управление проектом** нажмите на название проекта, который необходимо обновить.

2. В левом навигационном окне нажмите **Подробности**.

3. Нажмите **Действия** > **Обновить основную информацию** в правом верхнем углу.

4. Измените или введите **Отображаемое имя** и **Описание**.

5. Нажмите **Обновить**.

## Обновление квоты проекта

Обновление ресурсных квот для проекта в каждом связанном кластере.

### Ограничения и лимиты

Когда проект связан с **Аномальным** кластером, обновление квот, назначенных проекту в этом кластере, не поддерживается.

### Процедура

1. В представлении **Управление проектом** нажмите на название проекта, который необходимо обновить.

2. В левом навигационном окне нажмите **Подробности**.

3. Нажмите **Обновить квоту** с правой стороны области квоты.

4. Обновите квоты, назначенные проекту в кластере, в соответствии с следующими рекомендациями:

:::note

- Квоты ресурсов GPU (vGPU/pGPU) могут быть настроены только в случае, если ресурсы GPU развернуты в кластер.<br /><br />Когда ресурс GPU является **vGPU**, квоты **vMemory** также могут быть настроены.<br /><br />**Единицы GPU**: 100 единиц виртуальных ядер эквивалентны 1 физическому ядру (1 pGPU = 1 ядро = 100 vGPU); 1 единица видеопамяти составляет 256 Ми.<br />Единицы pGPU выражаются целыми числами и могут быть назначены только целыми числами.

- Если квота не установлена для конкретного ресурса, ресурс будет иметь неограниченную квоту по умолчанию.

- Установленное значение квоты должно находиться в пределах диапазона квоты, отображаемого на странице.
:::

5. Нажмите **Обновить**.

## Удаление проекта

Удаление проектов, которые больше не используются.

:::warning
После удаления проекта ресурсы, занимаемые проектом в кластере, будут освобождены.
:::

### Процедура

1. В представлении **Управление проектом** нажмите на название проекта, который нужно удалить.

2. В левом навигационном меню нажмите **Подробности**.

3. Нажмите **Действия** > **Удалить проект** в правом верхнем углу.

4. Введите название проекта и нажмите **Удалить**.
