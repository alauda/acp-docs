# Управление проектом

Этот гид объясняет, как обновить основную информацию и квоты проекта для указанного проекта или удалить проект.

## Обновление основной информации о проекте

Обновите основную информацию для указанного проекта, такую как отображаемое имя и описание.

### Процедура

1. В представлении **Управление проектом** щелкните по имени проекта, который нужно обновить.

2. В левой панели навигации нажмите на **Детали**.

3. Нажмите **Действия** > **Обновить основные сведения** в правом верхнем углу.

4. Измените или введите **Отображаемое имя** и **Описание**.

5. Нажмите **Обновить**.

## Обновление квоты проекта

Обновите квоты ресурсов для проекта в каждом связанном кластере.

### Ограничения и лимиты

Когда проект связан с кластером **Аномальный**, обновление квот, назначенных проекту в этом кластере, не поддерживается.

### Процедура

1. В представлении **Управление проектом** щелкните по имени проекта, который нужно обновить.

2. В левой панели навигации нажмите на **Детали**.

3. Нажмите **Обновить квоту** на правой стороне области квоты.

4. Обновите квоты, назначенные проекту в кластере, в соответствии со следующими рекомендациями:

:::note

- Квоты ресурсов GPU (vGPU/pGPU) могут быть настроены только тогда, когда ресурсы GPU развернуты в кластере.<br /><br />Когда ресурс GPU является **vGPU**, также могут быть настроены квоты **vMemory**.<br /><br />**GPU-единицы**: 100 единиц виртуальных ядер эквивалентны 1 физическому ядру (1 pGPU = 1 ядро = 100 vGPU); 1 единица видеопамяти составляет 256 Mi; единицы pGPU — целые числа, и могут быть назначены только целыми числами.

- Если квота не установлена для определенного ресурса, у ресурса по умолчанию будет неограниченная квота.

- Установленное значение квоты должно находиться в пределах диапазона квоты, отображаемой на странице.
  :::

5. Нажмите **Обновить**.

## Удаление проекта

Удалите проекты, которые больше не используются.

:::warning
После удаления проекта ресурсы, занятые проектом в кластере, будут освобождены.
:::

### Процедура

1. В представлении **Управление проектом** щелкните по имени проекта, который нужно удалить.

2. В левой панели навигации нажмите на **Детали**.

3. Нажмите **Действия** > **Удалить проект** в правом верхнем углу.

4. Введите имя проекта и нажмите **Удалить**.
