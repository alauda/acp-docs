# Особенности

## Образы Виртуальных Машин

- Управление образами виртуальных машин
  
  Платформа поддерживает создание, обновление и удаление образов виртуальных машин.

- Управление учетными данными образов
  
  Поддерживается создание, обновление и удаление учетных данных образов.

## Виртуальные Машины

- Создание виртуальных машин/групп виртуальных машин

  Поддерживается быстрая установка отдельных виртуальных машин или групп виртуальных машин с одинаковыми конфигурациями.

- Управление виртуальными машинами

  Поддерживает различные операции управления, включая сброс пароля, обновления ключевой пары, обновления спецификаций, обновления меток/аннотаций, переустановку ОС и др.

- Управление дисками виртуальных машин

  Поддерживает операции управления дисками, такие как создание, удаление, присоединение, отсоединение и расширение дисков.

- Управление ключевыми парами

  Поддерживает создание, удаление и обновление SSH ключевых пар.

- Пакетные операции для виртуальных машин

  Поддерживает массовые операции для запуска, остановки, перезапуска и удаления виртуальных машин.

- VNC-вход в виртуальные машины

  Поддерживает вход в виртуальные машины через VNC.

- Быстрое нахождение виртуальных машин

  Отображает списки виртуальных машин по кластерам, позволяя администраторам платформы быстро находить пространство имен виртуальной машины.

- Мониторинг и уведомления

  Мониторинг и уведомления о загрузке процессора, памяти, хранилища и сетевого трафика виртуальных машин.

## Сетевая Инфраструктура Виртуальных Машин

- Использование сетей контейнеров

  Поддерживает настройку сетей контейнеров для виртуальных машин, включая режимы NAT и мост.

- Настройка внутренней маршрутизации

  Позволяет открывать виртуальные машины внутри или вне кластера, создавая объекты типа Service.

- Использование SR-IOV

  Поддерживает SR-IOV (Single Root I/O Virtualization) для высокопроизводительных сетевых интерфейсов.

## Резервное Копирование и Восстановление

- Снимки виртуальных машин

  Поддерживает создание снимков для виртуальных машин и восстановление их из снимков.
