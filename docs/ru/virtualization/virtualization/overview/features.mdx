# Возможности

## Образы Виртуальных Машин

- Управление Образами Виртуальных Машин

  Платформа поддерживает создание, обновление и удаление образов виртуальных машин.

- Управление Учетными Данные для Образов

  Поддерживает создание, обновление и удаление учетных данных для образов.

## Виртуальные Машины

- Создание Виртуальных Машин/Групп Виртуальных Машин

  Поддерживает быстрое создание индивидуальных виртуальных машин или групп виртуальных машин с одинаковыми конфигурациями.

- Управление Виртуальными Машинами

  Поддерживает различные операции управления, включая сброс пароля, обновление пар ключей, обновление спецификаций, обновление меток/аннотаций, переустановку ОС и т.д.

- Управление Дисками Виртуальных Машин

  Поддерживает операции управления дисками, такие как создание, удаление, присоединение, отсоединение и увеличение дисков.

- Управление Парой Ключей

  Поддерживает создание, удаление и обновление пар SSH-ключей.

- Пакетные Операции для Виртуальных Машин

  Поддерживает массовые операции по запуску, остановке, перезапуску и удалению виртуальных машин.

- VNC Вход в Виртуальные Машины

  Поддерживает вход в виртуальные машины на основе VNC.

- Быстрое Поиск Виртуальных Машин

  Отображает списки виртуальных машин по кластерам, позволяя администраторам платформы быстро находить пространство имен виртуальной машины.

- Мониторинг и Оповещения

  Мониторинг и оповещения о использовании ЦП, памяти, хранения и сети виртуальных машин.

## Сеть Виртуальных Машин

- Использование Сетей Контейнеров

  Поддерживает настройку сетей контейнеров для виртуальных машин, включая NAT и мостовые режимы.

- Настройка Внутренней Маршрутизации

  Позволяет раскрывать виртуальные машины внутри или вне кластера, создавая объекты Service.

- Использование SR-IOV

  Поддерживает SR-IOV (виртуализация ввода-вывода с одним корнем) для высокопроизводительных сетевых интерфейсов.

## Резервное Копирование и Восстановление

- Снимки Виртуальных Машин

  Поддерживает создание снимков для виртуальных машин и их восстановление из снимков.
