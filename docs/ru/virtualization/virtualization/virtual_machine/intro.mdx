# Введение

KubeVirt предоставляет CRD (Определения Пользовательских Ресурсов), такие как **VirtualMachine** и **VirtualMachineInstance**, для абстрагирования ресурсов виртуальных машин (ВМ). На основе этих CRD пользователи получают обширные возможности управления ВМ. На этой основе **ACP Виртуализация С KubeVirt** дополнительно улучшает удобство, предлагая **Веб Консоль**, которая позволяет пользователям выполнять различные операции с большей легкостью.

## Преимущества

- **Комплексное Управление ВМ**

  - Создание/удаление ВМ, переустановка ОС, сброс паролей, обновление SSH ключей, настройка распределения ресурсов и доступ к VNC консолям.
  - Экспорт/клонирование ВМ, выполнение горячей миграции и многое другое.

- **Удобные Операции**

  - Большинство операций с ВМ можно выполнить без усилий через интуитивно понятный Веб интерфейс.

- **Визуальный Мониторинг и Оповещения**

  - Мониторинг критически важной метрики ВМ (например, CPU, память, использование диска) через Веб интерфейс.
  - Настройка оповещений для проактивного обнаружения потенциальных проблем в работе (например, исчерпание ресурсов, ухудшение производительности).
