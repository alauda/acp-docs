# Введение

KubeVirt предоставляет CRD (Определения Пользовательских Ресурсов), такие как **VirtualMachine** и **VirtualMachineInstance**, для абстрагирования ресурсов виртуальной машины (VM). На основе этих CRD пользователи получают комплексные возможности управления виртуальными машинами. Основываясь на этом, **ACP Виртуализация с KubeVirt** дополнительно улучшает удобство использования, предлагая **Веб Консоль**, позволяя пользователям выполнять различные операции с большим комфортом.

## Преимущества

- **Комплексное управление ВМ**

  - Создавайте/удаляйте ВМ, переустанавливайте ОС, сбрасывайте пароли, обновляйте SSH-ключи, регулировать распределение ресурсов и получите доступ к VNC-консолям.
  - Экспортируйте/копируйте ВМ, выполняйте живую миграцию и многое другое.

- **Удобные операции**

  - Большинство операций с ВМ можно выполнить без особых усилий через интуитивно понятный Веб UI.

- **Визуальный мониторинг и оповещения**

  - Мониторинг критически важных метрик ВМ (например, CPU, оперативная память, использование диска) через Веб UI.
  - Настройка оповещений для проактивного выявления потенциальных проблем во время выполнения (например, исчерпание ресурсов, ухудшение производительности).
