# Сообщения об ошибках горячей миграции и решения

| Сообщение об ошибке                                                                                                                                                                                                                                                                                                                                                                                                                                   | Причина                                                                                            | Решение                                                                                                                                                                                                                                                                                                                                                                                                                       |
| :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------ | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| невозможно выполнить миграцию VMI, который не использует маскарад, мост с \<annotation> аннотацией VM или переносимый плагин для подключения к сети пода                                                                                                                                                                                                                                                                                                 | Сетевая конфигурация виртуальной машины не поддерживает горячую миграцию.                         | Пожалуйста, проверьте следующие конфигурации: <ul><li>Проверьте, какой сетевой плагин CNI используется в текущем кластере; рекомендуется использовать Kube-OVN.</li><li>Проверьте, существует ли аннотация "kubevirt.io/allow-pod-bridge-network-live-migration": "true" в полях `metadata.annotations` и `spec.template.metadata.annotations` соответствующего YAML-файла виртуальной машины; если нет, добавьте ее вручную.</li></ul> |
| <ul><li>невозможно выполнить миграцию VMI: Не удалось определить, является ли PVC \<pvc name> общим, горячая миграция требует, чтобы все PVC были общими (используя режим доступа ReadWriteMany)</li><li>невозможно выполнить миграцию VMI: PVC \<pvc name> не является общим, горячая миграция требует, чтобы все PVC были общими (используя режим доступа ReadWriteMany)</li><li>невозможно выполнить миграцию VMI: PVC бэкенд-хранилища не RWX</li><li>невозможно выполнить миграцию VMI с неразделяемым HostDisk</li></ul> | Тип хранилища виртуальной машины не поддерживает режим доступа с чтением и записью для нескольких узлов (RWX). | Параметры, связанные с виртуальной машиной, не могут быть изменены после создания. Поэтому, пожалуйста, воссоздайте виртуальную машину и выберите тип хранилища, который поддерживает режим доступа с чтением и записью для нескольких узлов (RWX); рекомендуется использовать блочное хранилище CephRBD. Если проблемы сохраняются после воссоздания, пожалуйста, обратитесь к соответствующим лицам за помощью.                                                                                           |
