# Ошибки и решения при горячей миграции

| Сообщение об ошибке                                                                                                                                                                                                                                                                                                                                                                                                                                   | Причина                                                                                             | Решение                                                                                                                                                                                                                                                                                                                                                                                                                       |
| :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------ | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| невозможно выполнить миграцию VMI, который не использует маскировку, мост с \<annotation> аннотацией VM или мигрируемым плагином для подключения к сети подов                                                                                                                                                                                                                                                                                                  | Сетевая конфигурация виртуальной машины не поддерживает горячую миграцию.                  | Пожалуйста, проверьте следующие настройки: <ul><li>Проверьте используемый плагин сети CNI в текущем кластере; рекомендуется использовать Kube-OVN.</li><li>Проверьте наличие аннотации "kubevirt.io/allow-pod-bridge-network-live-migration": "true" в полях `metadata.annotations` и `spec.template.metadata.annotations` соответствующего YAML-файла виртуальной машины; если её нет, пожалуйста, добавьте её вручную.</li></ul> |
| <ul><li>невозможно выполнить миграцию VMI: Невозможно определить, является ли PVC \<имя pvc> общим, горячая миграция требует, чтобы все PVC были общими (с использованием режима доступа ReadWriteMany)</li><li>невозможно выполнить миграцию VMI: PVC \<имя pvc> не общий, горячая миграция требует, чтобы все PVC были общими (с использованием режима доступа ReadWriteMany)</li><li>невозможно выполнить миграцию VMI: PVC бэкенд-хранилища не RWX</li><li>невозможно выполнить миграцию VMI с не общим HostDisk</li></ul> | Тип хранилища виртуальной машины не поддерживает многосетевой режим чтения и записи (RWX). | Параметры, относящиеся к виртуальной машине, не могут быть изменены после создания. Поэтому, пожалуйста, воссоздайте виртуальную машину и выберите тип хранилища, который поддерживает многосетевое чтение и запись (RWX); рекомендуется использовать блочное хранилище CephRBD. Если проблемы сохраняются после воссоздания, пожалуйста, обратитесь за помощью к соответствующему персоналу.                                                                                           |
