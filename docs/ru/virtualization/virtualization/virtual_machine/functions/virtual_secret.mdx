# Управление ключевыми парами

Создание, обновление или удаление ключевых пар.

## Создание ключевых пар

1. Перейдите в **Container Platform**.

2. В левой панели навигации нажмите **Виртуализация** > **Ключевые пары**.

3. Нажмите **Создать ключевую пару**.

   В настоящее время поддерживаются только SSH ключевые пары. Вы можете вручную импортировать ключи или позволить системе автоматически сгенерировать ключевую пару. При использовании сгенерированной системой ключевой пары платформа поддерживает автоматическую загрузку закрытого ключа на ваш локальный компьютер. Платформа не будет сохранять закрытый ключ.

4. Нажмите **Создать**.

## Обновление ключевых пар

1. Перейдите в **Container Platform**.

2. В левой панели навигации нажмите **Виртуализация** > **Ключевые пары**.

3. Найдите ***Название ключевой пары***, нажмите ⋮ > **Обновить**.

4. После повторного импорта или генерации системы новой ключевой пары нажмите **Обновить**.

## Удаление ключевых пар

1. Перейдите в **Container Platform**.

2. В левой панели навигации нажмите **Виртуализация** > **Ключевые пары**.

3. Найдите ***Название ключевой пары***, нажмите ⋮ > **Удалить** и подтвердите.
