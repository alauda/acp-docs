# Пакетные операции с виртуальными машинами

Выполняйте пакетные операции, такие как запуск, остановка, перезапуск и удаление виртуальных машин.

## Процедура

1. Получите доступ к **Платформе контейнеров**.

2. Нажмите на **Виртуализация** > **Виртуальные машины** в левой навигационной панели.

3. Найдите целевую виртуальную машину, нажмите ⋮ для выполнения операций с одной виртуальной машиной или обратитесь к изображению ниже для пакетных операций с виртуальными машинами.

   **Примечание**:

   - Операция **Запустить/Пакетный запуск** может быть выполнена, когда виртуальная машина находится в приостановленном или остановленном состоянии; операция **Остановить/Пакетная остановка** может быть выполнена, когда виртуальная машина находится в состоянии Подготовка, Запуск, Работа, Приостановка, Неизвестное или Исключение; операция **Перезапустить/Пакетный перезапуск** может быть выполнена, когда виртуальная машина находится в рабочем состоянии.

   - Принудительное выполнение операции **Перезапуск/Остановка** на виртуальной машине эквивалентно отключению питания виртуальной машины, что может привести к потере данных, которые еще не были записаны на диск.

4. Завершите операции в соответствии с подсказками на интерфейсе. Когда виртуальная машина изменит состояние на приведенные ниже, операция считается успешной.

   | Операция                     | Статус  |
   | :--------------------------- | :------ |
   | **Запустить виртуальную машину** | Работа  |
   | **Остановить виртуальную машину** | Остановлено |
   | **Перезапустить виртуальную машину** | Работа  |
