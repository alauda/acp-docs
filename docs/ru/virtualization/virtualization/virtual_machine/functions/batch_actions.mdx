# Пакетные операции с виртуальными машинами

Выполняйте пакетные операции, такие как запуск, остановка, перезагрузка и удаление виртуальных машин.

## Процедура

1. Получите доступ к **Container Platform**.

2. Щелкните на **Виртуализация** > **Виртуальные машины** в левой навигационной панели.

3. Найдите целевую виртуальную машину, щелкните ⋮ чтобы выполнить операции с одной виртуальной машиной, или обратитесь к изображению ниже для пакетных операций с виртуальными машинами.

   **Примечание**:

   - Операция **Запуск/Пакетный запуск** может быть выполнена, когда виртуальная машина находится в состоянии приостановки или остановки; операция **Остановка/Пакетная остановка** может быть выполнена, когда виртуальная машина находится в состоянии Подготовка, Запуск, Работа, Приостановлена, Неизвестно или Исключение; операция **Перезагрузка/Пакетная перезагрузка** может быть выполнена, когда виртуальная машина находится в состоянии Работа.

   - Принудительная операция **Перезагрузка/Остановка** виртуальной машины эквивалентна отключению питания от виртуальной машины, что может привести к потере данных, которые не были записаны на диск.

4. Завершите операции в соответствии с подсказками на интерфейсе. Когда виртуальная машина изменит состояние на приведенные ниже, операция будет успешной.

   | Операция                     | Статус  |
   | :--------------------------- | :------ |
   | **Запустить виртуальную машину**   | Работа   |
   | **Остановить виртуальную машину**    | Остановлена |
   | **Перезагрузить виртуальную машину** | Работа   |
