# Быстрое определение местоположения виртуальных машин

Платформа поддерживает отображение списка виртуальных машин по кластерам, что позволяет администраторам платформы быстро находить пространство имен виртуальной машины и выполнять такие операции, как увеличение мощности или устранение неполадок, тем самым повышая операционную эффективность.

## Предпосылки

Убедитесь, что функция виртуализации включена для текущего кластера перед использованием. Пожалуйста, обратитесь к [Установке](../../installation.mdx).

## Процедура

1. Перейдите в раздел **Управление платформой**.

2. В левом навигационном меню выберите **Управление виртуализацией** > **Виртуальные машины**.

3. Выберите ***Кластер***, чтобы просмотреть список виртуальных машин в этом кластере.

4. Вы можете быстро найти виртуальную машину по ее имени, IP-адресу или создателю.

5. Нажмите на ссылку ***Имя*** виртуальной машины, чтобы перейти на страницу с деталями этой виртуальной машины, где вы можете выполнять такие операции, как увеличение мощности или устранение неполадок.
