# Вход в виртуальную машину с помощью VNC

Войдите в виртуальную машину с помощью веб-консоли (VNC) как метод аварийной операции.

## Процедура

1. Перейдите на **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Виртуализация** > **Виртуальные машины**.

3. Нажмите ⋮ > **VNC Вход**.

4. Окно консоли откроется автоматически; вам нужно ввести ваше имя пользователя и пароль для входа.

   <img src="../../../../../en/virtualization/virtualization/virtual_machine/functions/assets/vmi00.png" width="600" />

   **Примечание**:

   - Поддерживает отправку общих команд с клавиатуры.

   - Поддерживает копирование и вставку команд и параметров.
