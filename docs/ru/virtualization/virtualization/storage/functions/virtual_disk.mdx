# Управление виртуальными дисками

Данные диски могут использоваться для удовлетворения требований бизнеса в отношении сохранения данных.

## Создание виртуального диска

Создайте **дата-диск** для виртуальной машины. В одно время можно добавить только **один** виртуальный диск; если необходимо больше дисков, повторите эту операцию.

**Примечание**: Виртуальные диски могут быть смонтированы онлайн, когда виртуальная машина находится в состоянии **работы**.

### Процедуры

1. Получите доступ к **Платформе контейнеров**.

2. В левой навигационной панели щелкните **Виртуализация** > **Виртуальный диск**.

3. Щелкните **Создать виртуальный диск**.

4. Настройте информацию на основании следующих инструкций.

   | Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                 |
   | :---------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Режим тома**    | - **Файловая система**: смонтировать диск таким образом, чтобы он монтировал файловый каталог.<br />- **Блочное устройство**: смонтировать диск как блочное устройство.                                                                                                                                                                                                                                                                       |
   | **Класс хранения**  | Платформа поддерживает диски виртуальных машин, автоматически создавая и управляя заявками на постоянный том. Вам необходимо указать класс хранения, необходимый для динамического создания заявок на постоянный том.<br /><br />Разные классы хранения поддерживают разные режимы тома. Если для выбранного режима тома нет доступных классов хранения, пожалуйста, свяжитесь с администратором для добавления. |
   | **Удалить с ВМ** | Если включено, данные диска также будут удалены, когда виртуальная машина будет удалена.                                                                                                                                                                                                                                                                                                                         |
   | **Монтирование**          | - **Не монтировать**: только создать виртуальный диск; его можно смонтировать позже, когда это будет необходимо.<br />- **Смонтировать на ВМ**: выберите целевую виртуальную машину, на которую нужно смонтировать виртуальный диск.                                                                                                                                                                                                             |

5. Щелкните **Создать**.

## Монтирование виртуального диска

Смонтируйте **дата-диск** на виртуальной машине, присоединив уже созданный виртуальный диск к целевой виртуальной машине.

**Примечание**: Виртуальные диски могут быть смонтированы онлайн, когда виртуальная машина находится в состоянии **работы**.

### Процедуры

1. Получите доступ к **Платформе контейнеров**.

2. В левой навигационной панели щелкните **Виртуализация** > **Виртуальный диск**.

3. Щелкните ⋮ > **Смонтировать** рядом с виртуальным диском, который нужно смонтировать.

4. Выберите целевую виртуальную машину и щелкните **Смонтировать**.

## Расширение виртуального диска

Расширьте **системный диск** и **дата-диск**, уже смонтированные на виртуальной машине.

### Процедуры

1. Получите доступ к **Платформе контейнеров**.

2. В левой навигационной панели щелкните **Виртуализация** > **Виртуальная машина**.

3. Щелкните по имени виртуальной машины, чтобы перейти на страницу **Подробности**.

4. В области **Виртуальный диск** найдите диск, который нужно расширить, и щелкните ⋮ > **Расширить**.

5. Введите новую емкость и щелкните **Расширить**.

## Демонтаж виртуального диска

Демонтаж **дата-диска** с виртуальной машины; только виртуальные машины в состоянии **остановлено** могут демонтировать диски.

### Процедуры

1. Получите доступ к **Платформе контейнеров**.

2. В левой навигационной панели щелкните **Виртуализация** > **Виртуальный диск**.

3. Щелкните ⋮ > **Демонтировать** рядом с виртуальным диском, который нужно демонтировать, и подтвердите.

## Удаление виртуального диска

Удаление поддерживается только в том случае, если виртуальный диск находится в состоянии не смонтированного.

**Примечание**: Системные диски нельзя удалить.

### Процедуры

1. Получите доступ к **Платформе контейнеров**.

2. В левой навигационной панели щелкните **Виртуализация** > **Виртуальный диск**.

3. Щелкните ⋮ > **Удалить** рядом с виртуальным диском, который нужно удалить, и подтвердите.
