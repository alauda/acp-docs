# Управление виртуальными дисками

Данные диски могут быть использованы для удовлетворения требований бизнеса к долговременному хранению данных.

## Создание виртуального диска

Создайте **данный диск** для виртуальной машины. Можно добавить только **один** виртуальный диск за раз; если требуется несколько дисков, повторите данную операцию.

**Примечание**: Виртуальные диски могут быть подключены в онлайн-режиме, когда виртуальная машина находится в состоянии **работы**.

### Процедуры

1. Доступ к **Платформе контейнеров**.

2. В левомNavigation-bar нажмите на **Виртуализация** > **Виртуальный диск**.

3. Нажмите на **Создать виртуальный диск**.

4. Настройте информацию на основе следующих инструкций.

   | Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                 |
   | :----------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Режим тома**    | - **Файловая система**: Подключить диск таким образом, чтобы подключить файловый каталог.<br />- **Блочное устройство**: Подключить диск как блочное устройство.                                                                                                                                                                                                                                                                       |
   | **Класс хранения**  | Платформа управляет дисками виртуальной машины, автоматически создавая и управляя постоянными объемами. Вам нужно указать класс хранения, необходимый для динамического создания постоянных объемов.<br /><br />Разные классы хранения поддерживают разные режимы тома. Если для выбранного режима тома нет доступных классов хранения, обратитесь к администратору для их добавления. |
   | **Удалить с ВМ** | Если включено, данные диска также будут удалены при удалении виртуальной машины.                                                                                                                                                                                                                                                                                                                         |
   | **Подключить**          | - **Не подключать**: Только создать виртуальный диск; его можно подключить позже при необходимости.<br />- **Подключить к ВМ**: Выберите целевую виртуальную машину, к которой необходимо подключить виртуальный диск.                                                                                                                                                                                                             |

5. Нажмите на **Создать**.

## Подключение виртуального диска

Подключите **данные диски** к виртуальной машине, прикрепив уже созданный виртуальный диск к целевой виртуальной машине.

**Примечание**: Виртуальные диски могут быть подключены в онлайн-режиме, когда виртуальная машина находится в состоянии **работы**.

### Процедуры

1. Доступ к **Платформе контейнеров**.

2. В левомNavigation-bar нажмите на **Виртуализация** > **Виртуальный диск**.

3. Нажмите ⋮ > **Подключить** рядом с виртуальным диском, который нужно подключить.

4. Выберите целевую виртуальную машину и нажмите **Подключить**.

## Расширение виртуального диска

Расширьте **системный диск** и **данный диск**, уже подключенные к виртуальной машине.

### Процедуры

1. Доступ к **Платформе контейнеров**.

2. В левомNavigation-bar нажмите на **Виртуализация** > **Виртуальная машина**.

3. Нажмите на имя виртуальной машины, чтобы войти на страницу **Подробности**.

4. В области **Виртуальный диск** найдите диск, который нужно расширить, и нажмите ⋮ > **Расширить**.

5. Введите новую емкость и нажмите **Расширить**.

## Отключение виртуального диска

Отключите **данные диски** от виртуальной машины; диски можно отключать только от виртуальных машин в состоянии **остановлено**.

### Процедуры

1. Доступ к **Платформе контейнеров**.

2. В левомNavigation-bar нажмите на **Виртуализация** > **Виртуальный диск**.

3. Нажмите ⋮ > **Отключить** рядом с виртуальным диском, который нужно отключить, и подтвердите.

## Удаление виртуального диска

Удаление поддерживается только когда виртуальный диск находится в состоянии отключения.

**Примечание**: Системные диски нельзя удалять.

### Процедуры

1. Доступ к **Платформе контейнеров**.

2. В левомNavigation-bar нажмите на **Виртуализация** > **Виртуальный диск**.

3. Нажмите ⋮ > **Удалить** рядом с виртуальным диском, который нужно удалить, и подтвердите.
