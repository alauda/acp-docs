# Введение

Виртуализация ACP с помощью KubeVirt Storage предоставляет возможности постоянного хранения для виртуальных машин (ВМ) за счет бесшовной интеграции с механизмами хранения, основанными на Kubernetes. Она использует **PersistentVolumeClaim** (PVC) для хранения данных дисков ВМ и применяет **Container Storage Interface** (CSI) для интеграции с различными системами хранения. Кроме того, **Containerized Data Importer** (CDI) используется для инициализации данных диска ВМ. Основываясь на этих основах, платформа расширяет продвинутые функции управления дисками ВМ, обеспечивая всесторонний контроль жизненного цикла.

## Преимущества

- Удобные операции

  Большинство операций с дисками ВМ можно легко выполнять через **Web UI**, минимизируя необходимость в знаниях CLI.

- Управление жизненным циклом дисков ВМ

  Настройте, должны ли диски ВМ автоматически удаляться, когда связанная ВМ завершается.
