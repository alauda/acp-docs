# Введение

Виртуализация ACP с помощью хранилища KubeVirt предоставляет возможности постоянного хранения для виртуальных машин (VM), безшовно интегрируя с механизмами хранения, родными для Kubernetes. Она использует **PersistentVolumeClaim** (PVC) для хранения данных диска VM и использует **Container Storage Interface** (CSI) для интеграции с различными системами хранения. Кроме того, используется **Containerized Data Importer** (CDI) для инициализации данных диска VM. Основываясь на этих принципах, платформа расширяет расширенные функциональные возможности для управления диском VM, позволяя комплексное управление жизненным циклом.

## Преимущества

- Удобные операции

  Большинство операций с дисками VM можно легко выполнять через **Web UI**, что минимизирует необходимость в знании CLI.

- Управление жизненным циклом дисков VM

  Настройте, следует ли автоматически удалять диски VM при завершении работы связанной VM.
