# Введение

**Виртуализация ACP с помощью Kubevirt** предоставляет возможности создания снимков виртуальных машин, позволяя пользователям создавать резервные копии и восстанавливать ВМ через снимки.

## Сценарии применения

- Восстановление после катастроф и откат при сбоях

  Когда виртуальная машина сталкивается с потерей данных из-за сбоя аппаратного обеспечения, человеческих ошибок (например, случайное удаление файлов) или злонамеренных атак (например, программ-вымогателей), снимки служат последней линией обороны для восстановления работы.

## Ограничения по использованию

- Создание снимка требует предварительной остановки виртуальной машины.
- PVC (Persistent Volume Claim), используемый диском виртуальной машины, должен быть настроен с режимом совместного доступа для нескольких узлов.
