# Введение

ACP Виртуализация с KubeVirt глубоко интегрирована с Kube-OVN, расширяя поддержку требований к сетевому взаимодействию традиционных виртуальных машин (VM) и оптимизируя производительность для конкретных сценариев.

## Преимущества

- Поддержка IPv6

  Полная поддержка IPv6.

- Сохранение статического IP

  Обеспечивает сохранение теми же IP-адресами ВМ после перезагрузки, что соответствует традиционным шаблонам использования ВМ.

- Поддержка многоуровневого сетевого режима

  Поддерживает несколько сетевых режимов, таких как контейнерные сети и SR-IOV, что соответствует разнообразным пользовательским сценариям.
