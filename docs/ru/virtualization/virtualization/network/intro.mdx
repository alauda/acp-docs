# Введение

Виртуализация ACP с KubeVirt глубоко интегрирована с Kube-OVN, расширяя поддержку традиционных требований к сетевым подключениям виртуальных машин (VM) и оптимизируя производительность для конкретных сценариев.

## Преимущества

- Поддержка IPv6

  Полная поддержка IPv6.

- Сохранение статического IP

  Обеспечивает сохранение виртуальными машинами одного и того же IP-адреса после перезагрузок, что соответствует устоявшимся моделям использования виртуальных машин.

- Поддержка режима многосетевой работы

  Поддерживает несколько сетевых режимов, таких как контейнерные сети и SR-IOV, что позволяет учитывать разнообразные сценарии пользователей.
