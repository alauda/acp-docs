# Обновление/Удаление образов виртуальных машин

1. Перейдите в **Управление платформой**.

2. В левой боковой панели щелкните **Управление виртуализацией** > **Образы виртуальных машин**.

3. Нажмите ⋮ > **Обновить/Удалить**.

4. После подтверждения нажмите **Обновить/Удалить**.
