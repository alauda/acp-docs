# Обновление/Удаление учетных данных изображения

1. Перейдите в **Управление платформой**.

2. В левой панели навигации нажмите **Управление виртуализацией** > **Образы виртуальных машин**.

3. На вкладке **Учетные данные изображения** нажмите ⋮ > **Обновить/Удалить**.

4. После подтверждения нажмите **Обновить/Удалить**.
