# Введение

Alauda Container Platform Virtualization с KubeVirt использует расширенные возможности API Kubernetes для абстрагирования образов виртуальных машин в виде **определения пользовательского ресурса** (CRD). Она предоставляет интерфейс пользователя (UI), позволяющий пользователям легко импортировать образы виртуальных машин, хранящиеся в удаленных репозиториях, в ACP для использования.

## Преимущества

- Поддержка основных операционных систем

  Поддерживаются различные широко используемые дистрибутивы Linux и операционные системы Windows.

- Поддержка нескольких архитектур

  Совместима как с архитектурой **X86\_64**, так и с **ARM64**.

- Поддержка нескольких источников

  Позволяет импортировать образы виртуальных машин из:

  - Регистраторов образов
  - Файловых серверов
  - Объектного хранилища, совместимого с S3

- Поддержка нескольких форматов

  Поддерживает образы виртуальных машин в форматах **QCOW2** и **RAW**.
