# Введение

Платформа виртуализации Alauda Container Platform с KubeVirt использует расширенные API Kubernetes для абстрагирования образов виртуальных машин в виде **Определения настраиваемого ресурса** (CRD). Она предоставляет пользовательский интерфейс (UI), который позволяет пользователям легко импортировать образы виртуальных машин, хранящиеся в удаленных репозиториях, в ACP для дальнейшего использования.

## Преимущества

- Поддержка основных операционных систем

  Поддерживает различные часто используемые дистрибутивы Linux и операционные системы Windows.

- Поддержка нескольких архитектур

  Совместимость как с архитектурой **X86\_64**, так и с **ARM64**.

- Поддержка нескольких источников

  Позволяет импортировать образы виртуальных машин из:

  - Реестров образов
  - Файловых серверов
  - Объектного хранения, совместимого с S3

- Поддержка нескольких форматов

  Поддерживает образы виртуальных машин в форматах **QCOW2** и **RAW**.
