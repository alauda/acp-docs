# О служебной сетке Alauda

Служебная сетка — это PaaS-платформа на базе Istio, которая предоставляет облачные решения для приложений и микросервисов.

Платформа предлагает возможность развертывания и обновления инфраструктуры в один клик, визуализированное управление службами, эффективное управление производительностью приложений, а также высокую доступность и производительность для управления шлюзами служб. Она предоставляет комплексное, стабильное, надежное и открытое решение для микросервисов для предприятий, проходящих цифровую трансформацию, помогает повысить эффективность управления службами и снизить затраты на обслуживание микросервисной архитектуры, минимизируя зависимость разработчиков от архитектуры, что позволяет предприятиям сосредоточиться на развитии бизнеса и постоянно совершенствовать свою ключевую конкурентоспособность.
