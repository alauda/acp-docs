# О сервисной сетке Alauda

Сервисная сетка является PaaS платформой, основанной на Istio, которая предоставляет облачные решения для приложений и микросервисов.

Платформа предлагает возможности развертывания и обновления инфраструктуры в один клик, визуализированное управление сервисами, эффективное управление производительностью приложений, а также высокую доступность и производительность для управления шлюзом сервиса. Она предоставляет комплексное, стабильное, надежное и открытое решение для микросервисов для предприятий, проходящих цифровую трансформацию, помогая повысить эффективность управления сервисами и снизить затраты на обслуживание микросервисной структуры, при этом минимизируя зависимость разработчиков от фреймворка, позволяя предприятиям сосредоточиться на развитии бизнеса и непрерывно повышать свою核心竞争力.
