# Загрузка установочного пакета \{#download\_installation\_package}

Перед установкой необходимо скачать установочный пакет и файл подписи, а также использовать инструмент GPG для проверки его целостности.

<Steps>
  ### Скачивание установочных пакетов

  Войдите в **Портал Клиента**, чтобы скачать установочные пакеты и файлы подписи. Если у вас нет зарегистрированной учетной записи, пожалуйста, свяжитесь с технической поддержкой.

  Доступны два типа установочных пакетов:

  - **Основной пакет**: Требуется для всех установок
  - **Пакет расширений**: Дополнительно, скачивать только в случае необходимости для дополнительных функций

  ### Установка инструмента GPG

  Установите инструмент GPG в соответствии с вашей операционной системой (смотрите команды ниже).

  ```bash
  # Ubuntu
  sudo apt-get install gnupg
  # CentOS
  yum install gnupg
  # Mac
  brew install gpg
  ```

  ### Скачивание открытого ключа

  ```bash
  curl -O https://www.alauda.cn/download/verify-key.pub
  md5sum verify-key.pub
  ```

  Полученное значение MD5 должно быть: `2eaddfab97d2951a8915f327acb53562`.

  ### Проверка отпечатка открытого ключа

  ```bash
  gpg --import verify-key.pub
  gpg --fingerprint BB097AE6
  ```

  Отпечаток должен точно совпадать:

  ```
  09EE E7B9 A30C F4B3 5E31 A91B 2704 1C16 BB09 7AE6
  ```

  ### Проверка установочных пакетов

  Вы должны проверить каждый пакет, который вы скачали:

  Для основного пакета:

  ```bash
  gpg --verify <файл подписи основного пакета> <основной пакет>
  ```

  Пример:

  ```bash
  gpg --verify acp-4.0.0.tgz.sig acp-4.0.0.tgz
  ```

  Если вы скачали пакет расширений:

  ```bash
  gpg --verify <файл подписи пакета расширений> <пакет расширений>
  ```

  Пример:

  ```bash
  gpg --verify acp-extensions-4.0.0.tgz.sig acp-extensions-4.0.0.tgz
  ```

  Обычный вывод должен содержать информацию "Хорошая подпись" для каждой проверки.

  ### Документация пакета расширений

  Если вы скачали пакет расширений, вам нужно извлечь его документацию перед продолжением установки основного пакета:

  ```bash
  tar -xvf acp-extensions-4.0.0.tgz
  ```

  Обратитесь к извлеченной документации для получения информации о том, как использовать пакет расширений вместе с основным пакетом.
</Steps>

Руководство по установке подробно опишет процесс установки основного пакета. Документация по пакету расширений предоставит конкретные инструкции по использованию и настройке расширений.
