# Загрузка установочного пакета \{#download\_installation\_package}

Перед установкой необходимо загрузить и распаковать установочные пакеты. Если вам требуется пакет расширений, сначала распакуйте основной пакет, а затем пакет расширений в ту же директорию перед началом установки.

<Steps>
  ### Загрузка установочных пакетов

  Войдите в **Портал клиента**, чтобы загрузить установочные пакеты. Если у вас еще нет зарегистрированной учетной записи, пожалуйста, свяжитесь с технической поддержкой.

  Доступны два типа установочных пакетов:

  - **Основной пакет**: необходим для всех установок
  - **Пакет расширений**: необязателен, загружайте только при необходимости для дополнительной функциональности

  ### Документация по основному пакету

  Загрузите установщик основного пакета installer-core-v4.x.y-zzzz.tar. Здесь "zzzz" может быть x86, arm или hybrid, в зависимости от архитектуры, выбранной пользователем (то же самое относится и ниже).
  Распакуйте основной пакет с помощью команды tar:

  ```shell
  tar xvf installer-core-v4.x.y-zzzz.tar
  ```

  ### Документация по пакету расширений

  Загрузите пакет расширений installer-extensions-v4.x.y-zzzz.tar. Здесь "zzzz" должен соответствовать основному пакету. Пакет расширений содержит плагины, которые не включены в основной пакет.
  Если вы загрузили пакет расширений, вам необходимо распаковать пакет расширений в ту же директорию перед установкой:

  ```bash
  tar xvf installer-extensions-v4.x.y-zzzz.tar -C <директория распаковки основного пакета>
  ```

  Пример:
  Если основной пакет был распакован в /root, после распаковки в /root будет директория installer.
  Вы должны выполнить следующую команду в директории, где находится installer-extensions-v4.x.y-zzzz.tar: `tar xvf installer-extensions-v4.x.y-zzzz.tar -C /root/`
</Steps>

Руководство по установке подробно опишет процесс установки основного пакета. Документация по пакету расширений предоставит конкретные инструкции по использованию и настройке расширений.
