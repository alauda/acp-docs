# Архитектура

Платформа контейнеров Alauda (ACP) с объектным хранилищем MinIO - это высокопроизводительная распределенная система объектного хранилища, разработанная для облачно-ориентированных сред. Она использует кодирование с исправлением ошибок, распределенные хранилищные пулы и механизмы высокой доступности для обеспечения долговечности данных и масштабируемости в Kubernetes.

### Основные компоненты:

- **MinIO Operator**: Управляет развертыванием и обновлением кластеров MinIO.
- **MinIO Peer**: Конфигурирует и управляет функциональностью репликации сайтов MinIO.
- **MinIO Pool**: Ядро MinIO, ответственное за обработку запросов на объектное хранилище. Каждый пул соответствует StatefulSet и предоставляет ресурсы хранения.

### Архитектура развертывания:

Развертывание MinIO в Kubernetes требует определения арендатора MinIO, указания количества экземпляров серверов (подов) и количества томов (дисков) на экземпляр. Каждый сервер MinIO управляется через StatefulSet, что обеспечивает стабильные идентификаторы и постоянное хранилище. MinIO агрегирует все диски в один или несколько наборов кодирования с исправлением ошибок и применяет кодирование для обеспечения отказоустойчивости.

### Расширение Multi-Pool:

Кластеры MinIO могут масштабироваться путем добавления дополнительных серверных пулов, каждый из которых имеет свой набор кодирования с исправлением ошибок. Хотя это предоставляет большую емкость хранения, это вводит сложности в обслуживании кластера и снижает общую надежность кластера. Отказ в любом серверном пуле может сделать весь кластер MinIO недоступным, даже если другие пулы остаются работоспособными.

### Заключение:

MinIO - это высокомасштабируемое, облачно-ориентированное решение для объектного хранилища, которое балансирует производительность и надежность. При проектировании кластера MinIO крайне важно тщательно разработать хранилищные пулы, настроить параметры кодирования с исправлением ошибок и реализовать стратегии высокой доступности для обеспечения целостности данных и стабильности операций в средах Kubernetes.
