# Добавление устройств/классов устройств

## Добавление классов устройств

Унифицируйте классификацию устройств одного типа или жестких дисков с одинаковой бизнес-логикой в узлах кластера, настраивайте классы устройств в соответствии с потребностями хранения и выделяйте разные содержимое хранения различным типам дисков хранения.

### Примечания

Добавление классов устройств не поддерживается, когда тип кластера - **Расширяемый**.

### Процедура

<Steps>
  1. Войдите в **Управление платформой**.

  2. В левой панели навигации нажмите **Управление хранением** > **Распределенное хранилище**.

  3. Нажмите на вкладку **Классы устройств**.

  4. Нажмите **Добавить класс устройства** и настройте соответствующие параметры в соответствии с следующими инструкциями.

     | Параметр            | Описание                                                                                                                                                                                                                                                                                                                                            |
     | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
     | **Имя**             | Имя класса устройства. Следующие имена не могут быть использованы для класса устройства: `hdd`, `ssd`, `nvme`.                                                                                                                                                                                                                                           |
     | **Устройства хранения** | Выберите **Пустые диски** или **Неформатированные разделы дисков** в узле. <ul><li>Когда переключатель для всех пустых устройств включен: добавьте все пустые устройства в узле в этот класс устройства;</li><li>Когда переключатель для всех пустых устройств выключен: введите имена пустых устройств в узле вручную, например, `sda`.</li></ul> |
</Steps>

## Добавление устройств

Соотнесите доступные жесткие диски с устройствами хранения для использования и управления.

**Примечание**: После того как жесткие диски добавлены в качестве устройств хранения, обновление или удаление их через интерфейс не поддерживается.

### Процедура

<Steps>
  1. Войдите в **Управление платформой**.

  2. В левой панели навигации нажмите **Управление хранением** > **Распределенное хранилище**.

  3. Нажмите на вкладку **Классы устройств**.

  4. Справа от класса устройства нажмите **Добавить устройство**, и настройте соответствующие параметры в соответствии с следующими инструкциями.

     | Параметр            | Описание                                                                                                                                                                                                                                                                                                                                                                                              |
     | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
     | **Тип узла**        | Выберите тип узла, в котором находится жесткий диск, который вы хотите добавить в качестве устройства хранения. <br />**Вычислительный узел**: Узел, в который еще не добавлены устройства хранения. <br />**Храните узел**: Узел, в который были добавлены устройства хранения.                                                                                                                                                                            |
     | **Тип добавления**   | Выберите метод добавления жестких дисков в качестве устройств хранения. <br />**Все пустые диски**: Выберите добавление всех неразмеченных монтированных дисков в узле в качестве устройств хранения. <br />**Указанные диски**: Выберите добавление некоторых дисков в узле в качестве устройств хранения, включая пустые диски или уже размеченные монтированные диски. <br />Когда тип узла - **Хранилище узел**, можно выбрать только **Указанные диски**. |
     | **Указанные диски** | Когда тип добавления - **Указанные диски**, введите имена всех жестких дисков, которые нужно добавить в качестве устройств хранения, например, `sda`, `sdb`. После ввода каждого имени жесткого диска нажмите Enter для подтверждения. <br /><br /><b>Примечание</b>: Рекомендуется использовать весь жесткий диск в качестве устройства хранения, а не разделы на жестком диске.                                                                     |

  5. Нажмите **Добавить**.
</Steps>

## Статус жесткого диска

- **Нормальный**: Соответствующий статус устройства хранения - IN+UP.

- **Аномальный**: Соответствующий статус устройства хранения - IN+DOWN.

- **Оффлайн**: Соответствующий статус устройства хранения - OUT+UP.

- **Ошибка**: Соответствующий статус устройства хранения - OUT+DOWN.
