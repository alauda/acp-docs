# Добавление устройств/классов устройств

## Добавление классов устройств

Унифицируйте классификацию устройств одного типа или жестких дисков с одинаковой бизнес-логикой в узлах кластера, настраивайте классы устройств в соответствии с потребностями в хранении и выделяйте разные содержимое хранения для различных типов дисков хранения.

### Заметки

Добавление классов устройств не поддерживается, когда тип кластера **Extend**.

### Процедура

<Steps>
  1. Войдите в **Управление платформой**.

  2. В левом навигационном меню нажмите **Управление хранилищем** > **Распределенное хранилище**.

  3. Нажмите на вкладку **Классы устройств**.

  4. Нажмите **Добавить класс устройства** и настройте соответствующие параметры в соответствии с следующими инструкциями.

     | Параметр           | Описание                                                                                                                                                                                                                                                                                                                                            |
     | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
     | **Имя**            | Имя класса устройства. Следующие имена не могут использоваться для класса устройства: `hdd`, `ssd`, `nvme`.                                                                                                                                                                                                                                           |
     | **Устройства хранения** | Выберите **Пустые диски** или **Форматированные дисковые разделы** в узле. <ul><li>Когда переключатель для всех пустых устройств включен: добавьте все пустые устройства под узлом в этот класс устройства;</li><li>Когда переключатель для всех пустых устройств выключен: вручную введите имена пустых устройств под узлом, например, `sda`.</li></ul> |
</Steps>

## Добавление устройств

Сопоставьте доступные жесткие диски с устройствами хранения для использования и управления.

**Заметка**: После добавления жестких дисков в качестве устройств хранения обновление или удаление их через интерфейс не поддерживается.

### Процедура

<Steps>
  1. Войдите в **Управление платформой**.

  2. В левом навигационном меню нажмите **Управление хранилищем** > **Распределенное хранилище**.

  3. Нажмите на вкладку **Классы устройств**.

  4. Справа от класса устройства нажмите **Добавить устройство** и настройте соответствующие параметры в соответствии с следующими инструкциями.

     | Параметр           | Описание                                                                                                                                                                                                                                                                                                                                                                                              |
     | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
     | **Тип узла**       | Выберите тип узла, в котором расположен жесткий диск, который вы хотите добавить в качестве устройства хранения. <br />**Узел вычислений**: Узел, в который не добавлены устройства хранения. <br />**Узел хранения**: Узел, в который добавлены устройства хранения.                                                                                                                                                                            |
     | **Тип добавления**  | Выберите метод добавления жестких дисков в качестве устройств хранения. <br />**Все пустые диски**: Выберите, чтобы добавить все незапартиционированные смонтированные диски в узле в качестве устройств хранения. <br />**Указанные диски**: Выберите, чтобы добавить некоторые диски в узле в качестве устройств хранения, включая пустые диски или уже запартиционированные смонтированные диски. <br />Когда тип узла **Узел хранения**, можно выбрать только **Указанные диски**. |
     | **Указанные диски** | Когда тип добавления - **Указанные диски**, введите имена всех жестких дисков, которые будут добавлены в качестве устройств хранения, например, `sda`, `sdb`. После ввода каждого имени жесткого диска нажмите Enter для подтверждения. <br /><br /><b>Заметка</b>: Рекомендуется использовать весь жесткий диск в качестве устройств хранения, а не разделы на жестком диске.                                                                     |

  5. Нажмите **Добавить**.
</Steps>

## Статус жесткого диска

- **Нормальный**: Соответствующий статус устройства хранения IN+UP.

- **Аномальный**: Соответствующий статус устройства хранения IN+DOWN.

- **Оффлайн**: Соответствующий статус устройства хранения OUT+UP.

- **Неисправность**: Соответствующий статус устройства хранения OUT+DOWN.
