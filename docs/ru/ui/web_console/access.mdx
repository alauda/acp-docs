# Доступ к веб-консоли

## Введение

Веб-консоль платформы Alauda Container предоставляет централизованный графический интерфейс для управления контейнеризованными приложениями и инфраструктурой. Этот документ описывает методы доступа к веб-консоле и навигации между ее различными представлениями.

## Предварительные требования

Перед доступом к веб-консоли убедитесь, что у вас есть:

- Поддерживаемый веб-браузер (Chrome, Firefox, Safari или Edge)
- Сетевое соединение с конечной точкой веб-консоли платформы
- Действительные учетные данные для аутентификации на платформе

## Доступ к веб-консоли

### Доступ по URL

Веб-консоль можно открыть через ее URL, который, как правило, имеет следующий формат:

```
https://<console-route-hostname>
```

Обратитесь к вашему системному администратору, чтобы получить конкретный URL для вашей установки платформы Alauda Container.

### Аутентификация

После доступа по URL веб-консоле вам будет представлена страница входа. Введите свои учетные данные и выберите метод аутентификации, если настроено несколько методов.

Платформа поддерживает различные провайдеры аутентификации, которые могут включать:

- Локальная аутентификация
- LDAP/Active Directory
- OpenID Connect
- Провайдеры OAuth

После успешной аутентификации вы будете направлены на страницу Портала.

## Навигация по веб-консоли

### Страница Портала

Страница Портала служит начальной точкой доступа к веб-консоли, отображая карточки для всех доступных представлений. Каждая карточка представляет собой различную функциональную область платформы.

### Переключение между представлениями

Чтобы переключаться между представлениями:

1. Найдите переключатель представлений в верхнем левом углу консоли
2. Нажмите на переключатель представлений, чтобы увидеть доступные представления
3. Выберите желаемое представление из выпадающего меню

### Элементы интерфейса пользователя

Интерфейс веб-консоли включает несколько ключевых элементов:

- **Основное навигационное меню**: Находится с левой стороны, обеспечивает доступ к ресурсам, специфичным для представления
- **Переключатель представлений**: Находится в верхнем левом углу, позволяет переключаться между представлениями
- **Меню пользователя**: Находится в верхнем правом углу, предоставляет доступ к настройкам пользователя и выходу
- **Меню помощи**: Обеспечивает доступ к документации и ресурсам поддержки
