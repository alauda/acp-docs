# Доступ к веб-консоли

## Введение

Веб-консоль платформы Alauda Container предоставляет централизованный графический интерфейс для управления контейнеризованными приложениями и инфраструктурой. Этот документ описывает способы доступа к веб-консоли и навигации между ее различными представлениями.

## Предварительные требования

Перед доступом к веб-консоли убедитесь, что у вас есть:

- Поддерживаемый веб-браузер (Chrome, Firefox, Safari или Edge)
- Сетевое подключение к конечной точке веб-консоли платформы
- Действительные учетные данные для аутентификации на платформе

## Доступ к веб-консоли

### Доступ через URL

Веб-консоль можно получить через ее URL, который, как правило, имеет следующий формат:

```
https://<console-route-hostname>
```

Свяжитесь с вашим системным администратором, чтобы получить конкретный URL для вашей установки платформы Alauda Container.

### Аутентификация

При обращении к URL веб-консоли перед вами появится страница входа. Введите ваши учетные данные и выберите метод аутентификации, если настроено несколько методов.

Платформа поддерживает различные провайдеры аутентификации, которые могут включать:

- Локальная аутентификация
- LDAP/Active Directory
- OpenID Connect
- Провайдеры OAuth

После успешной аутентификации вы будете перенаправлены на страницу Портала.

## Навигация по веб-консоли

### Страница Портала

Страница Портала служит исходной точкой для веб-консоли, отображая карточки для всех доступных представлений. Каждая карточка представляет собой различную функциональную область платформы.

### Переключение между представлениями

Чтобы переключаться между представлениями:

1. Найдите переключатель представлений в верхнем левом углу консоли
2. Нажмите на переключатель представлений, чтобы увидеть доступные представления
3. Выберите желаемое представление из выпадающего меню

### Элементы пользовательского интерфейса

Интерфейс веб-консоли включает несколько ключевых элементов:

- **Главное меню навигации**: Находится с левой стороны, предоставляет доступ к ресурсам, специфичным для определенного представления
- **Переключатель представлений**: Находится в верхнем левом углу, позволяет переключаться между представлениями
- **Пользовательское меню**: Находится в верхнем правом углу, предоставляет доступ к настройкам пользователя и выходу из системы
- **Меню помощи**: Предоставляет доступ к документации и ресурсам поддержки
