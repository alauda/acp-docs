# Обзор

Веб-консоль платформы Alauda Container Platform предоставляет графический интерфейс для визуализации данных вашего проекта и выполнения административных, управленческих и диагностических задач. Веб-консоль разработана для улучшения пользовательского опыта, упорядочивая функциональность платформы по различным аспектам, которые соответствуют различным ролям пользователей и рабочим процессам.

## Главная страница

При входе в веб-консоль вы по умолчанию попадаете на главную страницу. Главная страница отображает все доступные карточки просмотра, поддерживаемые веб-консолей. Эти карточки служат точками доступа к различным функциональным областям платформы.

## Переключение представлений

Вы можете переключаться между различными представлениями, используя переключатель представлений, расположенный в верхнем левом углу веб-консоли. Это позволяет легко перемещаться между различными функциональными контекстами within the platform.

## Основные представления

Веб-консоль Alauda Container Platform предлагает несколько ключевых представлений, каждое из которых адаптировано для конкретных ролей пользователей и задач:

### Представление контейнерной платформы

Представление контейнерной платформы разработано для разработчиков. В этом представлении разработчики могут управлять ресурсами в своих проектах, такими как:

- Рабочие нагрузки приложений
- Объемы хранения
- Конфигурации
- Привязки службы
- Маршруты и сетевые ресурсы

### Представление управления платформой

Представление управления платформой ориентировано на администраторов. Это представление позволяет администраторам:

- Создавать или импортировать кластеры
- Устанавливать и управлять операторами
- Конфигурировать общеплатформенные настройки
- Мониторить состояние системы и метрики
- Управлять безопасностью платформы

### Представление управления проектом

Представление управления проектом сосредотачивается на администрировании на уровне проекта, позволяя пользователям:

- Создавать и конфигурировать проекты
- Управлять квотами проекта и лимитами ресурсов
- Администрировать участников проекта и привязки ролей
- Конфигурировать политики на уровне проекта

## Страницы обзора

Каждое представление обычно включает страницу обзора, которая предоставляет сводку релевантной информации. Например, страница обзора управления платформой отображает общеплатформенные данные, включая:

- Общее количество проектов
- Статус и здоровье кластеров
- Метрики использования ресурсов
- Последние события и оповещения
- Индикаторы здоровья платформы

Эти страницы обзора предоставляют наглядную информацию о статусе и производительности вашей контейнеризированной инфраструктуры, помогая пользователям быстро оценить состояние системы и выявить области, требующие внимания.
