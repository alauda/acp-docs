# Обзор

Веб-консоль платформы контейнеров Alauda обеспечивает графический интерфейс для визуализации данных вашего проекта и выполнения административных, управленческих и диагностических задач. Веб-консоль разработана для улучшения пользовательского опыта, организуя функциональные возможности платформы в разные перспективы, которые учитывают различные роли пользователей и рабочие процессы.

## Страница портала

При входе в веб-консоль вы по умолчанию попадаете на страницу портала. Страница портала отображает все доступные карточки представлений, поддерживаемые веб-консолью. Эти карточки служат точками входа в различные функциональные области платформы.

## Переключение представлений

Вы можете переключаться между разными представлениями, используя переключатель представлений, расположенный в верхнем левом углу веб-консоли. Это позволяет легко перемещаться между различными функциональными контекстами платформы.

## Основные представления

Веб-консоль платформы контейнеров Alauda предлагает несколько ключевых представлений, каждое из которых адаптировано для конкретных ролей пользователей и задач:

### Представление платформы контейнеров

Представление платформы контейнеров предназначено для разработчиков. В этом представлении разработчики могут управлять ресурсами в своих проектах, такими как:

- Рабочие нагрузки приложений
- Хранилища
- Конфигурации
- Связывание сервисов
- Маршруты и сетевые ресурсы

### Представление управления платформой

Представление управления платформой ориентировано на администраторов. Это представление позволяет администраторам:

- Создавать или импортировать кластеры
- Устанавливать и управлять операторами
- Настраивать настройки по всей платформе
- Мониторить здоровье и метрики системы
- Управлять безопасностью платформы

### Представление управления проектом

Представление управления проектом сосредоточено на администрировании на уровне проекта, позволяя пользователям:

- Создавать и настраивать проекты
- Управлять квотами проекта и ограничениями ресурсов
- Администрировать участников проекта и связывание ролей
- Настраивать политики на уровне проекта

## Страницы обзора

Каждое представление обычно включает страницу обзора, которая предоставляет краткое резюме актуальной информации. Например, страница обзора управления платформой отображает данные по всей платформе, включая:

- Общее количество проектов
- Статус и здоровье кластера
- Метрики использования ресурсов
- Последние события и тревоги
- Показатели здоровья платформы

Эти страницы обзора предлагают быстрые взгляды на статус и производительность вашей контейнеризированной инфраструктуры, помогая пользователям быстро оценивать здоровье системы и выявлять участки, требующие внимания.
