# Настройка веб-консоли

## Введение

Веб-консоль платформы Alauda Container может быть настроена для отражения брендинга и предпочтений вашей организации. В этом документе описаны различные варианты настройки, доступные администраторам.

## Предпочтения пользователя

### Настройки темы и языка

Отдельные пользователи могут настроить свой опыт работы с веб-консолей через меню пользователя, расположенное в верхнем правом углу консоли.

Чтобы изменить настройки темы и языка:

1. Нажмите на ваше имя пользователя в верхнем правом углу
2. Выберите "Предпочтения пользователя" в выпадающем меню
3. Перейдите в раздел "Тема" или "Язык"
4. Выберите желаемые настройки
5. Нажмите "Сохранить", чтобы применить изменения

**Примечание:** Если ваш предпочтительный язык не доступен в селекторе языков, вам может понадобиться установить соответствующий языковой плагин. Обратитесь к администратору вашей платформы за помощью в установке дополнительных языковых плагинов.

## Настройки администратора

Следующие настройки требуют прав администратора и могут быть доступны через вид управления платформой.

### Настройка страницы входа

Чтобы настроить страницу входа:

1. Перейдите к виду управления платформой
2. Выберите "Системные настройки" в меню навигации
3. Нажмите на "Страница входа"
4. Настройте следующие элементы:
   - Обновить логотип: Нажмите "Обновить логотип", чтобы загрузить изображение логотипа (рекомендуемый размер: 200px × 60px)
   - Обновить фоновое изображение: Нажмите "Обновить фоновое изображение", чтобы загрузить изображение фона (рекомендуемый размер: 1920px × 1080px)
5. Нажмите "Сохранить", чтобы применить изменения

### Настройка страницы портала

Чтобы настроить страницу портала:

1. Перейдите к виду управления платформой
2. Выберите "Системные настройки" в меню навигации
3. Нажмите на "Портал"
4. Настройте следующие элементы:
   - Главный заголовок: Введите текст, который будет отображаться в качестве основного заголовка
   - Подзаголовок: Введите текст, который будет отображаться в качестве подзаголовка
   - Верхнее изображение: Загрузите изображение, которое будет отображаться в верхней части страницы портала
   - Нижнее изображение: Загрузите изображение, которое будет отображаться в нижней части страницы портала
5. Нажмите "Сохранить", чтобы применить изменения

### Настройка брендинга платформы

Чтобы настроить элементы брендинга платформы:

1. Перейдите к виду управления платформой
2. Выберите "Системные настройки" в меню навигации
3. Нажмите на "Логотип"
4. Настройте следующие элементы:
   - Логотип платформы: Загрузите индивидуальный логотип для платформы (рекомендуемый размер: 180px × 40px)
   - Название платформы: Введите текст, который будет отображаться в качестве названия платформы
   - Favicon: Загрузите индивидуальный favicon (рекомендуемый размер: 32px × 32px)
5. Нажмите "Сохранить", чтобы применить изменения

### Настройка названий представлений

Чтобы настроить названия представлений:

1. Перейдите к виду управления платформой
2. Выберите "Системные настройки" в меню навигации
3. Нажмите на "Параметры платформы"
4. В разделе "Конфигурация продукта" найдите представление, которое вы хотите переименовать
5. Обновите поле "Имя" на желаемое название
6. Нажмите "Сохранить", чтобы применить изменения

Переименованное представление будет отображаться в переключателе представлений и на странице портала с новым названием.

## Интеграция внешних продуктов

Вы можете интегрировать URL-адреса внешних продуктов в платформу, которые будут отображаться в виде карточек на странице портала и в переключателе представлений.

Чтобы интегрировать внешний продукт:

1. Перейдите к виду управления платформой
2. Выберите "Системные настройки" в меню навигации
3. Нажмите на "Параметры платформы"
4. В разделе "Конфигурация продукта" нажмите "Интегрировать продукт"
5. Предоставьте следующую информацию:
   - Название: Название продукта, которое будет отображаться
   - Иконка: Загрузите иконку для продукта
   - URL: Введите URL-адрес внешнего продукта
   - Описание: Предоставьте краткое описание продукта
6. Нажмите "Отправить", чтобы завершить интеграцию

После интеграции внешний продукт будет отображаться в виде карточки на странице портала и в переключателе представлений. Нажатие на карточку или выбор представления перенаправит пользователей на указанный URL.

## Рекомендации по настройки

- Поддерживайте последовательный брендинг во всех настраиваемых элементах
- Используйте файлы изображений с соответствующими размерами и размерами файлов
- Убедитесь, что пользовательский текст ясен, краток и уместен для всех пользователей
- Тестируйте настройки в различных браузерах и на экранах разных размеров
- Учитывайте рекомендации по доступности при выборе цветов и уровней контраста
