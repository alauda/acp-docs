# Настройка веб-консоли

## Введение

Веб-консоль платформы Alauda Container может быть настроена в соответствии с брендингом и предпочтениями вашей организации. Данный документ описывает различные варианты настройки, доступные администраторам.

## Предпочтения пользователей

### Настройки темы и языка

Каждый пользователь может настроить свой опыт в веб-консоли через меню пользователя, расположенное в верхнем правом углу консоли.

Чтобы изменить настройки темы и языка:

1. Щелкните по своему имени пользователя в верхнем правом углу
2. Выберите "Предпочтения пользователя" из выпадающего меню
3. Перейдите в раздел "Тема" или "Язык"
4. Выберите предпочитаемые настройки
5. Нажмите "Сохранить", чтобы применить изменения

**Примечание:** Если ваш предпочитаемый язык недоступен в селекторе языка, возможно, вам потребуется установить соответствующий языковой плагин. Обратитесь к администратору платформы за помощью в установке дополнительных языковых плагинов.

## Настройки администратора

Следующие настройки требуют привилегий администратора и могут быть доступны через представление управления платформой.

### Настройка страницы входа

Чтобы настроить страницу входа:

1. Перейдите в представление управления платформой
2. Выберите "Системные настройки" из навигационного меню
3. Нажмите на "Страница входа"
4. Настройте следующие элементы:
   - Обновить логотип: нажмите "Обновить логотип", чтобы загрузить изображение пользовательского логотипа (рекомендуемый размер: 200px × 60px)
   - Обновить фоновое изображение: нажмите "Обновить фоновое изображение", чтобы загрузить изображение пользовательского фона (рекомендуемый размер: 1920px × 1080px)
5. Нажмите "Сохранить", чтобы применить изменения

### Настройка страницы портала

Чтобы настроить страницу портала:

1. Перейдите в представление управления платформой
2. Выберите "Системные настройки" из навигационного меню
3. Нажмите на "Портал"
4. Настройте следующие элементы:
   - Основной заголовок: введите текст, который будет отображаться в качестве основного заголовка
   - Подзаголовок: введите текст, который будет отображаться в качестве подзаголовка
   - Верхнее изображение: загрузите изображение, которое будет отображаться в верхней части страницы портала
   - Нижнее изображение: загрузите изображение, которое будет отображаться в нижней части страницы портала
5. Нажмите "Сохранить", чтобы применить изменения

### Настройка брендинга платформы

Чтобы настроить элементы брендинга платформы:

1. Перейдите в представление управления платформой
2. Выберите "Системные настройки" из навигационного меню
3. Нажмите на "Логотип"
4. Настройте следующие элементы:
   - Логотип платформы: загрузите пользовательский логотип для платформы (рекомендуемый размер: 180px × 40px)
   - Название платформы: введите текст, который будет отображаться как название платформы
   - Фавикон: загрузите пользовательский фавикон (рекомендуемый размер: 32px × 32px)
5. Нажмите "Сохранить", чтобы применить изменения

### Настройка названий представлений

Чтобы настроить названия представлений:

1. Перейдите в представление управления платформой
2. Выберите "Системные настройки" из навигационного меню
3. Нажмите на "Параметры платформы"
4. В разделе "Конфигурация продукта" найдите представление, которое вы хотите переименовать
5. Обновите поле "Название" с вашим предпочтительным названием
6. Нажмите "Сохранить", чтобы применить изменения

Переименованное представление будет отображаться в переключателе представлений и на странице портала с новым названием.

## Интеграция внешних продуктов

Вы можете интегрировать URL-адреса внешних продуктов в платформу, которые будут отображаться как карточки представлений на странице портала и в переключателе представлений.

Чтобы интегрировать внешний продукт:

1. Перейдите в представление управления платформой
2. Выберите "Системные настройки" из навигационного меню
3. Нажмите на "Параметры платформы"
4. В разделе "Конфигурация продукта" нажмите "Интегрировать продукт"
5. Укажите следующую информацию:
   - Название: название продукта, которое будет отображаться
   - Значок: загрузите значок для продукта
   - URL: введите URL-адрес внешнего продукта
   - Описание: предоставьте краткое описание продукта
6. Нажмите "Отправить", чтобы завершить интеграцию

После интеграции внешний продукт появится в виде карточки на странице портала и в переключателе представлений. Щелчок по карточке или выбор представления перенаправит пользователей на указанный URL.

## Рекомендации по настройке

- Поддерживайте согласованность брендинга для всех настроенных элементов
- Используйте файлы изображения с соответствующими размерами и объемами
- Убедитесь, что пользовательский текст ясен, краток и подходит для всех пользователей
- Тестируйте настройки на различных браузерах и размерах экранов
- Учитывайте рекомендации по доступности при выборе цветов и уровней контраста
