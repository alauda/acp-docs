# GatewayAPI

[GatewayAPI](https://gateway-api.sigs.k8s.io/) — это новый стандарт для входа в Kubernetes.

ALB также поддерживает GatewayAPI. Каждый ресурс Gateway будет преобразован в ресурс ALB.

Слушатель и маршрутизатор будут обрабатываться непосредственно в ALB. Они не будут преобразованы в `Frontend` и `Rule`.
