# GatewayAPI

[GatewayAPI](https://gateway-api.sigs.k8s.io/) является новым стандартом для входа в Kubernetes.

ALB также поддерживает GatewayAPI. Ресурс Gateway будет преобразован в ресурс ALB.

Слушатель и маршрутизатор будут обрабатываться непосредственно в ALB. Они не будут преобразовываться в Frontend и Rule.
