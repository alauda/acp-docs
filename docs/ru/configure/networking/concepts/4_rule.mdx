Мы определяем ресурс, называемый правилом, который используется для описания того, как экземпляр alb должен обрабатывать запрос на уровне 7.

## dslx

dslx это специализированный язык, который используется для описания критериев соответствия.

например, нижеприведенное правило соответствует запросу, который удовлетворяет всем следующим критериям:

- url начинается с /app-a или /app-b
- метод - post
- группа параметра url vip
- хост - \*.app.com
- заголовок location - east-1 или east-2
- есть cookie с именем uid
- IP-адреса источника находятся в пределах 1.1.1.1-1.1.1.100

```yaml
  dslx:                                     
  - type: METHOD
    values:
    - - EQ
      - POST
  - type: URL
    values:
    - - STARTS_WITH
      - /app-a
    - - STARTS_WITH
      - /app-b
  - type: PARAM
    key: group
    values:
    - - EQ
      - vip
  - type: HOST 
    values:
    - - ENDS_WITH
      - .app.com
  - type: HEADER
    key: LOCATION 
    values:
    - - IN
      - east-1
      - east-2
  - type: COOKIE
    key: uid
    values:
    - - EXIST 
  - type: SRC_IP
    values:
    - - RANGE
      - "1.1.1.1"
      - "1.1.1.100"
```
