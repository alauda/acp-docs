# Изоляция проекта

## Режим проекта

ALB может использоваться несколькими проектами, и эти проекты могут управлять этим ALB. Все порты ALB видны этим проектам.

## Режим портов проекта

Порт ALB может принадлежать различным проектам. Этот режим развертывания называется Режим портов проекта. Администратор должен указать сегмент портов, который может использовать каждый проект. Пользователи этого проекта могут создавать порты только в этом сегменте портов и могут видеть только порты в этом сегменте портов.

## Изоляция проекта

По умолчанию установлена изоляция проекта: каждый пользователь может видеть только правила своего проекта.
