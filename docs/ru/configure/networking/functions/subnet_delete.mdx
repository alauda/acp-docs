# Удалить подсеть

## Заметки

- Когда подсеть удаляется, если существуют группы контейнеров, использующие IP-адреса в пределах подсети, группы контейнеров могут продолжать работать, и IP-адреса останутся без изменений, но они не смогут взаимодействовать через сеть. Группы контейнеров могут быть восстановлены для использования IP-адресов в пределах подсети по умолчанию или назначить новую подсеть для пространства имен, в котором находятся группы контейнеров.

- Подсеть по умолчанию не может быть удалена.

## Шаги

1. Перейдите в **Управление платформой**.

2. В левой боковой панели навигации нажмите **Управление сетью** > **Подсети**.

3. Нажмите ⋮ > **Удалить**, и продолжите с удалением.
