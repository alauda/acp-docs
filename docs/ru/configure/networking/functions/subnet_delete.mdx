# Удаление подсети

## Примечания

- Когда подсеть удаляется, если есть группы контейнеров, использующие IP-адреса в пределах подсети, группы контейнеров могут продолжать работать, и IP-адреса останутся неизменными, но они не смогут общаться в сети. Группы контейнеров можно перестроить так, чтобы использовать IP-адреса в пределах подсети по умолчанию, или назначить новую подсеть пространству имен, в котором находятся группы контейнеров, для использования.

- Подсеть по умолчанию удалить нельзя.

## Шаги

1. Перейдите в **Управление платформой**.

2. В левой боковой панели навигации нажмите **Управление сетью** > **Подсети**.

3. Нажмите ⋮ > **Удалить** и продолжите с удалением.
