# Создание BGP-соседей

Узлы, устанавливающие соединения для обмена маршрутной информацией между разными АС или внутри одной АС, которые общаются по протоколу BGP.

## Терминология

| Термин           | Пояснение                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Номер АС**     | АС относится к совокупности маршрутизаторов, управляемых одной и той же технической административной организацией, использующей единые маршрутизационные политики. Каждой АС в сети BGP присваивается уникальный номер АС для отличия от других АС. Номера АС делятся на номера 2 байта и 4 байта. <ul><li>Диапазон номеров 2 байта составляет 1~~65535, где 1-64511 - это зарегистрированные публичные номера АС в Интернете, аналогичные публичным IP-адресам; 64512~~65535 - это частные номера АС, аналогичные частным IP-адресам.</li><li>Диапазон номеров 4 байта составляет 1\~4294967295.</li></ul> Устройства, поддерживающие номера АС 4 байта, могут быть совместимы с устройствами, поддерживающими номера АС 2 байта. |

## Предварительные требования

Пожалуйста, свяжитесь с администратором для включения соответствующих функций.

## Шаги

1. Перейдите в **Управление платформой**.

2. В левой навигационной панели нажмите **Управление сетью** > **BGP-соседи**.

3. Нажмите **Создать BGP-соседа**.

4. Ознакомьтесь с инструкциями ниже для настройки параметров.

   | Параметр                | Описание                                                                                                                                                                                                                                                                                                        |
   | ----------------------   | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
   | **Локальный номер АС**   | Номер АС, к которой принадлежит узел, подключенный по BGP.<br /><br /><b>Примечание</b>: Если нет специальных требований, рекомендуется использовать конфигурацию IBGP, что означает, что локальный номер АС должен совпадать с номером соседней АС.                                                              |
   | **Номер соседней АС**    | Номер АС, к которой принадлежит сосед BGP.                                                                                                                                                                                                                                                                |
   | **IP-соседа**           | IP-адрес соседа BGP, который должен быть действительным IP-адресом, способным установить соединение BGP.                                                                                                                                                                                                         |
   | **Локальный IP**        | IP-адрес узла, подключённого по BGP. Если у узла, подключённого по BGP, несколько IP-адресов, выберите указанный локальный IP для установки соединения BGP с соседом.                                                                                                                                                 |
   | **Порт соседа**         | Номер порта соседа BGP.                                                                                                                                                                                                                                                                                   |
   | **Узел, подключённый по BGP** | Узел, который устанавливает соединение BGP. Если этот параметр не настроен, все узлы будут устанавливать соединение BGP.                                                                                                                                                                                       |
   | **eBGP Multi-Hop**      | Позволяет устанавливать BGP-сессии между маршрутизаторами BGP, которые не подключены напрямую. Когда эта функция включена, значение TTL пакетов BGP по умолчанию составляет 5, что позволяет устанавливать отношения соседства BGP через несколько промежуточных сетевых устройств, делая проектирование сети более гибким.     |
   | **RouterID**            | 32-битное числовое значение (обычно представлено в десятичном формате с точками, аналогично формату IPv4), используемое для уникальной идентификации маршрутизатора BGP в сети BGP, обычно используется для установления отношений соседства BGP, обнаружения маршрутизируемых петель, выбора оптимальных маршрутов и устранения неполадок в сети. |

5. Нажмите **Создать**.
