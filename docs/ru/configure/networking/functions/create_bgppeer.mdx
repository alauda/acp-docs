# Создание BGP-соседей

Узлы, которые устанавливают соединения для обмена маршрутной информацией как между различными AS, так и внутри одной AS, общаются по протоколу BGP.

## Терминология

| Термин          | Объяснение                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Номер AS**    | AS относится к группе маршрутизаторов, управляемых одной и той же технической административной организацией, использующей единую маршрутную политику. Каждой AS в сети BGP присваивается уникальный номер AS для ее отличия от других AS. Номера AS делятся на 2-байтовые и 4-байтовые номера AS. <ul><li>Диапазон 2-байтовых номеров AS составляет 1~~65535, где 1-64511 — это зарегистрированные публичные номера AS в Интернете, аналогичные публичным IP-адресам; 64512~~65535 — это частные номера AS, аналогичные частным IP-адресам.</li><li>Диапазон 4-байтовых номеров AS составляет 1\~4294967295.</li></ul> Устройства, поддерживающие 4-байтовые номера AS, могут быть совместимы с устройствами, поддерживающими 2-байтовые номера AS. |

## Предварительные условия

Пожалуйста, свяжитесь с администратором для включения соответствующих функций.

## Шаги

1. Войдите в **Управление платформой**.

2. В левой навигационной панели нажмите **Управление сетью** > **BGP-соседи**.

3. Нажмите **Создать BGP-соседа**.

4. Ознакомьтесь с инструкциями ниже для настройки параметров.

   | Параметр              | Описание                                                                                                                                                                                                                                                                                                        |
   | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
   | **Локальный номер AS** | Номер AS, к которому принадлежит узел, подключенный по BGP.<br /><br /><b>Примечание</b>: Если нет специальных требований, рекомендуется использовать конфигурацию IBGP, что означает, что локальный номер AS должен совпадать с номером AS соседа.                                                              |
   | **Номер AS соседа**   | Номер AS, к которому принадлежит BGP-сосед.                                                                                                                                                                                                                                                                |
   | **IP-соседа**         | IP-адрес BGP-соседа, который должен быть действительным IP-адресом, способным установить соединение BGP.                                                                                                                                                                                                         |
   | **Локальный IP**      | IP-адрес узла, подключенного по BGP. Когда у узла, подключенного по BGP, несколько IP-адресов, выберите указанный локальный IP для установления соединения BGP с соседом.                                                                                                                                                 |
   | **Порт соседа**       | Номер порта BGP-соседа.                                                                                                                                                                                                                                                                                   |
   | **Подключенный узел BGP** | Узел, который устанавливает соединение BGP. Если этот параметр не настроен, все узлы будут устанавливать соединения BGP.                                                                                                                                                                                       |
   | **eBGP Multi-Hop**    | Позволяет устанавливать BGP-сессии между маршрутизаторами BGP, которые не подключены напрямую. Когда эта функция включена, значение TTL по умолчанию для пакетов BGP составляет 5, что позволяет устанавливать отношения BGP-соседей через несколько промежуточных сетевых устройств, делая проектирование сети более гибким.     |
   | **RouterID**          | 32-битовое числовое значение (обычно представлено в десятичном формате с точками, аналогично формату IPv4-адреса), используемое для уникальной идентификации маршрутизатора BGP в сети BGP, обычно используется для установления отношений соседей BGP, обнаружения маршрутных петель, выбора оптимальных путей и устранения неполадок в сети. |

5. Нажмите **Создать**.
