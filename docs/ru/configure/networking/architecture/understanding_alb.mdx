# Понимание ALB

ALB (Another Load Balancer) — это шлюз Kubernetes, работающий на базе OpenResty, имеющий многолетний опыт эксплуатации от Alauda.

## Основные компоненты

![](../../../../en/configure/networking/assets/alb_architecture.png)

- **ALB Operator**: Оператор, который управляет жизненным циклом экземпляров ALB. Он отвечает за наблюдение за ALB CR и затем создает и обновляет экземпляры ALB для различных арендаторов.
- **ALB Instance**: Экземпляр ALB включает в себя Openresty, который действует как план данных, и контроллер на Go, который служит планом управления.
  Контроллер на Go мониторит различные CR (Ingress, Gateway, Rule и т. д.) и преобразует их в специфические для ALB DSL правила. OpenResty затем использует эти DSL правила для сопоставления и обработки входящих запросов.
