# Понимание ALB

ALB (Another Load Balancer) — это шлюз Kubernetes, работающий на базе OpenResty, который имеет многолетний опыт эксплуатации от Alauda.

## Основные компоненты

![](../../../../en/configure/networking/assets/alb_architecture.png)

- **ALB Оператор**: Оператор, который управляет жизненным циклом экземпляров ALB. Он отвечает за отслеживание ALB CR и последующее создание и обновление экземпляров ALB для различных арендаторов.
- **ALB Экземпляр**: Экземпляр ALB включает Openresty, который функционирует как уровень данных, и контроллер на Go, который является управляющим уровнем. Контроллер на Go отслеживает различные CR (Ingress, Gateway, Rule и т.д.) и преобразует их в специальные DSL-правила для ALB. OpenResty затем использует эти DSL-правила для сопоставления и обработки входящих запросов.
