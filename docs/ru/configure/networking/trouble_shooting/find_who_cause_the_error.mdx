# Найти Причину Ошибки

Поле `X-ALB-ERR-REASON` в заголовке ответа на ошибочный запрос указывает причину ошибки.

Причины ошибки могут быть следующими:

```bash
InvalidBalancer : no balancer found for xx # это означает, что для службы не найден ни один конечный узел
BackendError : read xxx byte data from backend # это означает, что бэкенд действительно ответил, код ошибки не вызван alb.
InvalidUpstream : no rule match # это означает, что запрос не соответствует ни одному правилу, поэтому alb возвращает 404.
```
