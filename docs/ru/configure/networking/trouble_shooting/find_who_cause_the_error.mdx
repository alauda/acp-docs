# Найдите, кто вызывает ошибку

Поле `X-ALB-ERR-REASON` в заголовке ответа на запрос с ошибкой указывает причину ошибки.

Причина ошибки может быть следующей:

```bash
InvalidBalancer : не найден балансировщик для xx # это значит, что для сервиса не найден конечный пункт
BackendError : прочитаны xxx байт данных из бэкенда # это значит, что бэкенд действительно ответил, код ошибки не вызван alb.
InvalidUpstream : правило не совпадает # это значит, что запрос не соответствует ни одному правилу, поэтому alb возвращает 404.
```
