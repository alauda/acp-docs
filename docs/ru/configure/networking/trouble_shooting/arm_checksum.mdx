# Как решить проблемы с межузловой связью в средах ARM?

При использовании более низких версий ядра и определенных отечественных сетевых карт может возникнуть проблема, когда сетевой адаптер неверно вычисляет контрольные суммы после включения аппаратного ускорения вычисления контрольных сумм (Checksum Offload). Это может привести к сбоям в связи между узлами в сети Overlay Kube-OVN. Конкретные решения следующие:

- **Решение 1: Обновление версии ядра**. Рекомендуется обновить версию ядра до 4.19.90-25.16.v2101 или более поздней версии.

- **Решение 2: Отключение аппаратного ускорения контактных сумм**. Если обновление версии ядра невозможно немедленно и возникают проблемы с межузловой связью, вы можете отключить аппаратное ускорение вычисления контрольных сумм для физической сетевой карты с помощью следующей команды.

  ```
  ethtool -K eth0 tx off
  ```
