# Обновление и удаление балансировщика нагрузки

## Обновление балансировщика нагрузки

**Примечание**: Обновление балансировщика нагрузки приведет к прерыванию обслуживания на 3-5 минут. Пожалуйста, выберите подходящее время для этой операции!

**Шаги для выполнения**

1. Войдите в **Управление платформой**.

2. В левом боковом меню нажмите **Управление сетью** > **Балансировщик нагрузки**.

3. Нажмите ⋮ > **Обновить**.

4. Обновите конфигурацию сети и ресурсов по мере необходимости.

   - Пожалуйста, разумно задавайте характеристики в соответствии с бизнес-потребностями. Вы также можете обратиться к соответствующему [Выбору и планированию балансировщика нагрузки](../how_to/decide_performance_selection.mdx) для получения рекомендаций.

   - **Внутренняя маршрутизация** поддерживает обновление только из состояния **Отключено** в состояние **Включено**.

5. Нажмите **Обновить**.

## Удаление балансировщика нагрузки

**Примечание**: После удаления балансировщика нагрузки связанные порты и правила также будут удалены и не могут быть восстановлены.

**Шаги для выполнения**

1. Войдите в **Управление платформой**.

2. В левом боковом меню нажмите **Управление сетью** > **Балансировщик нагрузки**.

3. Нажмите ⋮ > **Удалить** и подтвердите.
