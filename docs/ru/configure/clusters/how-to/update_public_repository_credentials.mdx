# Обновление учетных данных публичного репозитория

## Обзор

`Публичный репозиторий` — это сервис реестра изображений, предоставленный платформой и доступный в публичной сети интернет. Когда вы хотите, чтобы ваши кластеры использовали `Публичный репозиторий` в качестве своего реестра изображений, вам необходимо обновить встроенные учетные данные Cloud `public-registry-credential`. Это гарантирует, что у вашей платформы есть разрешение на загрузку изображений из публичного репозитория.

## Процедура

1. Войдите в **Портал Клиента** и загрузите файл аутентификации вашей организации из раздела **Управление Предприятием**, расположенного в выпадающем меню **Информация о Пользователе** в верхнем правом углу.

2. Перейдите в раздел **Кластеры** > **Cloud Credential** в левой навигационной панели консоли **Управления Платформой**.

3. Найдите учетные данные облака с именем `public-registry-credential` и выберите **Обновить** из выпадающего меню справа.

4. В разделе **Загрузить адрес публичного репозитория** загрузите файл аутентификации, который вы скачали с **Портала Клиента**.

5. Нажмите **Обновить**, чтобы применить изменения.
