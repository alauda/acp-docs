# Обновление учетных данных публичного репозитория

## Обзор

`Публичный репозиторий` — это сервис регистрации изображений, предоставленный платформой, доступный в публичном интернете. Когда вы хотите, чтобы ваши кластеры использовали `Публичный репозиторий` в качестве своего реестра изображений, вам необходимо обновить встроенные учетные данные Cloud `public-registry-credential`. Это гарантирует, что ваша платформа имеет разрешение на загрузку изображений из публичного реестра.

## Процедура

1. Войдите в **Портал клиента** и скачайте файл аутентификации вашей организации из раздела **Управление предприятием**, расположенного в выпадающем списке **Информация о пользователе** в правом верхнем углу.

2. Перейдите в **Кластеры** > **Cloud Credential** на левой боковой панели консоли **Управление платформой**.

3. Найдите учетные данные облака с именем `public-registry-credential` и нажмите **Обновить** в выпадающем меню справа.

4. В разделе **Загрузить адрес публичного репозитория** загрузите файл аутентификации, который вы скачали из **Портала клиента**.

5. Нажмите **Обновить**, чтобы применить изменения.
