# Конфигурация Feature Gate \{#feature\_toggles}

## Обзор

Вы можете настроить, какие функции включены или отключены на платформе через веб-консоль. Feature gates позволяют вам контролировать, отображается ли функциональность в веб-консоли или как ведут себя соответствующие API. То, влияет ли gate только на видимость интерфейса или также изменяет поведение API, зависит от реализации каждого конкретного feature gate. Для получения подробной информации, пожалуйста, обратитесь к документации по каждой функции.

## Процедура

### Доступ к странице Feature Gate

Получите доступ к feature gates по адресу `{platform-access-address}/console-platform/feature-gate`. Например, если ваш адрес доступа к платформе `https://demo.example.com/console-portal/`, страница feature gate будет `https://demo.example.com/console-platform/feature-gate`.

### Отключение всех Alpha функций

Чтобы отключить все Alpha функции на платформе:

1. Перейдите в раздел "Настройки этапа"
2. Отключите gate "Alpha Feature"

Beta функции также можно управлять подобным образом через раздел Настройки этапа.

### Включение конкретных функций для одного кластера

Чтобы включить конкретные функции для отдельного кластера:

1. Найдите список gate с отображаемым названием вашего целевого кластера в качестве заголовка
2. Включите конкретные функции, которые вам нужны, в этом списке

### Отключение конкретных функций глобально

Чтобы отключить конкретные функции для всех кластеров:

1. Перейдите в раздел "Системные настройки"
2. Отключите gates для функций, которые вы хотите отключить глобально
