# Конфигурация переключателей функций \{#feature\_toggles}

## Обзор

Вы можете настроить, какие функции включены или отключены на платформе, через веб-консоль. Переключатели функций позволяют вам контролировать, видима ли функциональность в веб-консоли или как себя ведут соответствующие API. То, затрагивает ли переключатель только видимость интерфейса или также изменяет поведение API, зависит от реализации каждого конкретного переключателя функций. Для получения подробной информации, пожалуйста, обратитесь к документации по каждой функции.

## Процедура

### Доступ к странице переключателей функций

Получите доступ к переключателям функций по адресу `{platform-access-address}/console-platform/feature-gate`. Например, если ваш адрес доступа к платформе `https://demo.example.com/console-portal/`, то страница переключателей функций будет `https://demo.example.com/console-platform/feature-gate`.

### Отключение всех Alpha функций

Чтобы отключить все Alpha функции на всей платформе:

1. Перейдите в раздел "Настройки этапа"
2. Отключите переключатель "Alpha функция"

Функции Beta можно аналогично управлять через раздел Настройки этапа.

### Включение конкретных функций для одного кластера

Чтобы включить конкретные функции для отдельного кластера:

1. Найдите список переключателей с отображаемым именем вашего целевого кластера в качестве заголовка
2. Включите конкретные функции, которые вам нужны, в этом списке

### Отключение конкретных функций глобально

Чтобы отключить конкретные функции для всех кластеров:

1. Перейдите в раздел "Системные настройки"
2. Отключите переключатели для функций, которые вы хотите отключить глобально
