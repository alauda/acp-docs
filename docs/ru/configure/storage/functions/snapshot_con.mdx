# Развертывание компонента снимка тома

Снимок тома относится к снимку постоянного тома, который является копией постоянного тома в конкретный момент времени. Если кластер использует постоянные тома, поддерживающие функциональность снимков, компонент снимка тома может быть развернут для активации этой функции.

В настоящее время платформа поддерживает создание снимков томов только для PVC, которые **создаются динамически** с использованием классов хранения. Вы можете создать новые привязки PVC на основе этих снимков.

**Совет**: Режимы доступа, поддерживаемые при создании PVC из снимков, отличаются от тех, которые поддерживаются при создании PVC с использованием классов хранения, которые указаны **жирным шрифтом** в таблице ниже.

| Используемый класс хранения для создания снимков томов | Чтение-запись для одного узла (RWO) | Только чтение для нескольких узлов (ROX) | Чтение-запись для нескольких узлов (RWX) |
| ----------------------------------------------------- | ------------------------------------ | -------------------------------------- | ---------------------------------------- |
| **TopoLVM**                                           | Поддерживается                       | Не поддерживается                      | Не поддерживается                        |
| **CephRBD Block Storage**                             | Поддерживается                       | Не поддерживается                      | Не поддерживается                        |
| **CephFS File Storage**                               | Поддерживается                       | **Поддерживается**                    | Поддерживается                            |

## Процедура

1. Перейдите в **Управление платформой**.

2. В левой боковой панели навигации нажмите **Управление хранилищем** > **Снимки томов**.

3. Нажмите **Быстрое развертывание**, что перенаправит вас к плагину кластера.

4. Нажмите ⋮ > **Развернуть** рядом с **Контроллером снимков** и подождите некоторое время, пока развертывание не будет успешным.
