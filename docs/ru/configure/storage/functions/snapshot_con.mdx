# Развертывание компонента снимка объема

Снимок объема относится к снимку постоянного объема, который является копией постоянного объема в определенный момент времени. Если кластер использует постоянные объемы, которые поддерживают функцию снимков, компонент снимка объема может быть развернут для активации этой функции.

В настоящее время платформа поддерживает создание снимков объемов только для PVC, которые автоматически создаются с использованием классов хранения. Вы можете создавать новые связывания PVC на основе этих снимков.

**Совет**: Режимы доступа, поддерживаемые при создании PVC из снимков, отличаются от тех, что поддерживаются при создании PVC с использованием классов хранения, которые указаны **жирным шрифтом** в таблице ниже.

| Класс хранения, используемый для создания снимков объемов | Чтение-запись для одного узла (RWO) | Чтение только для нескольких узлов (ROX) | Чтение-запись для нескольких узлов (RWX) |
| -------------------------------------------------------- | ----------------------------------- | -------------------------------------- | --------------------------------------- |
| **TopoLVM**                                              | Поддерживается                      | Не поддерживается                      | Не поддерживается                       |
| **CephRBD блочное хранилище**                           | Поддерживается                      | Не поддерживается                      | Не поддерживается                       |
| **CephFS файловое хранилище**                           | Поддерживается                      | **Поддерживается**                    | Поддерживается                           |

## Шаги для выполнения операций

1. Перейдите в **Управление платформой**.

2. В левой навигационной панели щелкните по **Управлению хранилищем** > **Снимки объемов**.

3. Щелкните по **Быстрое развертывание**, что перенаправит вас к плагину кластера.

4. Щелкните на ⋮ > **Развернуть** рядом с **Контроллером снимков** и подождите, пока развертывание не будет завершено успешно.
