# Развертывание компонента снимка тома

Снимок тома относится к снимку постоянного тома, который является копией постоянного тома в определенный момент времени. Если кластер использует постоянные тома, которые поддерживают функциональность снимков, компонент снимка тома может быть развернут для включения этой функции.

В настоящее время платформа поддерживает создание снимков томов только для PVC, которые **создаются динамически** с использованием классов хранения. Вы можете создавать новые привязки PVC на основе этих снимков.

**Совет**: Режимы доступа, поддерживаемые при создании PVC из снимков, отличаются от тех, которые поддерживаются при создании PVC с использованием классов хранения, что указано **жирным шрифтом** в таблице ниже.

| Класс хранения, используемый для создания снимков томов | Чтение-запись для одного узла (RWO) | Чтение только для нескольких узлов (ROX) | Чтение-запись для нескольких узлов (RWX) |
| ------------------------------------------------------ | ----------------------------------- | --------------------------------------- | --------------------------------------- |
| **TopoLVM**                                            | Поддерживается                      | Не поддерживается                       | Не поддерживается                       |
| **CephRBD Block Storage**                              | Поддерживается                      | Не поддерживается                       | Не поддерживается                       |
| **CephFS File Storage**                                | Поддерживается                      | **Поддерживается**                     | Поддерживается                          |

## Шаги для выполнения

1. Перейдите в **Управление платформой**.

2. В левой навигационной панели нажмите на **Управление хранилищем** > **Снимки томов**.

3. Нажмите на **Быстрое развертывание**, что перенаправит вас к плагину кластера.

4. Нажмите на ⋮ > **Развернуть** рядом с **Контроллером снимков** и подождите некоторое время, пока развертывание не будет успешным.
