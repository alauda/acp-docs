# Удалить/Убрать пространство имен

**Операции с пространством имен**:

Вы можете либо навсегда удалить пространство имен, либо убрать его из текущего проекта.

**Удалить пространство имен**: Навсегда удаляет пространство имен и все связанные с ним ресурсы (например, рабочие нагрузки, сервисы), освобождая выделенные квоты. Это действие необратимо.

**Убрать пространство имен**: Убирает пространство имен из проекта, не затрагивая его ресурсы на уровне кластера, включая приложения, постоянные тома, конфигурации и т.д.

Удаленное пространство имен может быть импортировано в другие проекты, связанные с его целевым кластером, с помощью функции **Импорт пространства имен**. Подробности [Импорт пространства имен](./import_namespace.mdx).
