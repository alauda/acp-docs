# Удалить/Удалить пространство имен

**Операции с пространствами имен**:

Вы можете либо навсегда удалить пространство имен, либо удалить его из текущего проекта.

**Удалить пространство имен**: П permanently удаляет пространство имен и все связанные с ним ресурсы (например, рабочие нагрузки, сервисы), освобождая выделенные квоты. Это действие необратимо.

**Удалить пространство имен**: Удаляет пространство имен из проекта, не затрагивая его ресурсы на уровне кластера, включая приложения, постоянные тома, конфигурации и т.д.

Удаленное пространство имен может быть импортировано в другие проекты, связанные с его целевым кластером, с помощью функции **Импорт пространства имен**. Для получения подробной информации [Импортировать пространство имен](./import_namespace.mdx).
