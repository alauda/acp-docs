# Введение

Смотрите официальную документацию Kubernetes: [Namespaces](https://kubernetes.io/en/docs/concepts/overview/working-with-objects/namespaces/)

> В Kubernetes пространства имен предоставляют механизм для изоляции групп ресурсов в рамках одного кластера. Имена ресурсов должны быть уникальными в пределах одного пространства имен, но не по всему кластеру. Области видимости, основанные на пространствах имен, применимы только для объектов с пространством имен (например, Deployments, Services и т. д.), а не для объектов, действующих на уровне кластера (например, StorageClass, Nodes, PersistentVolumes и т. д.).
