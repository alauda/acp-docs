# Основные концепции

## Изоляция узлов

Изоляция узлов относится к изоляции узлов в кластере с целью предотвращения одновременного использования одного и того же узла контейнерами из разных проектов, тем самым избегая конкуренции за ресурсы и ухудшения производительности.
