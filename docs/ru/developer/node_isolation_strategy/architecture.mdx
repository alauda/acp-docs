# Архитектура

![Архитектура стратегии изоляции узлов](../../../en/developer/node_isolation_strategy/assets/node-isolation.png "Архитектура стратегии изоляции узлов")

Стратегия изоляции узлов реализована на основе компонента Cluster Core платформы контейнеров, обеспечивая возможность изоляции узлов между проектами путем выделения узлов на каждом кластере рабочей нагрузки. Когда контейнеры создаются в проекте, они жестко планируются на узлы, выделенные для этого конкретного проекта.
