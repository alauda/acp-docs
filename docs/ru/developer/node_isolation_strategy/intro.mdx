# Введение

Стратегия изоляции узлов предоставляет стратегию изоляции узлов на уровне проекта, которая позволяет проектам эксклюзивно использовать узлы кластера.

## Преимущества

Удобное распределение узлов между проектами в эксклюзивном или совместном режиме, предотвращая конфликт ресурсов между проектами.

## Сценарии применения

Стратегия изоляции узлов подходит для сценариев, где требуется усиленная изоляция ресурсов между проектами, и где существует желание предотвратить использование узлов компонентами других проектов, что может привести к ограничениям ресурсов или невозможности удовлетворить требования к производительности.
