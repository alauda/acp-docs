# Введение

Стратегия изоляции узлов предоставляет стратегию изоляции узлов на уровне проектов, позволяя проектам исключительно использовать узлы кластера.

## Преимущества

Удобное распределение узлов между проектами в эксклюзивном или совместном режиме, предотвращая конкуренцию за ресурсы между проектами.

## Сценарии применения

Стратегия изоляции узлов подходит для сценариев, в которых требуется усиленная изоляция ресурсов между проектами, а также для предотвращения использования узлов компонентами других проектов, что может привести к ограничению ресурсов или невозможности удовлетворить требования по производительности.
