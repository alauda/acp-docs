# Введение

**ACP Builds** — это облачный контейнерный инструмент, предоставляемый <Term name="product" textCase="capitalize" /> и интегрирующий возможности Source to Image (S2I) с автоматизированными конвейерами. Он ускоряет переход предприятий к облачным технологиям, позволяя создавать полностью автоматизированные CI/CD конвейеры, поддерживающие несколько языков программирования, включая Java, Go, Python и Node.js. Кроме того, ACP Builds предлагает визуальное управление версиями и бесшовную интеграцию с инструментами, поддерживающими Kubernetes, такими как Helm и GitOps, обеспечивая эффективное управление жизненным циклом приложений с момента разработки до развертывания.

## Концепция Source to Image

Source to Image (S2I) — это инструмент и рабочий процесс для построения воспроизводимых контейнерных образов из исходного кода. Он внедряет исходный код приложения в заранее определенное базовое изображение и автоматически выполняет такие шаги, как компиляция и упаковка, в конечном итоге создавая исполняемый контейнерный образ. Это позволяет разработчикам сосредоточиться на разработке бизнес-кода, не беспокоясь о деталях контейнеризации.

## Основные функции

ACP Builds упрощает полный стек облачного рабочего процесса от кода до приложения, позволяя создавать сборки на нескольких языках и визуально управлять версиями. Он использует возможности, поддерживающие Kubernetes, для преобразования исходного кода в исполняемые контейнерные образы, гарантируя бесшовную интеграцию в комплексную облачную платформу.

- **Сборка на нескольких языках**: Поддерживает разработку приложений на различных языках программирования, таких как Java, Go, Python и Node.js, удовлетворяя разнообразные требования к разработке.
- **Визуальный интерфейс**: Обеспечивает интуитивно понятный интерфейс, который позволяет легко создавать, настраивать и управлять задачами сборки без глубоких технических знаний.
- **Управление полным жизненным циклом**: Охватывает весь жизненный цикл, от коммита кода до развертывания приложения, автоматизируя сборку, развертывание и управление эксплуатацией.
- **Глубокая интеграция**: Бесшовно интегрируется с вашим продуктом Container Platform, обеспечивая бесперебойный опыт разработки.
- **Высокая расширяемость**: Поддерживает пользовательские плагины и расширения, чтобы удовлетворить ваши конкретные потребности.

## Основные преимущества

- **Ускоренная разработка**: Оптимизирует процесс сборки, ускоряя доставку приложений.
- **Повышенная гибкость**: Поддерживает разработку на нескольких языках программирования.
- **Повышенная эффективность**: Автоматизирует процессы сборки и развертывания, уменьшая необходимость в ручном вмешательстве.
- **Увеличенная надежность**: Предоставляет подробные журналы сборок и визуальный мониторинг для облегчения устранения неполадок.

## Сценарии применения

Основные сценарии применения S2I следующие:

- Веб-приложения

  S2I поддерживает различные языки программирования, такие как Java, Go, Python и Node.js. Используя возможности управления приложениями <Term name="product" textCase="capitalize" />, он позволяет быстро создавать и развертывать веб-приложения, просто введя URL-адрес репозитория кода.

- CI/CD

  S2I бесшовно интегрируется с конвейерами DevOps, используя инструменты, поддерживающие Kubernetes, такие как Helm и GitOps, для автоматизации процессов сборки и развертывания образов. Это позволяет осуществлять непрерывную интеграцию и непрерывное развертывание приложений.

## Ограничения использования

> Текущая версия поддерживает только языки Java, Go, Python и Node.js.
