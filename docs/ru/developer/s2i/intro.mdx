# Введение

**ACP Builds** — это облачный контейнерный инструмент, предоставленный <Term name="product" textCase="capitalize" />, который интегрирует возможности Source to Image (S2I) с автоматизированными конвейерами. Он ускоряет бизнес-процессы на основе облачных технологий, позволяя полностью автоматизированные CI/CD конвейеры, которые поддерживают несколько языков программирования, включая Java, Go, Python и Node.js. Кроме того, ACP Builds предлагает визуальное управление релизами и бесшовную интеграцию с инструментами, специфичными для Kubernetes, такими как Helm и GitOps, обеспечивая эффективное управление жизненным циклом приложений от разработки до производства.

## Концепция Source to Image

Source to Image (S2I) — это инструмент и рабочий процесс для создания воспроизводимых контейнерных изображений из исходного кода. Он внедряет исходный код приложения в предварительно определенное изображение-строитель и автоматически завершает такие этапы, как компиляция и упаковка, в конечном итоге генерируя исполняемое контейнерное изображение. Это позволяет разработчикам сосредоточиться больше на разработке бизнес-кода, не беспокоясь о деталях контейнеризации.

## Основные возможности

ACP Builds облегчает полный стек облачного рабочего процесса от кода до приложения, позволяя создавать многомовленные сборки и визуально управлять релизами. Он использует возможности, специфичные для Kubernetes, для преобразования исходного кода в исполняемые контейнерные изображения, обеспечивая бесшовную интеграцию в комплексную облачную платформу.

- **Многомовленные сборки**: Поддерживает создание приложений на различных языках программирования, таких как Java, Go, Python и Node.js, удовлетворяя различные потребности в разработке.
- **Визуальный интерфейс**: Обеспечивает интуитивно понятный интерфейс, который позволяет легко создавать, настраивать и управлять задачами сборки без глубоких технических знаний.
- **Полное управление жизненным циклом**: Охватывает весь жизненный цикл от коммита кода до развертывания приложения, автоматизируя сборку, развертывание и оперативное управление.
- **Глубокая интеграция**: Бесшовно интегрируется с вашим контейнерным продуктом, обеспечивая беспрепятственный опыт разработки.
- **Высокая расширяемость**: Поддерживает пользовательские плагины и расширения для удовлетворения ваших конкретных нужд.

## Основные преимущества

- **Ускоренная разработка**: Оптимизирует процесс сборки, увеличивая скорость доставки приложений.
- **Повышенная гибкость**: Поддерживает сборку на нескольких языках программирования.
- **Улучшенная эффективность**: Автоматизирует процессы сборки и развертывания, снижая ручное вмешательство.
- **Повышенная надежность**: Обеспечивает детализированные журналы сборки и визуальное мониторинг для легкой отладки.

## Сценарии применения

Основные сценарии применения S2I следующие:

- Веб-приложения

  S2I поддерживает различные языки программирования, такие как Java, Go, Python и Node.js. Используя возможности управления приложениями <Term name="product" textCase="capitalize" />, он позволяет быстро создавать и развертывать веб-приложения, просто введя URL-адрес репозитория кода.

- CI/CD

  S2I бесшовно интегрируется с конвейерами DevOps, используя инструменты, специфичные для Kubernetes, такие как Helm и GitOps, для автоматизации процессов сборки и развертывания изображений. Это позволяет осуществлять непрерывную интеграцию и непрерывное развертывание приложений.

## Ограничения использования

> Текущая версия поддерживает только языки Java, Go, Python и Node.js.
