# Создание приложения из кода

> Используя мощные возможности установки Alauda Container Platform Builds для достижения полного процесса от исходного кода Java до создания приложения и, в конечном итоге, обеспечения эффективного выполнения приложения в контейнеризованном виде на Kubernetes.

## Предварительные требования

Перед использованием этой функции убедитесь, что:

- [Установка Alauda Container Platform Builds](../install_builds/install_builds_operator.mdx)
- На платформе есть доступный репозиторий изображений. Если нет, пожалуйста, свяжитесь с администратором для [Установки ACP Registry](/developer/registry/install_registry.mdx)

## Процедура

1. **Container Platform**, нажмите **Applications** > **Applications**.

2. Нажмите **Create**.

3. Выберите **Create from Code**.

4. Завершите настройку в соответствии с параметрами ниже:
   | **Параметр**        | **Рекомендуемая конфигурация**                                                                                                          |
   | :------------------- | :------------------------------------------------------------------------------------------------------------------------------------- |
   | **Репозиторий кода**  | **Тип**: `Input`<br /> **URL репозитория**: `https://github.com/alauda/spring-boot-hello-world`                                        |
   | **Метод сборки**     | `Build`                                                                                                                                |
   | **Репозиторий изображений** | Свяжитесь с администратором.                                                                                                             |
   | **Приложение**      | **Приложение**: `spring-boot-hello-world`<br /> **Имя**: `spring-boot-hello-world` <br />**Ограничения ресурсов**: Используйте значение по умолчанию. |
   | **Сеть**          | **Целевой порт**: `8080`                                                                                                                |

5. После заполнения параметров нажмите **Create**.

6. Вы можете проверить соответствующий статус приложения на странице **Details**.
