# Архитектура

![Source to Image Architecture](../../../en/developer/s2i/assets/s2i-archtecture.png "Архитектура Source to Image")

Функция Source to Image (S2I) реализуется через оператор **ACP Builds**, который позволяет автоматизировать создание контейнерных изображений из исходного кода Git-репозитория и последующую отправку в назначенный реестр изображений. Основные компоненты включают:

- Оператор **ACP Builds**: Управляет полным жизненным циклом сборки и оркестрирует конвейеры Tekton.
- Конвейеры **Tekton**: Выполняют рабочие процессы S2I с помощью ресурсов `TaskRun`, родных для Kubernetes.
