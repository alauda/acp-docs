# Архитектура

![Source to Image Architecture](../../../en/developer/s2i/assets/s2i-archtecture.png "Source to Image Architecture")

Возможность Source to Image (S2I) реализуется через оператор **ACP Builds**, который позволяет автоматизировать создание образов контейнеров из исходного кода Git-репозитория и последующую отправку в назначенный реестр образов. Основные компоненты включают в себя:

- Оператор **ACP Builds**: управляет полным циклом сборки и управляет пайплайнами Tekton.
- Пайплайны **Tekton**: выполняют рабочие процессы S2I через нативные для Kubernetes ресурсы `TaskRun`.
