# Управление приложениями, созданными из кода

## Ключевые особенности

- Введите URL-адрес репозитория кода, чтобы запустить процесс S2I, который преобразует исходный код в образ и публикует его как приложение.
- Когда исходный код обновляется, инициируйте действие **Пересобрать** через визуальный интерфейс, чтобы обновить версию приложения одним щелчком.

## Преимущества

- Упрощает процесс создания и обновления приложений из кода.
- Снижает барьер для разработчиков, устраняя необходимость понимать детали контейнеризации.
- Обеспечивает визуальный процесс построения и управления операциями, облегчая локализацию проблем, анализ и устранение неполадок.

## Предварительные условия

- [Установка Alauda Container Platform Builds](../install_builds/install_builds_operator.mdx) завершена.
- Необходим доступ к репозиторию образов; если он недоступен, свяжитесь с администратором для [Установки Alauda Container Platform Registry](/developer/registry/install_registry.mdx)

<a id="s2iparameter" />

## Процедура

1. **Платформа контейнеров**, перейдите в **Приложение** > **Приложение**.

2. Нажмите **Создать**.

3. Выберите **Создать из кода**.

## Связанные операции

### Сборка

После создания приложения соответствующая информация может быть просмотрена на странице деталей.

| Параметр       | Описание                                                                                                  |
| :-------------- | :--------------------------------------------------------------------------------------------------------- |
| **Сборка**     | Нажмите на ссылку, чтобы просмотреть конкретную информацию о сборке (Build) и задаче сборки (BuildRun) и YAML. |
| **Начать сборку** | Когда сборка завершается с ошибкой или исходный код изменяется, вы можете нажать эту кнопку, чтобы повторно выполнить задачу сборки.   |
