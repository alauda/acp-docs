# Управление приложениями, созданными из кода

## Ключевые функции

- Введите URL-адрес репозитория кода, чтобы запустить процесс S2I, который преобразует исходный код в образ и публикует его как приложение.
- Когда исходный код обновляется, инициируйте действие **Пересобрать** через визуальный интерфейс, чтобы обновить версию приложения одним щелчком.

## Преимущества

- Упрощает процесс создания и обновления приложений из кода.
- Снижает барьер для разработчиков, устраняя необходимость понимания тонкостей контейнеризации.
- Обеспечивает визуальный процесс сборки и управления, способствуя локализации проблем, анализу и устранению неисправностей.

## Предварительные условия

- Завершена установка [ACP Builds](../install_builds/install_builds_opeator.mdx).
- Необходим доступ к репозиторию образов; если его нет, свяжитесь с администратором для [Установки ACP Registry](/developer/registry/install_registry.mdx)

## Процедура

1. На платформе **Контейнер**, перейдите в раздел **Приложение** > **Приложение**.

2. Нажмите **Создать**.

3. Выберите **Создать из кода**.

## Связанные операции

### Сборка

После создания приложения соответствующая информация может быть просмотрена на странице деталей.

| Параметр       | Описание                                                                                                |
| :-------------- | :--------------------------------------------------------------------------------------------------------- |
| **Сборка**       | Нажмите на ссылку, чтобы просмотреть конкретную информацию о сборке (Build) и задаче сборки (BuildRun) и YAML. |
| **Запустить сборку** | Когда сборка не удалась или исходный код изменился, вы можете нажать эту кнопку, чтобы повторно выполнить задачу сборки.   |
