# Удаление/Удаление пространств имен

Вы можете либо навсегда удалить пространство имен, либо удалить его из текущего проекта.

## Удалить пространство имен

**Удалить пространство имен**: Навсегда удаляет пространство имен и все ресурсы в нем (например, Pods, Services, ConfigMaps). Это действие нельзя отменить и освобождает выделенные квоты ресурсов.

```bash
kubectl delete namespace <namespace-name>
```

## Удалить пространство имен

**Удалить пространство имен из проекта**: Удаляет пространство имен из текущего проекта, не удаляя его ресурсы. Пространство имен остается в кластере и может быть импортировано в другие проекты через [Импорт пространства имен](./import_namespace.mdx).

**Примечание**

- Эта функция эксклюзивна для <Term name="product" />.
- Kubernetes не поддерживает "удаление" пространств имен из проектов.

```bash
kubectl label namespace <namespace-name> cpaas.io/project- --overwrite
```
