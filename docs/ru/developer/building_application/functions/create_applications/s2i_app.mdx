# Создание приложений из кода

Создание приложений из кода реализуется с использованием технологии Source to Image (S2I). S2I — это автоматизированная структура для создания контейнерных образов непосредственно из исходного кода. Этот подход стандартизирует и автоматизирует процесс сборки приложений, позволяя разработчикам сосредоточиться на разработке исходного кода, не беспокоясь о деталях контейнеризации.

## Предпосылки

- Завершите установку [Alauda Container Platform Builds](../../../s2i/install_builds/install_builds_operator.mdx)

## Процедура

1. **Платформа контейнеров**, и перейдите в **Приложение** > **Приложения**.

2. Нажмите **Создать**.

3. Выберите **Создать из кода**.

4. Для подробных описаний параметров, пожалуйста, обратитесь к [Управлению приложениями, созданными из кода](/developer/s2i/functions/s2i_application_management.mdx#s2iparameter).

5. После завершения ввода параметров нажмите **Создать**.

6. Соответствующее развертывание можно просмотреть на странице **Подробная информация**.
