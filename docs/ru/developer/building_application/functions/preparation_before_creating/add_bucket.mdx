# Создание запросов на ведро

Создайте запрос на ведро, который динамически генерирует ведро с использованием класса объектного хранилища и связывает его.

## Предварительные требования

Пожалуйста, свяжитесь с администратором для установки **Плагина интерфейса объектного хранилища**.

## Процедура

1. **Платформа контейнеров**.

2. В левой навигационной панели нажмите **Хранилище** > **Запросы на ведро**.

3. Нажмите **Создать запрос на ведро**.

4. Ознакомьтесь со следующими инструкциями для настройки соответствующих параметров.

   **Инструкции**: Следующее содержимое представлено в формате формы, и вы также можете переключиться на формат YAML для выполнения операции.

   | Параметр                | Описание                                                                                   |
   | ----------------------- | ------------------------------------------------------------------------------------------ |
   | **Имя**                 | Имя декларации ведра.                                                                      |
   | **Класс объектного хранилища** | Динамически генерирует ведро на основе класса объектного хранилища и соответствия. |

5. Нажмите **Создать**. Подождите, пока статус не изменится на `Доступно`, что указывает на успешное соответствие.

## Операции

Вы можете нажать кнопку **Действия** в правом верхнем углу страницы деталей, чтобы удалить декларацию ведра по мере необходимости. После удаления декларации ведра данные в ведре будут очищены. Пожалуйста, действуйте осторожно, учитывая соображения безопасности данных.
