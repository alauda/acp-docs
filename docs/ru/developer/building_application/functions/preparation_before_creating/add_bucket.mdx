# Создание запросов на создание бакета

Создайте запрос на создание бакета, который динамически генерирует бакет с использованием класса объектного хранения и связывает его.

## Предварительные требования

Пожалуйста, свяжитесь с администратором для установки **Плагина интерфейса объектного хранения**.

## Процедура

1. **Платформа контейнеров**.

2. В левой навигационной панели щелкните **Хранилище** > **Запросы на создание бакета**.

3. Щелкните **Создать запрос на создание бакета**.

4. Ознакомьтесь с следующими инструкциями для настройки соответствующих параметров.

   **Инструкции**: Следующий контент представлен в форме, и вы также можете переключиться на формат YAML для завершения операции.

   | Параметр                | Описание                                                                             |
   | ------------------------ | ------------------------------------------------------------------------------------- |
   | **Название**            | Название декларации бакета.                                                          |
   | **Класс объектного хранения** | Динамически создавайте бакет на основе класса объектного хранения и соответствующей связи. |

5. Щелкните **Создать**. Подождите, пока статус не изменится на `Доступен`, что указывает на успешное соответствие.

## Операции

Вы можете щелкнуть кнопку **Действия** в правом верхнем углу страницы деталей, чтобы удалить декларацию бакета по мере необходимости. После удаления декларации бакета данные в бакете будут очищены. Пожалуйста, выполняйте операции с осторожностью с учетом вопросов безопасности данных.
