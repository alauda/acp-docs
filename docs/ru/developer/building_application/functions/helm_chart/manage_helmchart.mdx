# Управление шаблонами приложений

Обновление шаблонов приложений зависит от обновления шаблонов приложений (Helm Charts). Если определенные шаблоны не могут быть автоматически обновлены через обновление платформы, публичные репозитории или пользовательские приватные репозитории, вы можете вручную добавить их в репозиторий шаблонов типа Chart или OCI Chart с правами **Управления** для развертывания соответствующей версии приложения.

## Предварительные требования

Шаблонные репозитории добавляются администраторами платформы. Пожалуйста, свяжитесь с администратором платформы для получения доступных имен репозиториев шаблонов типа Chart или OCI Chart с правами **Управления**.

## Добавление шаблонов в управляемые репозитории

1. Перейдите в **Каталог**.

2. В левой навигационной панели нажмите **Helm Charts**.

3. Нажмите **Добавить шаблон** в правом верхнем углу страницы и выберите репозиторий шаблонов на основе приведенных ниже параметров.

   | Параметр               | Описание                                                                                                                                                                                                                                                                                                      |
   | ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Репозиторий шаблонов** | Синхронизируйте шаблон напрямую с репозиторием шаблонов типа Chart или OCI Chart с правами **Управления**. Владельцы проектов, назначенные этому **Репозиторию шаблонов**, могут напрямую использовать шаблон.                                                                                                       |
   | **Директория шаблонов**  | Когда выбранный тип репозитория шаблонов является OCI Chart, необходимо выбрать директорию для хранения Helm Chart или ввести ее вручную. <br />**Примечание**: При ручном вводе новой директории шаблонов платформа создаст эту директорию в репозитории шаблонов, но существует риск неудачного создания. |

4. Нажмите **Загрузить шаблон** и загрузите локальный шаблон в репозиторий.

5. Нажмите **Подтвердить**. Процесс загрузки шаблона может занять несколько минут, пожалуйста, проявите терпение.

   **Примечание**: Когда статус шаблона изменяется с `Загрузка` на `Загрузка успешна`, это означает, что шаблон был успешно загружен.

6. Если загрузка не удалась, пожалуйста, проведите диагностику согласно следующим подсказкам.

   **Примечание**: Неправильный формат файла означает, что в загруженном сжатом пакете есть проблемы с файлами, такие как отсутствие контента или неправильное форматирование.

## Удаление конкретных версий шаблонов

Если версия шаблона больше не применима, ее можно удалить.

### Шаги для выполнения

1. Перейдите в **Каталог**.

2. В левой навигационной панели нажмите **Helm Charts**.

3. Нажмите на карточку Chart, чтобы просмотреть детали.

4. Нажмите **Управление версиями**.

5. Найдите шаблон, который больше не применим, нажмите **Удалить** и подтвердите.

   После удаления версии соответствующее приложение не сможет быть обновлено.
