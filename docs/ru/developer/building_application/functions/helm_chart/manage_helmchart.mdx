# Управление шаблонами приложений

Обновление шаблонов приложений зависит от обновлений шаблонов приложений (Helm Charts). Если определенные шаблоны не могут быть автоматически обновлены через обновление платформы, публичные репозитории или пользовательские приватные репозитории, вы можете вручную добавить их в репозиторий шаблонов типа Chart или OCI Chart с правами **Управления** для развертывания соответствующей версии приложения.

## Предварительные условия

Репозитории шаблонов добавляются администраторами платформы. Пожалуйста, свяжитесь с администратором платформы, чтобы узнать доступные имена репозиториев шаблонов типа Chart или OCI Chart с правами **Управления**.

## Добавление шаблонов в управляемые репозитории

1. Перейдите в **Каталог**.

2. В левой боковой панели нажмите **Helm Charts**.

3. Нажмите **Добавить шаблон** в правом верхнем углу страницы и выберите репозиторий шаблонов на основе следующих параметров.

   | Параметр               | Описание                                                                                                                                                                                                                                                                                                      |
   | ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Репозиторий шаблонов** | Синхронизируйте шаблон непосредственно с репозиторием шаблонов типа Chart или OCI Chart с правами **Управления**. Владельцы проекта, назначенные этому **Репозиторию шаблонов**, могут непосредственно использовать шаблон.                                                                                                       |
   | **Каталог шаблонов**   | Когда выбранный тип репозитория шаблонов — OCI Chart, необходимо выбрать или вручную ввести каталог для хранения Helm Chart. <br />**Примечание**: При ручном вводе нового каталога шаблонов платформа создаст этот каталог в репозитории шаблонов, но существует риск неудачи создания. |

4. Нажмите **Загрузить шаблон** и загрузите локальный шаблон в репозиторий.

5. Нажмите **Подтвердить**. Процесс загрузки шаблона может занять несколько минут, пожалуйста, наберитесь терпения.

   **Примечание**: Когда статус шаблона изменится с `Загрузка` на `Успешная загрузка`, это означает, что шаблон был успешно загружен.

6. Если загрузка не удалась, пожалуйста, устраните неполадки в соответствии со следующими подсказками.

   **Примечание**: Неправильный формат файла означает, что в загруженном сжатом пакете есть проблемы с файлами, такие как отсутствие содержимого или неправильное форматирование.

## Удаление конкретных версий шаблонов

Если версия шаблона больше не актуальна, ее можно удалить.

### Шаги для выполнения

1. Перейдите в **Каталог**.

2. В левой боковой панели нажмите **Helm Charts**.

3. Нажмите на карточку Chart, чтобы просмотреть детали.

4. Нажмите **Управлять версиями**.

5. Найдите шаблон, который больше не актуален, нажмите **Удалить** и подтвердите.

   После удаления версии соответствующее приложение не сможет быть обновлено.
