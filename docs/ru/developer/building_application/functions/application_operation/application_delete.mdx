# Удаление приложений

Удаление приложения одновременно удаляет само приложение и все его непосредственно содержащие ресурсы Kubernetes. Кроме того, это действие разрывает любые связи, которые приложение могло иметь с другими ресурсами Kubernetes, которые не были непосредственно частью его определения.
