# Запуск и остановка приложений

## Запуск приложения

1. Доступ к **Платформе контейнеров**.

2. В левой навигационной панели нажмите **Приложение** > **Приложения**.

3. Щелкните по имени приложения.

4. Нажмите **Запустить**.

## Остановка приложения

1. Доступ к **Платформе контейнеров**.

2. В левой навигационной панели нажмите **Приложение** > **Приложения**.

3. Щелкните по имени приложения.

4. Нажмите **Остановить**.

5. Прочитайте сообщение с запросом, и после подтверждения, что все правильно, нажмите **Остановить**.
