# Запуск и остановка нативных приложений

## Запуск приложения

1. Получите доступ к **Платформе контейнеров**.

2. В левом навигационном меню нажмите **Управление приложениями** > **Нативные приложения**.

3. Нажмите на название приложения.

4. Нажмите **Запустить**.

## Остановка приложения

1. Получите доступ к **Платформе контейнеров**.

2. В левом навигационном меню нажмите **Управление приложениями** > **Нативные приложения**.

3. Нажмите на название приложения.

4. Нажмите **Остановить**.

5. Прочитайте сообщение с приглашением, и после подтверждения, что все верно, нажмите **Остановить**.
