# Описание статуса

## Нативные приложения

Статус нативных приложений и их соответствующие значения следующие. Числа, следующие за статусом, указывают количество вычислительных компонентов.

| Статус                      | Значение                                                            |
| --------------------------- | ------------------------------------------------------------------- |
| **Работает**                | Все вычислительные компоненты работают в нормальном режиме.        |
| **Частично работает**       | Некоторые вычислительные компоненты работают, в то время как другие остановлены. |
| **Остановлено**             | Все вычислительные компоненты остановлены.                         |
| **Обработка**               | По крайней мере, один вычислительный компонент находится в ожидании. |
| **Нет вычислительных компонентов** | Под приложением нет вычислительных компонентов.                 |
| **Неудача**                 | Развертывание завершилось неудачей.                                |

**Примечание**: Точно так же, числа в статусе вычислительного компонента указывают количество групп контейнеров.

# Развертывание

- Работает: Все Pods работают в нормальном режиме.

- Обработка: Есть Pods, которые не находятся в рабочем состоянии.

- Остановлено: Все Pods остановлены.

- Неудача: Развертывание завершилось неудачей.
