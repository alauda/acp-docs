# Развертывание нативных приложений без шаблона с помощью Chart

Чтобы разработать текущее приложение для распространения на другие среды в будущем, вы можете отсоединить приложение от шаблона: перейдите в раздел **Container Platform** > **Application Management** > **Native Applications**, откройте страницу деталей приложения и нажмите **Actions** > **Detach from Template**.

Прежде чем отсоединить от шаблона, прямые операции с базовыми ресурсами (такими как рабочие нагрузки, внутренние маршруты и т. д.) будут иметь временное действие и не будут записываться в шаблон. После обновления приложения конфигурация в шаблоне перезапишет временную конфигурацию. Отсоединенное приложение неотличимо от приложения, созданного через платформу [UI form](../create_applications/image_app.mdx), и не подлежит упомянутым выше ограничениям.
