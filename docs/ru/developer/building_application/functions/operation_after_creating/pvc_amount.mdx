# Управление использованием хранилища

Просмотрите данные об использовании хранилища смонтированных постоянных томов для ссылки при использовании рабочих нагрузок или приложений. Если доступной емкости недостаточно, емкость запроса постоянного тома может быть увеличена соответствующим образом.

## Просмотр использования хранилища постоянного тома

### Заметки

- Данные об использовании хранилища предназначены только для справки и поддерживают сбор данных об использовании хранилища только для смонтированных постоянных томов.

- Общая емкость и данные об использовании PVC, такие как HostPath, CephRBD и TopoLVM, могут иметь некоторые расхождения с фактическими значениями.

### Шаги

1. Перейдите в **Container Platform**.

2. В левой боковой панели навигации щелкните **Storage > Persistent Volume Claims (PVC)**.

3. Щелкните по имени запроса постоянного тома.

4. На вкладке **Details** прокрутите вниз до области **Storage Usage**.

5. Просмотрите текущую используемую емкость смонтированных постоянных томов и ее тренд.

## Увеличение емкости запроса постоянного тома

1. В левой боковой панели навигации щелкните **Storage > Persistent Volume Claims (PVC)**.

2. Найдите запрос постоянного тома и щелкните ⋮ > **Expand**.

3. Укажите новую емкость.

4. Нажмите **Expand**. Процесс расширения может занять некоторое время, пожалуйста, будьте терпеливы.
