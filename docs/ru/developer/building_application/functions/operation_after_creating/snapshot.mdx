# Использование снимков томов

Снимок тома - это копия постоянного запроса тома (PVC) на определенный момент времени, которая может быть использована для настройки новых постоянных запросов томов (предварительно заполняя их данными из снимка) или для восстановления существующих постоянных запросов томов до предыдущего состояния, что достигает эффекта резервного копирования данных приложений и восстановления их по мере необходимости, обеспечивая тем самым надежность данных приложений.

## Предварительные требования

- Администратор развернул компонент снимков томов **Snapshot Controller** для текущего кластера и активировал функции, связанные со снимками, в кластере хранения.

- Постоянный запрос тома должен быть создан динамически, а его статус должен быть **Привязан**.

- Класс хранения, связанный с постоянным запросом тома, должен поддерживать функциональность снимков, такие как **CephRBD встроенное хранилище**, **CephFS встроенное хранилище** или **TopoLVM**.

## Создание снимков томов

### Создание снимка тома на основе указанного постоянного запроса тома (PVC)

**Способ первый**

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранение > Постоянные запросы томов (PVC)**.

3. Нажмите ⋮ рядом с соответствующим постоянным запросом тома в списке и выберите **Создать снимок тома**.

4. Заполните описание снимка. Это описание может помочь зафиксировать текущее состояние постоянного тома, например, *Перед обновлением приложения*.

5. Нажмите **Создать**. Время, необходимое для создания снимка, зависит от сетевых условий и объема данных; пожалуйста, наберитесь терпения.

   Когда статус снимка изменится на `Доступно`, это указывает на успешное завершение создания.

**Способ второй**

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранение > Постоянные запросы томов (PVC)**.

3. Нажмите на имя постоянного запроса тома в списке.

4. Переключитесь на вкладку **Снимки томов**.

5. Нажмите **Создать снимок тома** и настройте соответствующие параметры по мере необходимости.

6. Нажмите **Создать**. Время, необходимое для создания снимка, зависит от сетевых условий и объема данных; пожалуйста, наберитесь терпения.

   Когда статус снимка изменится на `Доступно`, это указывает на успешное завершение создания.

### Создание снимков томов настраиваемым способом

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранение > Снимки томов**.

3. Нажмите **Создать снимок тома** и настройте соответствующие параметры по мере необходимости.

4. Нажмите **Создать**. Время, необходимое для создания снимка, зависит от сетевых условий и объема данных; пожалуйста, наберитесь терпения.

   Когда статус снимка изменится на `Доступно`, это указывает на успешное завершение создания.

## Создание постоянных запросов томов из снимков томов

В настоящее время платформа поддерживает создание снимков томов только с использованием PVC, созданных из классов хранения с **Динамическим провизированием**. Вы можете создать новые PVC на основе этого снимка и связать их.

**Примечание**: Режимы доступа, поддерживаемые при создании PVC из снимка, отличаются от тех, которые поддерживаются при создании PVC из класса хранения, как указано **жирным шрифтом** в таблице.

| Класс хранения, используемый для создания снимков томов | Однопользовательский режим чтения-записи (RWO) | Многопользовательский режим только для чтения (ROX) | Многопользовательский режим чтения-записи (RWX) |
| ------------------------------------------------------ | ----------------------------------------------- | ------------------------------------------------- | ----------------------------------------------- |
| **TopoLVM**                                           | Поддерживается                                   | Не поддерживается                                   | Не поддерживается                                  |
| **CephRBD блочное хранилище**                        | Поддерживается                                   | Не поддерживается                                   | Не поддерживается                                  |
| **CephFS файловое хранилище**                         | Поддерживается                                   | **Поддерживается**                                 | Поддерживается                                     |

### Способ первый

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранение > Постоянные запросы томов (PVC)**.

3. Нажмите на имя постоянного запроса тома в списке.

4. Переключитесь на вкладку **Снимки томов**.

5. Нажмите ⋮ рядом с соответствующим снимком тома в списке и выберите **Создать постоянный запрос тома**, настроив соответствующие параметры.

6. Нажмите **Создать**.

### Способ второй

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранение > Снимки томов**.

3. Нажмите ⋮ рядом с соответствующим снимком тома в списке и выберите **Создать постоянный запрос тома**, настроив соответствующие параметры.

4. Нажмите **Создать**.
