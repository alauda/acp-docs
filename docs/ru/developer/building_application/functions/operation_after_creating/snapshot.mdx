# Использование снимков томов

Снимок тома — это копия определенного момента временной метки постоянного запроса тома (PVC), которая может быть использована для настройки новых постоянных запросов тома (предварительно заполняя данными из снимка) или для возврата существующих постоянных запросов тома к предыдущему состоянию, достигая эффекта резервного копирования данных приложений и восстановления их по мере необходимости, тем самым обеспечивая надежность данных приложений.

## Предварительные условия

- Администратор развернул компонент снимка тома **Snapshot Controller** для текущего кластера и включил функции, связанные со снимками, в кластер хранения.

- Постоянный запрос тома должен быть создан динамически, и его статус должен быть **Zаблокирован**.

- Класс хранения, связанный с постоянным запросом тома, должен поддерживать функциональность снимков, такую как **CephRBD встроенное хранилище**, **CephFS встроенное хранилище** или **TopoLVM**.

## Создание снимков томов

### Создание снимка тома на основе указанного постоянного запроса тома (PVC)

**Метод первый**

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранилище > Постоянные запросы тома (PVC)**.

3. Нажмите ⋮ рядом с соответствующим постоянным запросом тома в списке и выберите **Создать снимок тома**.

4. Заполните описание снимка. Это описание может помочь вам записать текущее состояние постоянного тома, например, *Перед обновлением приложения*.

5. Нажмите **Создать**. Время, необходимое для создания снимка, зависит от условий сети и объема данных; пожалуйста, будьте терпеливы.

   Когда статус снимка изменится на `Доступен`, это указывает на то, что создание было успешным.

**Метод второй**

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранилище > Постоянные запросы тома (PVC)**.

3. Нажмите на имя постоянного запроса тома в списке.

4. Переключитесь на вкладку **Снимки томов**.

5. Нажмите **Создать снимок тома** и настройте соответствующие параметры по мере необходимости.

6. Нажмите **Создать**. Время, необходимое для создания снимка, зависит от условий сети и объема данных; пожалуйста, будьте терпеливы.

   Когда статус снимка изменится на `Доступен`, это указывает на то, что создание было успешным.

### Создание снимков томов произвольным способом

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранилище > Снимки томов**.

3. Нажмите **Создать снимок тома** и настройте соответствующие параметры по мере необходимости.

4. Нажмите **Создать**. Время, необходимое для создания снимка, зависит от условий сети и объема данных; пожалуйста, будьте терпеливы.

   Когда статус снимка изменится на `Доступен`, это указывает на то, что создание было успешным.

## Создание постоянных запросов томов из снимков томов

На данный момент платформа поддерживает создание снимков томов только с использованием PVC, созданных из классов хранения с **ДинамическимProvisioning**. Вы можете создавать новые PVC на основе этого снимка и связывать их.

**Примечание**: Поддерживаемые режимы доступа при создании PVC из снимка отличаются от тех, которые поддерживаются при создании PVC из класса хранения, как указано **жирным** в таблице.

| Класс хранения, используемый для создания снимков томов | Чтение и запись для одного узла (RWO) | Чтение для нескольких узлов (ROX) | Чтение и запись для нескольких узлов (RWX) |
| ------------------------------------------------------ | ------------------------------------ | -------------------------------- | ------------------------------------------- |
| **TopoLVM**                                            | Поддерживается                       | Не поддерживается                | Не поддерживается                           |
| **CephRBD блочное хранилище**                         | Поддерживается                       | Не поддерживается                | Не поддерживается                           |
| **CephFS файловое хранилище**                          | Поддерживается                       | **Поддерживается**              | Поддерживается                               |

### Метод первый

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранилище > Постоянные запросы тома (PVC)**.

3. Нажмите на имя постоянного запроса тома в списке.

4. Переключитесь на вкладку **Снимки томов**.

5. Нажмите ⋮ рядом с соответствующим снимком тома в списке и выберите **Создать постоянный запрос тома**, настроив соответствующие параметры.

6. Нажмите **Создать**.

### Метод второй

1. Войдите в **Платформу контейнеров**.

2. В левой навигационной панели нажмите **Хранилище > Снимки томов**.

3. Нажмите ⋮ рядом с соответствующим снимком тома в списке и выберите **Создать постоянный запрос тома**, настроив соответствующие параметры.

4. Нажмите **Создать**.
