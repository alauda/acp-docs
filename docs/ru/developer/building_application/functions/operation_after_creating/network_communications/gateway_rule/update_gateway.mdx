# Обновление шлюзов

**Примечание**: Обновление входного шлюза приведет к перерыву в обслуживании продолжительностью 3-5 минут. Пожалуйста, выберите подходящее время для этой операции.

**Шаги**

1. Получите доступ к **Платформе контейнеров**.

2. В левом навигационном меню нажмите **Сеть** > **Входной шлюз**.

3. Нажмите ⋮ > **Обновить**.

4. Обновите конфигурацию входного шлюза по мере необходимости.

   **Примечание**: Пожалуйста, установите спецификации разумно, исходя из бизнес-требований.

5. Нажмите **Обновить**.
