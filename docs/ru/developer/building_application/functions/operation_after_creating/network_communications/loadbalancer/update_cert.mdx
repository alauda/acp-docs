# Обновление сертификатов по умолчанию

Как правило, администраторыRenew или заменяют сертификаты до истечения их срока действия. Если администратор заменяет сертификат, используемый для порта прослушивания HTTPS или gRPC, вам необходимо обновить порт прослушивания, чтобы использовать новый сертификат как сертификат по умолчанию.

## Шаги для выполнения

1. Перейдите в **Container Platform**.

2. В левой навигационной панели нажмите **Network** > **Load Balancing**.

3. Нажмите на имя балансировщика нагрузки.

4. Нажмите на имя порта прослушивания.

5. Нажмите **Actions** > **Update Default Certificate**.

6. Выберите новый сертификат и нажмите **Update**.
