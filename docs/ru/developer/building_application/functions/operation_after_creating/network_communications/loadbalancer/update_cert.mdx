# Обновление Сертификатов По Умолчанию

Как правило, администраторы обновляют или заменяют сертификаты до истечения их срока действия. Если администратор заменяет сертификат, используемый для HTTPS или gRPC порта прослушивания, необходимо обновить порт прослушивания, чтобы использовать новый сертификат в качестве сертификата по умолчанию.

## Шаги Для Выполнения

1. Перейдите в **Платформа Контейнеров**.

2. В левой навигационной панели нажмите **Сеть** > **Балансировка Нагрузки**.

3. Нажмите на имя балансировщика нагрузки.

4. Нажмите на имя порта прослушивания.

5. Нажмите **Действия** > **Обновить Сертификат По Умолчанию**.

6. Выберите новый сертификат и нажмите **Обновить**.
