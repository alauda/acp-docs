# Загрузка конфигурации ConfigMap

После обновления смонтированного ConfigMap приложение должно загрузить обновленную конфигурацию, чтобы изменения вступили в силу. Для приложений, которые не могут автоматически загрузить конфигурацию, **Загрузить конфигурацию** предоставляет возможность вручную инициировать загрузку ConfigMap.

## Предварительные условия

- Приложение должно поддерживать обновления конфигурации в реальном времени.

- ConfigMap должен быть смонтирован как хранилище в поде, и SubPath использовать нельзя. ConfigMap, упомянутые через переменные окружения, не поддерживают загрузку конфигурации.

## Шаги

1. Войдите в **Платформу контейнеров**.

2. В левом навигационном меню нажмите **Конфигурация** > **ConfigMaps**.

3. Щелкните на ***Имя ConfigMap***.

4. Нажмите **Добавить элемент конфигурации** или нажмите **Добавить бинарный элемент конфигурации**; также можно нажать <span style={{ display: 'inline-flex', alignItems: 'center', gap: '4px', verticalAlign: 'middle' }}><img src="../../../../../en/developer/building_application/functions/operation_after_creating/assets/blackpencil.png" alt="expand" style={{ verticalAlign: 'middle' }} /> <span>Обновить элемент конфигурации и подтвердить.</span></span> по мере необходимости.

5. Перейдите на вкладку **Рабочие нагрузки** и нажмите <span style={{ display: 'inline-flex', alignItems: 'center', gap: '4px', verticalAlign: 'middle' }}><img src="../../../../../en/developer/building_application/functions/operation_after_creating/assets/fold.png" alt="expand" style={{ verticalAlign: 'middle' }} /> <span>Развернуть список его контейнеров.</span></span>.

6. Нажмите **Загрузить конфигурацию**, введите пользователя и выполните команду, затем подтвердите. По умолчанию выполняется в контейнере от имени пользователя **root**.

7. Введите соответствующую команду для загрузки конфигурации в зависимости от типа приложения.

   - Пример 1: Nginx

     ```
     nginx -s reload
     ```

   - Пример 2: Logstash

     ```
     kill -SIGHUP <идентификатор процесса Logstash>
     ```
