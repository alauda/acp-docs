# Параметры Подов

Интерфейс платформы предоставляет различную информацию о подах для быстрого справки. Ниже приведены некоторые объяснения параметров.

| Параметр                    | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Запросы/лимиты ресурсов** | Эффективные значения запросов и лимитов ресурсов (ЦП, память) для подов. Способ расчета значений запросов и лимитов одинаков; данный документ вводит использование значений лимита в качестве примера, конкретные правила и алгоритмы приведены ниже: <ul><li>Когда поды содержат только **бизнес-контейнеры** (контейнеры), значение лимита **ЦП/память** равно сумме значений лимита **ЦП/память** всех контейнеров внутри подов. **Например**: если поды включают два бизнес-контейнера со значениями лимита **ЦП/память** 100m/100Mi и 50m/200Mi, значение лимита **ЦП/память** подов будет равно 150m/300Mi.</li><li>Когда поды содержат как **инициализационные контейнеры** (initContainers), так и **бизнес-контейнеры**, шаги расчета значений лимитов **ЦП/память** подов следующие: <ol><li>Взять максимальное значение лимитов **ЦП/память** всех инициализационных контейнеров.</li> <li>Взять сумму значений лимитов **ЦП/память** всех бизнес-контейнеров.</li><li>Сравнить результаты и взять максимальные значения **ЦП** и **память** как лимиты **ЦП/память** подов из инициализационных и бизнес-контейнеров.</li></ol></li><br />**Пример расчета**: если поды содержат два инициализационных контейнера со значениями лимита **ЦП/память** 100m/200Mi и 200m/100Mi, максимальное значение лимита **ЦП/память** для инициализационных контейнеров составит 200m/200Mi. В то же время, если поды также содержат два бизнес-контейнера со значениями лимита **ЦП/память** 100m/100Mi и 50m/200Mi, общее значение лимита для бизнес-контейнеров будет равно 150m/300Mi. Таким образом, общее значение лимита **ЦП/память** для подов составит 200m/300Mi.</ul> |
| **Источник**                | Вычислительный компонент, к которому принадлежат поды.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Количество перезапусков**  | Количество перезапусков, когда статус подов ненормальный.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Узел**                    | Название узла, на котором находятся поды.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **Сервисная учетная запись** | Сервисная учетная запись — это учетная запись, которая позволяет процессам и службам в поде получать доступ к API-серверу Kubernetes, предоставляя идентификацию для процессов и служб. Поле **Сервисная учетная запись** отображается только в том случае, если текущий вошедший пользователь имеет либо роль администратора платформы, либо роль аудитора платформы, и файла YAML сервисной учетной записи может быть просмотрен.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
