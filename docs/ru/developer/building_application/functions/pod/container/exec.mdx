# Вход в контейнер через EXEC

## Вход в контейнер через приложения

Вы можете войти во внутренний экземпляр контейнера, используя команду `kubectl exec`, что позволяет вам выполнять командные операции в окне консоли Web. Кроме того, вы можете легко загружать и скачивать файлы внутри контейнера с помощью функции передачи файлов.

### Предварительные условия

- Контейнер должен корректно работать.

- При использовании функции передачи файлов инструмент `tar` должен быть доступен в контейнере, а операционная система контейнера не может быть Windows.

### Процедура

1. Войдите в **Платформа контейнеров**.

2. В левом боковом меню нажмите **Приложение** > **Приложения**.

3. Щелкните на ***Имя приложения***.

4. Найдите нагрузку и нажмите **EXEC** > ***Имя Pod***.

5. Введите команду, которую вы хотите выполнить.

6. Нажмите **OK**, чтобы войти в окно консоли Web и выполнить командные операции.

7. Нажмите **Передача файлов**. Введите **Путь загрузки**, чтобы загрузить файлы для тестирования в контейнер; или введите **Путь скачивания**, чтобы скачать журналы и другие файлы из контейнера на ваш локальный компьютер для анализа.
