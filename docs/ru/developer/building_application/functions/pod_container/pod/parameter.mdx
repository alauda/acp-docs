# Параметры Pod

Интерфейс платформы предоставляет различные данные о подах для быстрого ознакомления. Ниже приведены объяснения некоторых параметров.

| Параметр                     | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Запросы/ограничения ресурсов** | Эффективные значения запросов и ограничений ресурсов (ЦП, память) для подов. Метод расчета значений запросов и ограничений одинаков; в данном документе представлено использование значений ограничений в качестве примера, а конкретные правила и алгоритмы следующие: <ul><li>Когда поды содержат только **бизнес-контейнеры** (контейнеры), значение ограничения **ЦП/памяти** является суммой значений ограничений **ЦП/памяти** всех контейнеров в подах. **Например**: если поды включают два бизнес-контейнера с лимитами ЦП/памяти 100m/100Mi и 50m/200Mi, значение ограничения **ЦП/памяти** подов будет 150m/300Mi.</li><li>Когда поды содержат как **инициализирующие контейнеры** (initContainers), так и **бизнес-контейнеры**, шаги расчета значений ограничений **ЦП/памяти** подов следующие: <ol><li>Взять максимальное значение из значений ограничений **ЦП/памяти** всех инициализирующих контейнеров.</li> <li>Взять сумму значений ограничений **ЦП/памяти** всех бизнес-контейнеров.</li><li>Сравнить результаты и взять максимальные значения **ЦП** и **памяти** как значение ограничений **ЦП/памяти** подов.</li></ol></li><br />**Пример расчета**: если поды содержат два инициализирующих контейнера с ограничениями **ЦП/памяти** 100m/200Mi и 200m/100Mi, максимальное значение **ЦП/памяти** для инициализирующих контейнеров будет 200m/200Mi. В то же время, если поды также содержат два бизнес-контейнера с ограничениями **ЦП/памяти** 100m/100Mi и 50m/200Mi, общее значение ограничений для бизнес-контейнеров составит 150m/300Mi. Следовательно, комплексное значение ограничений **ЦП/памяти** для подов будет 200m/300Mi.</ul> |
| **Источник**                 | Вычислительный компонент, которому принадлежат поды.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Количество перезапусков**  | Количество перезапусков, если статус подов ненормальный.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Узел**                     | Имя узла, на котором расположен под.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **Служебная учетная запись** | Служебная учетная запись — это учетная запись, которая позволяет процессам и службам в Pod получать доступ к APIServer Kubernetes, предоставляя идентификацию для процессов и служб. Поле **Служебная учетная запись** видно только тогда, когда текущий вошедший пользователь имеет либо роль администратора платформы, либо роль аудитора платформы, и файл YAML Служебной учетной записи может быть просмотрен.
