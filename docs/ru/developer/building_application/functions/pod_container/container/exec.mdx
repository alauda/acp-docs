# Вход в контейнер через EXEC

## Вход в контейнер через приложения

Вы можете войти во внутренний экземпляр контейнера, используя команду `kubectl exec`, что позволяет вам выполнять операции командной строки в окне веб-консоли. Кроме того, вы можете легко загружать и загружать файлы внутри контейнера, используя функциональность передачи файлов.

### Предварительные требования

- Контейнер должен работать должным образом.

- При использовании функции передачи файлов инструмент `tar` должен быть доступен в контейнере, а операционная система контейнера не может быть Windows.

### Процедура

1. Войдите в **Платформу контейнеров**.

2. В левом навигационном меню нажмите **Приложение** > **Приложения**.

3. Щелкните на ***Название приложения***.

4. Найдите рабочую нагрузку и нажмите **EXEC** > ***Имя Pod***.

5. Введите команду, которую вы хотите выполнить.

6. Нажмите **OK**, чтобы войти в окно веб-консоли и выполнить операции командной строки.

7. Нажмите **Передача файлов**. Введите **Путь загрузки**, чтобы загрузить файлы для тестирования в контейнер; или введите **Путь загрузки**, чтобы загрузить журналы и другие файлы из контейнера на ваш локальный компьютер для анализа.
