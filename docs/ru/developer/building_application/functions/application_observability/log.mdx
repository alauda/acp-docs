# Логи

Агрегируйте журналы выполнения контейнеров с возможностями визуального запроса. Когда приложения, рабочие нагрузки или другие ресурсы проявляют аномальное поведение, анализ журналов помогает диагностировать коренные причины.

## Процедура

> Эта процедура демонстрирует, как просматривать журналы выполнения контейнеров через интерфейс развертывания.

1. **Платформа контейнеров**, нажмите **Рабочие нагрузки** > **Развертывания**.

2. Нажмите на имя развертывания из списка.

3. Перейдите на вкладку **Логи**, чтобы просмотреть подробные записи.

   | Операция             | Описание                                                                                                                     |
   | :-------------------- | :------------------------------------------------------------------------------------------------------------------------------ |
   | **Pod**/**Контейнер** | Переключайтесь между Pod и Контейнерами, используя выпадающий селектор для просмотра соответствующих журналов.                                  |
   | **Предыдущие логи**     | Просмотр журналов из завершенных контейнеров (доступно, когда restartCount контейнера > 0).                                               |
   | **Строки**             | Настройка размера буфера отображаемых журналов: 1K/10K/100K строк.                                                                           |
   | **Перенос строки**         | Переключение переноса строк для длинных записей журналов (включено по умолчанию).                                                                 |
   | **Поиск**              | Полнотекстовый поиск с выделением совпадений и переходом по нажатию Enter.                                                                 |
   | **Сырой**               | Непроцессированные потоки журналов, непосредственно захваченные из интерфейсов выполнения контейнеров (CRI) без форматирования, фильтрации или усечения. |
   | **Экспорт**            | Скачивание сырых журналов.                                                                                                              |
   | **Полный экран**       | Нажмите на усеченную строку, чтобы просмотреть полный контент в модальном диалоговом окне.                                                                      |

:::warning

- **Обработка усечения**: Записи журналов, превышающие 2000 символов, будут усечены с многоточием `...`
  - Обрезанные части не могут быть найдены с помощью функции поиска на странице.
  - Нажмите на маркер многоточия `...` в усеченных строках, чтобы просмотреть полный контент в модальном диалоговом окне.
- **Надежность копирования**: Избегайте прямого копирования из отображаемого просмотрщика журналов, когда видите маркеры усечения (...) или коды цвета ANSI. Всегда используйте функции **Экспорт**, **Сырой** для получения полных журналов.
- **Политика хранения**: Живые журналы следуют конфигурации ротации журналов Kubernetes. Для исторического анализа используйте [Логи](../../../../observability/log/functions/log.mdx) в разделе Наблюдение.
:::
