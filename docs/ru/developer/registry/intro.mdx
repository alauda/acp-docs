# Введение

## Введение в реестр ACP

Реестр Docker "из коробки" является незаменимым компонентом в облачно-ориентированных средах, упрощая управление контейнерными изображениями, повышая эффективность разработки и улучшая стабильность системы. Реестр Docker предоставляет безопасное, надежное и высокопроизводительное решение для частного реестра изображений, позволяя централизованное управление и распределение изображений Docker. Это решение идеально подходит для сценариев развертывания с реестрами изображений меньшего масштаба и более строгими требованиями к управлению инфраструктурными ресурсами.

## Преимущества реестра ACP

**Ключевые преимущества реестра ACP следующие:**

- **Готовый к использованию:** Быстрое создание частного репозитория изображений без сложных настроек.
- **Гибкое развертывание:** Поддержка как внутрикластерного, так и внекласстерного доступа для удовлетворения различных потребностей в развертывании.
- **Безопасный и надежный:** Обеспечение авторизации и аутентификации для гарантии безопасности изображений.
- **Высокая доступность:** Обеспечение непрерывного сервиса.
- **Готов к производству:** Строгое тестирование и валидация для удовлетворения требований производственной среды.

## Сценарии применения реестра ACP

- В сценариях бизнеса с низким трафиком внедрение легковесного реестра "из коробки" упрощает быстрое развертывание и упрощает распределение бизнес-приложений.
- В сценариях крайних вычислений каждый крайний кластер может использовать свой собственный выделенный реестр изображений, позволяя полностью автономное управление операционными кластерами.
- Когда инфраструктура клиента остается недогруженной, интеграция платформы с технологией Source-to-Image (S2I) предоставляет готовые демонстрационные сценарии, позволяя комплексную демонстрацию бизнес-процессов от начала до конца.
