# Введение

## Введение в ACP Registry

Готовый к использованию docker registry является незаменимым компонентом в облачных нативных средах, упрощая управление контейнерными образами, повышая эффективность разработки и улучшая стабильность системы. Docker registry предлагает безопасное, надежное и высокопроизводительное решение для частного реестра образов, позволяя централизованное управление и распределение docker-образов. Это решение идеально подходит для сценариев развертывания с реестрами образов меньшего масштаба и более строгими требованиями к контролю ресурсов инфраструктуры.

## Преимущества ACP Registry

**Основные преимущества ACP Registry следующие:**

- **Готово к использованию:** Быстрое создание частного репозитория образов без сложных конфигураций.
- **Гибкое развертывание:** Поддерживает как доступ внутри кластера, так и вне его, чтобы удовлетворить различные потребности в развертывании.
- **Безопасность и надежность:** Обеспечивает авторизацию и аутентификацию для гарантии безопасности образов.
- **Высокая доступность:** Обеспечивает непрерывную работу сервиса.
- **Готово к производству:** Строго протестировано и верифицировано для соответствия требованиям производственной среды.

## Сценарии применения ACP Registry

- В сценариях с низкой загруженностью бизнеса внедрение легковесного готового к использованию Registry облегчает быстрое развертывание и упрощает распределение бизнес-приложений.
- В сценариях периферийных вычислений каждый крайний кластер может использовать свой собственный выделенный реестр образов, обеспечивая полностью автономное управление операционными кластерами.
- Когда инфраструктура клиента используется не в полную силу, интеграция платформы с технологией Source-to-Image (S2I) предоставляет готовые демонстрационные сценарии, позволяя провести всестороннюю демонстрацию бизнес-процессов от начала до конца.
