# Описание единицы ресурсов

- CPU: Допустимые единицы: core, m (милликор). Где 1 core = 1000 m.

- Память: Допустимые единицы: Mi (1 MiB = 2^20 байтов), Gi (1 GiB = 2^30 байтов). Где 1 Gi = 1024 Mi.

- Виртуальный GPU (дополнительно): Этот параметр активен только при наличии ресурсов GPU в кластере. Количество виртуальных ядер GPU; 100 виртуальных ядер равно 1 физическому ядру GPU. Поддерживает положительные целые числа.

- Видеопамять (дополнительно): Этот параметр активен только при наличии ресурсов GPU в кластере. Видеопамять виртуального GPU; 1 единица видеопамяти равна 256 Mi. Поддерживает положительные целые числа.
