# Описание единицы ресурсов

- CPU: Дополнительные единицы: core, m (милликор). Где 1 core = 1000 m.

- Память: Дополнительные единицы: Mi (1 MiB = 2^20 байт), Gi (1 GiB = 2^30 байт). Где 1 Gi = 1024 Mi.

- Виртуальный GPU (необязательно): Этот параметр эффективен только в тех случаях, когда в кластере есть ресурсы GPU. Количество виртуальных ядер GPU; 100 виртуальных ядер равны 1 физическому ядру GPU. Поддерживает положительные целые числа.

- Видео Память (необязательно): Этот параметр эффективен только в тех случаях, когда в кластере есть ресурсы GPU. Видеопамять виртуального GPU; 1 единица видеопамяти равна 256 Mi. Поддерживает положительные целые числа.
