# Журналы

Агрегируйте журналы времени выполнения контейнера с возможностями визуального запроса. Когда приложения, рабочие нагрузки или другие ресурсы проявляют аномальное поведение, анализ журналов помогает установить коренные причины.

## Процедура

> Эта процедура демонстрирует, как просматривать журналы времени выполнения контейнера через интерфейс развертывания.

1. **Контейнерная платформа**, нажмите **Рабочие нагрузки** > **Развертывания**.

2. Нажмите на имя развертывания из списка.

3. Перейдите на вкладку **Журналы**, чтобы просмотреть детализированные записи.

   | Операция             | Описание                                                                                                                      |
   | :------------------- | :--------------------------------------------------------------------------------------------------------------------------- |
   | **Pod**/**Контейнер** | Переключайтесь между Pod и контейнерами с помощью выпадающего селектора, чтобы просмотреть соответствующие журналы.            |
   | **Предыдущие журналы**     | Просмотр журналов из завершенных контейнеров (доступно, когда restartCount контейнера > 0).                                 |
   | **Строки**          | Настройка размера буфера журналов: 1K/10K/100K строк.                                                                        |
   | **Перенос строки**   | Переключение переноса строк для длинных записей журналов (включено по умолчанию).                                            |
   | **Найти**            | Полнотекстовый поиск с выделением совпадений и переходом по нажатию Enter.                                                  |
   | **Сырой**           | Непроцессированные потоки журналов, захваченные напрямую из интерфейсов времени выполнения контейнера (CRI) без форматирования, фильтрации или обрезки. |
   | **Экспорт**         | Скачивание сырых журналов.                                                                                                   |
   | **Во весь экран**    | Нажмите на обрезанную строку, чтобы увидеть полный контент в модальном диалоге.                                             |

:::warning

- **Обработка обрезки**: Записи журналов, превышающие 2000 символов, будут обрезаны с многоточием `...`
  - Обрезанные части не могут быть найдены с помощью функции поиска на странице.
  - Нажмите на маркер многоточия `...` в обрезанных строках, чтобы увидеть полный контент в модальном диалоге.
- **Надежность копирования**: Избегайте прямого копирования из визуализатора журналов, когда видите маркеры обрезки (...) или цветовые коды ANSI. Всегда используйте функции **Экспорт**, **Сырой** для получения полных журналов.
- **Политика хранения**: Живые журналы следуют конфигурации ротации журнала Kubernetes. Для исторического анализа используйте [Журналы](../../observability/log/functions/log.mdx) в разделе Наблюдение.
:::
