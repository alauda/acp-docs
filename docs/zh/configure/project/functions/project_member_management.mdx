# 项目成员管理

## 功能简介

项目成员管理是一个用于管理和控制项目访问权限的功能模块。它允许项目管理员通过导入现有用户或添加 OIDC 用户的方式，为项目分配成员并管理其权限。该功能可以：

- 批量授予用户项目及项目下命名空间等资源的操作权限
- 灵活分配不同级别的角色（项目管理员、命名空间管理员、开发人员等）
- 支持多种用户来源的统一管理

## 主要功能

- 成员导入：支持导入平台已有用户和 OIDC 用户
- 角色分配：为成员分配不同级别的项目角色
- 命名空间管理：支持指定角色的命名空间权限
- 成员移除：支持移除项目成员并回收相关权限
- 成员搜索：支持多维度的成员查询和过滤

## 功能优势

- 灵活性：支持多种用户来源和角色类型
- 批量操作：支持批量导入和权限分配
- 精细控制：可细化到命名空间级别的权限管理
- 安全可控：严格的用户验证和权限管理机制
- 易于管理：提供直观的成员管理界面

## 导入成员

按以下步骤导入项目成员：

<Steps>

### 进入项目成员页面

1. 在**项目管理**视图，单击项目名称
2. 在左侧导航栏中，单击**项目成员**

### 选择导入方式

支持两种导入方式：
- 成员列表：导入平台已有用户
- OIDC 用户：添加 OIDC 认证的用户

### 配置成员信息

根据不同导入方式配置相应参数：

| 参数 | 描述 |
| :--- | :--- |
| 用户名 | 用户的登录名称 |
| 角色 | 要分配的项目角色 |
| 命名空间 | 当角色需要指定命名空间时必填 |

### 确认导入

完成配置后点击"导入"按钮完成成员添加

</Steps>

## 移除成员

按以下步骤移除项目成员：

<Steps>

### 进入项目成员页面

1. 在**项目管理**视图，单击项目名称
2. 在左侧导航栏中，单击**项目成员**

### 查找目标成员

可通过以下维度过滤搜索：
- 用户名
- 显示名称
- 用户组

### 执行移除操作

1. 点击待移除成员记录右侧的**移除**按钮
2. 在确认提示框中点击**移除**完成操作

</Steps>
