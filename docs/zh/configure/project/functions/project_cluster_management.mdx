# 项目集群管理

## 功能简介

项目集群管理是一个用于管理项目与集群关联关系的功能模块。通过该功能，您可以将集群添加至项目以分配集群资源，或将已关联的集群移除以回收资源。该功能可以：

- 灵活分配集群资源给不同项目使用
- 精确控制项目可使用的资源配额
- 及时回收不再使用的集群资源

## 主要功能

- 集群添加：支持为项目添加新的集群并分配资源配额
- 资源配额：支持设置 CPU、内存、存储等多种资源的使用限制
- GPU 资源：支持配置 GPU 相关资源的分配
- 集群移除：支持移除项目关联的集群并回收资源

## 功能优势

- 资源隔离：实现不同项目间的资源隔离
- 配额管理：支持精细化的资源配额控制
- 灵活调整：可根据需求动态调整集群资源分配
- 安全可控：严格的资源分配和回收机制

## 添加集群

按以下步骤为项目添加集群：

<Steps>

### 进入项目详情页面

1. 在**项目管理**视图，单击项目名称
2. 在左侧导航栏中，单击**项目详情**
3. 单击右上角的**操作** > **添加集群**

### 配置资源配额

选择集群并设置资源配额，包括：

| 资源类型 | 说明 | 单位 |
| :--- | :--- | :--- |
| CPU | 计算资源配额 | 核 |
| 内存 | 内存资源配额 | Gi |
| 存储 | 存储资源配额 | Gi |
| PVC 数 | 持久卷声明数量 | 个 |
| Pods | Pod 数量限制 | 个 |
| GPU 资源 | GPU 相关资源配额 | 核 |
| 显存 | GPU 显存配额 | Gi/Mi |

**说明**：

- GPU 资源配置说明：
  - 仅当集群部署了 GPU 插件时可配置
  - GPU-Manager/MPS GPU 资源支持配置显存
  - 1 颗 pGPU = 1 核 = 100 GPU-Manager core = 100 MPS core
  - GPU-Manager 显存单位为 256 Mi，MPS GPU 显存单位为 1 Gi

- 配额设置说明：
  - 未输入配额默认为不限制
  - 配额需在页面提示的范围内设置
  - 可参考已分配配额和总额信息

### 确认添加

完成配置后点击**添加**按钮确认

</Steps>

## 移除集群

按以下步骤移除项目关联的集群：

<Steps>

### 进入项目详情页面

1. 在**项目管理**视图，单击项目名称
2. 在左侧导航栏中，单击**项目详情**
3. 单击右上角的**操作** > **移除集群**

### 确认移除操作

1. 在**移除集群**对话框中输入集群名称
2. 点击**移除**按钮确认操作

</Steps>
