# 简介

Kubernetes 提供灵活且可扩展的存储机制，用于在容器化环境中管理应用的数据持久性。通过抽象的存储资源（如 Volume、PersistentVolume 和 PersistentVolumeClaim），Kubernetes 能够将应用与底层存储解耦，支持动态供给、自动挂载和跨节点的数据持久化。

其核心功能包括：支持多种后端存储（如本地磁盘、NFS、云存储服务等）、动态存储供给、访问模式控制（如读写权限）以及生命周期管理，满足有状态应用的存储需求。对于需要高可用、数据持久化和多租户隔离的企业级应用，Kubernetes 存储是不可或缺的基础能力。

Kubernetes 的存储功能面向开发者、运维人员以及平台工程团队，帮助他们高效、安全地管理容器化工作负载中的数据。
