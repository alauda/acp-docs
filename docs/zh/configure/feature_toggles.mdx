# 功能开关配置 \{#feature\_toggles}

## 概述

您可以通过Web控制台自定义在平台上启用或禁用哪些功能。功能开关允许您控制功能在Web控制台中的可见性或相应API的行为。一个开关是否仅影响UI可见性，或是否还会改变API行为，取决于每个特定功能开关的实现。有关详细信息，请参考每个功能的文档。

## 步骤

### 访问功能开关页面

访问功能开关页面，地址为`{platform-access-address}/console-platform/feature-gate`。例如，如果您的平台访问地址是`https://demo.example.com/console-portal/`，功能开关页面将是`https://demo.example.com/console-platform/feature-gate`。

### 禁用所有Alpha功能

要禁用所有平台范围内的Alpha功能：

1. 导航到“阶段设置”部分
2. 关闭“Alpha功能”开关

Beta功能可以通过“阶段设置”部分进行类似管理。

### 为单个集群启用特定功能

要为单个集群启用特定功能：

1. 找到以目标集群的显示名称为标题的开关列表
2. 在该列表中启用您需要的特定功能

### 全局禁用特定功能

要在所有集群中禁用特定功能：

1. 转到“系统设置”部分
2. 关闭您希望全局禁用的功能开关
