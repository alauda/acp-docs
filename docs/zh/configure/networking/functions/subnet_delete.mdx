# 删除子网

## 注意事项

- 当子网被删除时，如果仍有容器组使用子网内的 IP 地址，则这些容器组可以继续运行，IP 地址将保持不变，但它们将无法进行网络通信。可以重建容器组以使用默认子网内的 IP 地址，或者为容器组所在的命名空间分配一个新的子网以供使用。

- 默认子网无法被删除。

## 步骤

1. 进入**平台管理**。

2. 在左侧导航栏，点击**网络管理** > **子网**。

3. 点击 ⋮ > **删除**，并继续进行删除操作。
