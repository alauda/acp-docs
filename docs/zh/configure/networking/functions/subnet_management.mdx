# 子网管理

## 管理 Kube-OVN 网络\{#ovn}

管理 Kube-OVN 网络下的子网，包括下层和上层传输网络。

### 更新网关

这包括更改出站流量方法、网关节点和 NAT 配置。

1. 转到 **平台管理**。

2. 在左侧边栏中，点击 **网络管理** > **子网**。

3. 点击子网的名称。

4. 选择 **操作** > **更新网关**。

5. 更新参数配置；有关详细信息，请参考 [参数说明](./subnet_kubeovn_overlay.mdx)。

6. 点击 **确定**。

### 更新保留 IP

网关 IP 不能从保留 IP 中删除，而其他保留 IP 可以自由编辑、删除或添加。

1. 转到 **平台管理**。

2. 在左侧边栏中，点击 **网络管理** > **子网**。

3. 点击子网的名称。

4. 选择 **操作** > **更新保留 IP**。

5. 更新完成后，点击 **更新**。

### 分配项目

将子网分配给特定项目有助于团队更好地管理和隔离不同项目的网络流量，确保每个项目有足够的网络资源。

1. 转到 **平台管理**。

2. 在左侧边栏中，点击 **网络管理** > **子网**。

3. 点击子网的名称。

4. 选择 **操作** > **分配项目**。

5. 在添加或移除项目后，点击 **分配**。

### 分配命名空间

将子网分配给特定命名空间可以实现更细粒度的网络隔离。

**注意**：分配过程将重建网关，出站数据包将被丢弃！请确保当前没有业务应用访问外部集群。

1. 转到 **平台管理**。

2. 在左侧边栏中，点击 **网络管理** > **子网**。

3. 点击子网的名称。

4. 选择 **操作** > **分配命名空间**。

5. 在添加或移除命名空间后，点击 **分配**。

### 扩展子网

当子网的保留 IP 范围达到使用上限或即将耗尽时，可以基于原始子网范围进行扩展，而不会影响现有服务的正常运行。

1. 转到 **平台管理**。

2. 在左侧边栏中，点击 **网络管理** > **子网**。

3. 点击子网的名称。

4. 选择 **操作** > **扩展子网**。

5. 完成配置后，点击 **更新**。

## 管理 Calico 网络

支持分配项目和命名空间；有关详细信息，请参考 [管理 Kube-OVN 网络](#ovn) 中的项目分配和命名空间分配部分。
