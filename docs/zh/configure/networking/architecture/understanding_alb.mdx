# 理解 ALB

ALB （另一种负载均衡器）是一个由 OpenResty 驱动的 Kubernetes 网关，拥有来自灵雀云数年的生产经验。

## 核心组件

![](../../../../en/configure/networking/assets/alb_architecture.png)

- **ALB 操作器**：一个管理 ALB 实例生命周期的操作器。它负责监视 ALB CR（Custom Resource）并为不同租户创建和更新 ALB 实例。
- **ALB 实例**：ALB 实例包括作为数据平面的 Openresty 和作为控制平面的 Go 控制器。
  Go 控制器监视各种 CR（Ingress、Gateway、Rule 等），并将它们转换为 ALB 特定的 DSL 规则。OpenResty 然后使用这些 DSL 规则来匹配和处理传入的请求。
