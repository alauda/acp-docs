# 更新和删除负载均衡器

## 更新负载均衡器

**注意**：更新负载均衡器将导致服务中断 3 到 5 分钟。请在合适的时间进行此操作！

**操作步骤**

1. 进入 **平台管理**。

2. 在左侧导航栏，点击 **网络管理** > **负载均衡器**。

3. 点击 ⋮ > **更新**。

4. 根据需要更新网络和资源配置。

   - 请根据业务需求合理设置规格。您还可以参考相关的 [负载均衡器选择和规划](../how_to/decide_performance_selection.mdx) 获取指导。

   - **内部路由** 仅支持从 **禁用** 状态更新到 **启用** 状态。

5. 点击 **更新**。

## 删除负载均衡器

**注意**：删除负载均衡器后，相关的端口和规则也将被删除，无法恢复。

**操作步骤**

1. 进入 **平台管理**。

2. 在左侧导航栏，点击 **网络管理** > **负载均衡器**。

3. 点击 ⋮ > **删除**，并确认。
