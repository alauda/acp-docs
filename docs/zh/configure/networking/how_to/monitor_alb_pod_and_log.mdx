# 日志与监控

通过将可视化的日志和监控数据结合起来，可以快速识别和解决负载均衡器的问题或故障。

## 查看日志

1. 访问 **平台管理**。

2. 在左侧导航栏中，点击 **网络管理** > **负载均衡器**。

3. 点击 ***负载均衡器名称***。

4. 在 **日志** 标签中，从容器的视角查看负载均衡器的运行日志。

## 监控指标

### 注意事项

负载均衡器所在的集群必须部署监控服务。

### 操作步骤

1. 访问 **平台管理**。

2. 在左侧导航栏中，点击 **网络管理** > **负载均衡器**。

3. 点击 ***负载均衡器名称***。

4. 在 **监控** 标签中，从节点的视角查看负载均衡器的指标趋势信息。

   - **使用率**：负载均衡器在当前节点上 CPU 和内存的实时使用情况。

   - **吞吐量**：负载均衡器实例的整体进出流量。
