# 找出造成错误的原因

错误请求的响应头中的 `X-ALB-ERR-REASON` 字段将指示错误的原因。

错误原因可能是：

```bash
InvalidBalancer : 未找到 xx 的负载均衡器 # 这意味着没有找到该服务的终端节点
BackendError : 从后端读取 xxx 字节数据 # 这意味着后端确实返回了响应，错误代码不是由 ALB 引起的。
InvalidUpstream : 没有规则匹配 # 这意味着请求与任何规则都不匹配，因此 ALB 返回 404。
```
