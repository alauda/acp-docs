# 用户管理

## 功能简介

用户管理是平台的核心功能之一，提供全面的用户生命周期管理能力。通过该功能，平台管理员可以集中管理用户账号，控制用户访问权限，并实现用户状态的灵活调整。

该功能主要解决企业在用户管理过程中遇到的以下问题：
- 用户账号的创建和维护
- 用户权限的精细化管理
- 批量用户操作的效率需求
- 用户访问控制的安全需求

## 主要功能

- 用户生命周期管理：创建、激活、禁用、删除用户
- 权限控制：角色分配与移除
- 安全管理：密码重置、有效期管理
- 用户组管理：将用户添加到本地用户组
- 批量操作：支持多用户同时操作

## 功能优势

- **灵活性**：支持单用户操作和批量操作，适应不同场景需求
- **安全性**：提供多重安全保障机制，包括密码策略和访问控制
- **可扩展性**：支持本地用户、LDAP用户等多种用户类型
- **易管理性**：提供直观的用户管理界面，支持批量操作提升效率

## 创建本地用户

<Steps>

### 访问用户管理页面

在左侧导航栏中，选择 **用户角色管理** > **用户管理**

### 创建用户

点击 **创建用户** 按钮，在弹出的对话框中填写用户信息：

| 参数 | 描述 |
|:--|:--|
| 用户名 | **必填**，用于登录平台的唯一标识 |
| 显示名称 | **必填**，用户在平台中显示的名称 |
| 密码 | **必填**，符合安全要求的登录密码 |
| 确认密码 | **必填**，重复输入密码进行确认 |
| 有效期 | **可选**，设置用户账号的有效期限 |

### 确认创建

点击 **确定** 完成用户创建

</Steps>

## 用户角色管理

<Steps>

### 添加角色

1. 导航至：**用户角色管理** > **用户管理**
2. 点击目标用户名称
3. 在角色列表区域，点击 **添加角色**
4. 在弹出对话框中：
   - 选择角色名称
   - 设置权限生效范围
5. 点击 **添加** 确认

### 移除角色

1. 导航至：**用户角色管理** > **用户管理**
2. 点击目标用户名称
3. 在角色列表中找到要移除的角色
4. 点击 **移除** 并确认操作

:::warning 注意事项
- 仅平台管理员可以管理其他用户的角色
- 管理员不能管理自己的角色
- 不支持为用户分配 global 集群的集群管理员角色
:::

</Steps>

## 用户管理操作

<Steps>

### 重置密码

1. 导航至：**用户角色管理** > **用户管理**
2. 点击操作菜单 ⋮ > **重置密码**
3. 选择密码类型：
   - 随机生成
   - 自定义（需符合密码策略）
4. 点击 **重置** 确认

### 更新有效期

1. 导航至：**用户角色管理** > **用户管理**
2. 点击操作菜单 ⋮ > **更新有效期**
3. 选择有效期类型：
   - 永久
   - 自定义时间范围

### 用户状态管理

支持以下状态转换操作：
- 激活：将禁用/锁定状态的用户恢复为正常状态
- 禁用：暂时停用用户的平台访问权限
- 删除：永久移除用户账号

### 批量操作

1. 在用户列表中选择多个用户
2. 点击 **批量操作** 选择所需操作：
   - 更新有效期
   - 激活
   - 禁用
   - 删除
3. 根据操作类型完成确认步骤

</Steps>
