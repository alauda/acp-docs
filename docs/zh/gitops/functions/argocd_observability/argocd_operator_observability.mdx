# Argo CD 组件监控

## 功能简介

通过平台 监控面板 可视化监控 Argo CD 组件，主动观测 Argo CD 组件资源及运行状态，确保健康可用。

## 前提条件 

- 平台已部署 GitOps 集群插件。
- 平台已部署 监控 集群插件。

## 查看 Argo CD 组件面板

1. 进入 **平台管理视图**，选择 **global** 集群。

2. 单击左导航 **运维中心** -> **监控** -> **监控面板**。

3. 单击 **切换** 按钮，选择 **container-platform**，可看到 **ArgoCD** 面板。 

4. 单击 **ArgoCD** 面板，可查看 Argo CD 组件监控信息。
