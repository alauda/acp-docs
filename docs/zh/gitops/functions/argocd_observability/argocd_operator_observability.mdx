# Argo CD 组件监控

## 概述

Web 控制台的监控仪表板提供了一种可视化的方法来监控 Argo CD 组件。它主动观察 Argo CD 组件的资源和操作状态，旨在确保它们的健康性和可用性。在这里，操作状态指的是 Kubernetes (K8s) 环境中组件的运行条件和性能指标。通过密切跟踪这些方面，我们可以及时发现并解决任何潜在问题，保持 Argo CD 在 K8s 集群中的顺利运行。

## 前提条件

- [安装 ACP GitOps](/gitops/install_gitops/install_gitops_plugin.mdx)
- [安装监控插件](/observability/monitor/install_monitor.mdx)

## 查看 Argo CD 组件仪表板

1. 登录并导航到 **平台管理**，选择 `global` 集群。

2. 点击 **运维中心** > **监控** > **监控仪表板**。

3. 点击 **切换** 按钮，选择 **container-platform** 查看 **ArgoCD** 仪表板。

4. 点击 **ArgoCD** 仪表板以查看 **Argo CD** 组件监控信息。
