# 深入理解 Health

## Health 介绍

Health 用于标记 Argo CD 的 Application 的健康状况，是否正常运行？是否可以处理请求？

## Health 范围

| 健康状态       | 说明                                                                 |
| :------------- | :-------------------------------------------------------------------- |
| **Health**     | 资源健康。                                                            |
| **Progressing**| 资源尚未健康，但仍在处理过程中，应该很快就会达到 **Health** 状态。 |
| **Degraded**   | 资源健康状况降级。                                                  |
| **Suspended**  | 资源已暂停并等待某些外部事件恢复，例如：暂停的 CronJob 或暂停的 Deployment。 |

## 参考资料
Argo CD 为几种标准 Kubernetes 类型提供内置健康评估，然后将其反映到整个应用程序的健康状态中，当然 Argo CD 也支持自定义健康检查。更加详细的说明请参考：[Health](https://argo-cd.readthedocs.io/en/stable/operator-manual/health/)
