# 创建一个 SpintBoot Argo CD Application

介绍通过平台 UI 创建一个 SpingBoot Argo CD Application 应用。

## 使用场景

- 使用平台 UI 创建一个 SpingBoot Argo CD Application 应用，体验通过 GitOps 管理应用的完整流程。

## 前置条件

- 已在 **global** 集群中部署 GitOps 集群插件。

- 平台上已分配好项目、命名空间。


## 操作步骤

> 按以下步骤使用功能：

1. 登录平台，在左侧导航栏点击 **GitOps 应用**，进入 **GitOps 应用** 列表页面。

2. 点击 **创建**，进入 **创建应用** 页面。

3. 选择 **Argo CD Application**，进入 **Argo CD Application** 创建页面。

4. 选择 **项目**、**命名空间**，填写 **应用名称**，选择 **集群**，选择 **源代码**，选择 **目标集群**，点击 **创建**。
