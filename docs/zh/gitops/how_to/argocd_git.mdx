# 使用 Argo CD UI 集成代码仓库

使用 Argo CD 原生 UI 集成代码仓库并分配仓库，方便业务人员通过可视化页面实现 GitOps 应用管理全流程。

## 使用场景

- 简化部署 GitOps 应用流程，通过平台 UI **创建 GitOps 应用** 时，选择使用已关联的仓库。
- 通过 Argo CD 原生 UI 创建 **Application** 时，可选择使用已关联的仓库。

## 前置条件

- 已在 **global** 集群中部署 GitOps 集群插件，**Argo CD 原生 UI** 开关已开启。
- 已获取到 **Argo CD 原生 UI** 访问地址、账号及密码；如平台管理员，可通过 GitOps 集群插件详情页面直接访问 **Argo CD 原生 UI**。

## 操作步骤

按以下步骤使用功能：

1. 连接代码仓库

   - 通过访问地址登录 **Argo CD**。

   - 在左侧导航栏中，单击 **Settings**。

   - 单击 **Repositories** 卡片。

   - 单击页面左上角 **CONNECT REPO**。

   - 按需选择连接仓库的方式并填写对应参数。

   - 单击 CONNECT。

2. 关联项目

   - 在左侧导航栏中，单击 **Settings**。

   - 单击 **Projects** 卡片。

   - 单击页面中需要创建 GitOps 应用的项目。

   说明：Argo CD 会自动同步集群中的项目，无需手动创建。

   - 单击 **SOURCE REPOSITORIES** 区域的 **EDIT**。

   - 单击 **ADD SOURCE**，填写 **连接仓库** 中的仓库地址，与项目进行关联。

   - 单击 **SAVE**。

## 操作结果

返回到平台，访问 **创建 GitOps 应用** 页面，即可看到已关联的仓库。
