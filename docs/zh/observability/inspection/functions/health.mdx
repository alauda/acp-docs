# 组件健康状态

平台健康状态页面，将为您呈现平台已部署功能的健康状态统计数据。当您的账号拥有平台管理、平台审计相关权限时，您还可以查看指定功能的详细健康数据，包括：未部署该功能的集群列表、部署了该功能的集群的健康状态以及该功能关联的集群下的组件的检测数据。能够帮助您快速定位问题，提升平台运维效率。


## 操作步骤

1. 进入已部署的产品视图或平台中心（平台管理、项目管理、运维中心）视图页面。

2、单击顶部导航栏右上角的问号按钮> **平台健康状态**。
    
3、查看功能卡片，功能卡片展示了功能的健康状态信息，若功能组件存在异常，会在卡片中体现为 `故障`.
          
4、单击功能卡片上的健康/故障数值，可在页面右侧展开功能健康状态详情页面，您可在该页面查看故障组件的详细问题信息。
