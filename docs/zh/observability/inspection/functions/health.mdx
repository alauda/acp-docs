# 组件健康状态

平台健康状态页面将为您呈现平台已安装功能的健康状态统计数据。当您的账号拥有平台管理或审计权限时，您还可以查看指定功能的详细健康数据，包括：未安装该功能的集群列表、安装了该功能的集群的健康状态，以及与该功能相关的集群内组件的检测数据。这可以帮助您快速识别问题，提高平台的运维效率。

## 操作步骤

1. 进入已安装产品的视图页面或平台中心（平台管理、项目管理、运维中心）。

2. 单击顶部导航栏右上角的问号按钮 > **平台健康状态**。

3. 查看功能卡片；功能卡片展示了该功能的健康状态信息。如果功能组件存在异常，则会在卡片中显示为 `故障`。

4. 单击功能卡片上的健康/故障数值，可在页面右侧展开功能健康状态详细页面，您可以在该页面查看故障组件的详细问题信息。
