# 关于可观测

## 产品介绍

可观测模块是一个以应用为核心、开箱即用的云原生可观测性平台，专为企业提供全面的监控解决方案。它能够实时监测应用及其资源，采集各类指标、日志和事件数据，帮助分析应用的健康状态。该模块不仅具备强大的告警功能，还提供全面、清晰的多维度数据可视化，兼容主流开源组件，支持快速故障定位和一键监控诊断。


## 产品优势

可观测模块的核心优势如下：

- 统一数据采集

  实现指标、日志和链路的统一采集，提供全方位的系统视图，简化管理流程。

- 多维度告警机制

  支持对指标和日志进行多维度的告警设置，确保用户及时获知潜在问题。

- 直观可视化界面

  提供简洁明了的可视化管理界面，让用户快速获取关键信息，提升决策效率。

- 兼容性强

  与主流开源组件完美兼容，用户能够轻松集成，提升系统灵活性。

- 开箱即用

  提供预配置的模板和最佳实践，用户无需复杂配置即可快速上手。    


## 产品应用场景

可观测模块的主要应用场景如下：

- 微服务架构监控

  在云原生微服务架构中，应用由多个独立服务组成。可观测模块能够实时监控各个微服务的性能和健康状态，帮助开发团队快速识别和解决服务间的依赖问题，确保系统整体的稳定性。

- 容器和Kubernetes监控

  对于部署在Kubernetes环境中的容器应用，可观测模块可以监控容器的资源使用情况、状态和日志，提供容器的生命周期管理和故障排除，确保容器化应用的高可用性。

- 动态资源管理

  在云环境中，资源使用量可能会随业务需求变化而波动。可观测模块能够实时监控资源的使用情况，支持动态调整资源分配，优化成本和性能。

- 多云环境监控

  对于采用多云策略的企业，可观测模块能够统一管理和监控不同云平台上的应用和资源，确保跨云环境的可见性和一致性。
