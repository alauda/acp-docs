# 核心概念

## 监控
### 指标（Metrics）
指标用于量化描述系统的运行状态，每个指标包含四个基本要素：
- 指标名称：用于标识监控对象，如 `cpu_usage`
- 指标值：具体的测量数值，如 `85.5`
- 时间戳：记录测量时间点
- 标签：用于多维度数据分类，如 `{pod="nginx-1", namespace="default"}`

### PromQL
PromQL 是 Prometheus 查询语言，用于从监控系统中查询和聚合指标数据。 

### 内置指标（Indicator）
平台基于长期运维经验，预置了一系列常用的监控指标。您可以在配置告警规则或创建监控面板时直接使用这些指标，无需额外配置。

### Exporter
Exporter 是监控数据采集组件，主要职责包括：
- 从目标系统收集原始监控数据
- 将数据转换为标准的时序指标格式
- 通过 HTTP 接口提供指标数据供查询

### ServiceMonitor
ServiceMonitor 用于声明式地管理监控配置，主要定义：
- 监控目标的选择规则
- 指标采集的接口配置
- 采集任务的执行参数（间隔、超时等）

## 告警

### 告警规则 
告警规则定义了触发告警的具体条件：
- 告警表达式：使用 PromQL 语句描述告警触发条件
- 告警阈值：明确的触发界限值
- 持续时间：条件需持续满足的时长
- 告警级别：区分告警的严重程度（如 P0/P1/P2）

### 告警策略
告警策略将多个告警规则组织在一起，统一配置：
- 告警对象：规则作用的目标范围
- 通知方式：告警发送的渠道
- 发送间隔：告警重复通知的时间间隔

## 通知 

### 通知策略
通知策略管理告警消息的发送规则：
- 接收人：告警通知的目标用户
- 通知渠道：支持的消息发送方式
- 通知模板：消息内容的格式定义

### 通知模板
通知模板定制告警消息的展示形式：
- 标题模板：告警消息的标题格式
- 内容模板：告警详情的组织方式
- 变量替换：支持动态数据填充

## 监控面板

### 面板
面板是多个相关图表的集合展示，用于提供系统状态的整体视图。支持灵活的布局方式，可以按行或列组织图表。

### 图表
图表是监控数据的可视化展现形式，支持多种展示类型：

| 序号 | 图表名称 | 说明 | 使用场景建议 |
| ---- | ---- | --- |----|
|**1**|趋势图|通过折线展示数据随时间变化的趋势，支持多条折线对比。支持配置数据点、图例位置、Y轴范围等。|适用于展示连续变化的时序数据，如 CPU 使用率、内存使用量、网络流量等指标的变化趋势。|
|**2**|阶梯折线图|使用阶梯状的线段连接数据点，突出显示数值的离散变化。支持配置阶梯样式、数据标签等。|适用于展示状态变化或事件发生，如告警次数、容器重启次数、部署版本变更等。|
|**3**|柱状图|使用垂直柱子表示数据大小，支持分组对比、堆叠展示、排序等多种呈现方式。|适用于多维度数据对比，如各命名空间的容器组数量、不同类型告警的分布统计等。|
|**4**|条形图|使用水平条形表示数据，当标签较长或对比项较多时更易读。支持排序和多层分组。|适用于展示多个维度的对比数据，如 Top N 进程的资源占用、各节点的负载情况等。|
|**5**|环形测量图|用环形或半环形展示单个指标的当前值及占比，支持多阈值颜色区分和动态告警阈值。|适用于展示资源使用率类指标，如 CPU 使用率、内存使用率、磁盘使用率等。|
|**6**|柱形测量图|使用垂直柱形展示单个指标值，支持设置多个阈值和颜色区间，可显示历史对比数据。|适用于展示需要阈值预警的指标，如请求延迟、错误率、队列深度等。|
|**7**|饼图|展示整体中各部分的占比关系，支持突出显示重要部分，可配置图例位置和标签样式。|适用于展示分布占比，如 HTTP 状态码分布、资源分配比例、告警级别分布等。|
|**8**|表格|以行列方式展示结构化数据，支持排序、过滤、分页等交互操作，可嵌入状态图标。|适用于展示详细的监控数据，如节点列表、容器组状态、告警历史记录等。|
|**9**|统计图|突出展示单个核心指标的实时值，支持显示环比变化、趋势图表、预警阈值等。|适用于展示关键指标的即时状态，如在线节点数、告警总数、服务可用性等。|
|**10**|散点图|在坐标系中展示数据点分布，支持气泡大小映射、颜色分组等多维度展示。|适用于分析指标相关性，如响应时间与并发量、CPU与内存使用关系等。|
|**11**|文本卡片|展示文字信息，支持 Markdown 格式、变量替换、链接跳转等功能。|适用于展示说明文档、操作指南、告警说明等静态或动态文本内容。|

### 数据源
监控数据的来源配置。目前仅支持使用当前集群的监控组件作为数据源，暂不支持自定义数据源。

### 变量
变量作为数值的占位符，可用于指标查询中。通过面板顶部的变量选择器，您可以动态调整查询条件，使图表内容实时更新。
