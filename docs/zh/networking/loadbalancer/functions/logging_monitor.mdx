# 日志与监控


结合可视化的日志及监控数据，当负载均衡器出现异常或故障时，可快速定位并解决问题。

## 查看日志

1. 进入 **平台管理**。

1. 在左侧导航栏中，单击 **网络管理** > **负载均衡器**。

3. 单击 ***负载均衡器的名称***。

4. 在 **日志** 页签中，从容器维度，查看负载均衡器运行时的日志。

 
## 监控指标

### 注意事项

负载均衡器所在集群需部署监控服务。

### 操作步骤

1. 进入 **平台管理**。

1. 在左侧导航栏中，单击 **网络管理** > **负载均衡器**。

3. 单击 ***负载均衡器的名称***。

4. 在 **监控** 页签中，从节点维度，查看负载均衡器的指标趋势信息。

    
    * **使用率**：当前节点的负载均衡器对 CPU、内存的实时使用情况。

    * **吞吐量**：负载均衡器实例整体的流入及流出流量。
