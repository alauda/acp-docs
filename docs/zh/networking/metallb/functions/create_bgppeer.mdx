# 创建 BGP 对等体


在不同 AS 或同一 AS 内建立连接以交换路由信息的节点，这些节点之间通过 BGP 协议进行通信。

## 名词解释

|名词|解释|
|---|---|
|**AS 号**|AS 是指由同一个技术管理机构管理、使用统一选路策略的一些路由器的集合。BGP 网络中的每个 AS 都被分配一个唯一的 AS 号，用于区分不同的 AS，AS 号分为 2 字节 AS 号和 4 字节 AS 号。<ul><li>2 字节 AS 号的范围为 1~65535，其中 1-64511 是互联网上注册公有 AS 号，类似公网IP地址；64512~65535 是私有 AS 号，类似私网 IP 地址。</li><li>4 字节 AS 号的范围为 1~4294967295。</li></ul>支持 4 字节 AS 号的设备能够与支持 2 字节 AS 号的设备兼容。|


## 前提条件

请联系管理员开启相关功能。


## 操作步骤

1. 进入 **平台管理**。

1. 在左侧导航栏中，单击 **网络管理** > **BGP 对等体**。

2. 单击 **创建 BGP 对等体**。

3. 参考以下说明，配置参数。

    |参数|说明|
    |---|---|
    |**本地 AS 号**|BGP 连接节点所在 AS 的 AS 号。<br/><br/><b>说明</b>：若无特殊要求，建议使用 IBGP 配置，即本地 AS 号与对端 AS 号保持一致。|
    |**对端 AS 号**|BGP 对等体所在 AS 的 AS 号。|
    |**对端 IP**|BGP 对等体的 IP 地址，此地址需要是能够进行 BGP 连接的有效 IP 地址。|
    |**本地 IP**|BGP 连接节点的 IP 地址，当 BGP 连接节点有多个 IP 时，选取指定的本地 IP 和对端建立 BGP 连接。|
    |**对端端口**|BGP 对等体的端口号。|
    |**BGP 连接节点**|建立 BGP 连接的节点，若不配置此参数，则所有节点都会建立 BGP 连接。|
    |**eBGP Multi-Hop**|允许不直接相连的 BGP 路由器之间建立 BGP 会话。启用此功能后，BGP 报文的 TTL 值默认为 5，从而可以跨越多个中间网络设备建立 BGP 对等体关系，使网络设计更加灵活。|
    |**RouterID**|32 位的数值（通常以点分十进制形式表示，类似 IPv4 地址格式），用于在 BGP 网络中唯一标识一台 BGP 路由器，一般用于 BGP 邻居关系建立、路由环路检测、最优路径选择及网络故障排查等。|

4. 单击 **创建**。
