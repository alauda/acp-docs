# 介绍

## 域名

设置用于从外部访问计算组件的域名地址。

平台上创建的域名需要和集群的负载均衡地址做解析之后，才可通过域名访问集群下的计算组件。因此，需要您确保在平台上添加的域名已成功注册并通过域名解析将域名指向了集群的负载均衡地址。

平台管理员通过域名管理功能，可统一管理用于本平台的企业网络域名资源，并通过将域名绑定到项目，实现项目间域名资源的分配和管理。

**注意**：平台上创建的域名需要和集群的负载均衡地址做解析之后，才可通过域名访问集群下的计算组件。因此，需要您确保在平台上添加的域名已成功注册并通过域名解析将域名指向了集群的负载均衡地址。

平台上创建并分配成功的域名，可用于以下几个功能模块中：

* 入站规则

* 自定义应用 - 添加入站规则

* 负载均衡器 - 添加监听端口

# 证书


通过入站规则或负载均衡器向外部提供业务的 HTTPS 访问前，需为访问域名或地址准备 TLS 证书用于 HTTPS 加密。
