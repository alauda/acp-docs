# ARM 环境中出现跨节点通信问题该如何解决？

当使用较低的内核版本及某些国产网卡时，可能存在启用 Checksum Offload 后网卡计算校验和（Checksum）出现错误的问题，从而导致 Kube-OVN Overlay 网络下节点间通信出现故障，具体的解决方案如下：

* **方案一：升级内核版本**。建议将内核版本升级至 4.19.90-25.16.v2101 或更高版本。

* **方案二：关闭 Checksum Offload**。若无法立即升级内核版本且出现跨节点通信问题，可通过如下命令关闭物理网卡的 Checksum Offload。

    ```
    ethtool -K eth0 tx off
    ```
