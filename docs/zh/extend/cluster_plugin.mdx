# 集群插件

## 概述

集群插件允许您通过安装额外的组件来扩展平台的功能。本文件指导您查看、安装和升级平台上的集群插件。

## 查看可用插件

要查看平台提供的所有插件：

1. 导航到平台管理视图
2. 点击左侧导航菜单：**平台管理** > **市场** > **集群插件**

此页面显示所有可用插件及其当前状态。

## 安装插件

如果某个插件显示“缺失”状态，请按照以下步骤安装它：

1. 下载插件包：
   - 访问自定义门户下载相应的插件包
   - 如果您无法访问自定义门户，请联系技术支持

2. 将包上传到平台：
   - 使用 `violet` 工具将包发布到平台
   - 有关使用此工具的详细说明，请参阅 [CLI](/ui/cli_tools/index.mdx)

3. 验证上传：
   - 导航到 **平台管理** > **市场** > **上传包**
   - 切换到 **集群插件** 标签
   - 找到上传的插件名称
   - 插件详细信息将显示上传到平台的版本信息

4. 安装插件：
   - 如果插件显示“准备好”状态，请点击 **安装**
   - 某些插件需要安装参数；请参阅特定插件的用户文档以获取参数详细信息
   - 有些插件没有安装参数，点击安装后将立即开始安装

## 升级插件

要将现有插件升级到新版本：

1. 上传新版本：
   - 按照相同的过程将插件的新版本上传到平台

2. 验证新版本：
   - 导航到 **平台管理** > **市场** > **上传包**
   - 切换到 **集群插件** 标签
   - 在相应插件的包详细信息中，您可以看到已上传的新版本

3. 执行升级：
   - 导航到 **平台管理** > **集群** > **集群**
   - 安装了可升级插件的集群将显示一个升级图标
   - 输入集群详细信息并切换到 **特性** 标签
   - 特性组件下的升级按钮将被激活
   - 点击 **升级** 以完成插件升级过程

## 支持

如果您在安装或升级过程中遇到任何问题，请联系技术支持以获得帮助。
