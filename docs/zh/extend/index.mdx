# 扩展

该平台提供了一个全面的扩展系统，允许用户增强其 Kubernetes 集群的功能。此系统旨在灵活且用户友好，使用户能够轻松地为其集群添加新的功能和能力。

该系统由两种主要的扩展类型组成：

- 操作器：操作器基于操作器生命周期管理器（OLM）v0 框架构建，为平台提供专业的操作能力。这些扩展支持对集群内复杂应用程序和服务的自动管理。

- 集群插件：该平台具有专有的集群插件系统，专门为 Chart 类型的插件设计。该系统提供了比标准方法更优的安装和管理体验，并为处理基于 Chart 的扩展提供了用户友好的界面。

凭借对众多操作器和集群插件的支持，用户可以显著扩展平台的能力，以满足具体的操作需求和使用场景。
