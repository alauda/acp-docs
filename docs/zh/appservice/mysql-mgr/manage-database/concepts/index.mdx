## 概述

管理 MySQL-MGR 数据库实例涉及多个核心概念，理解这些概念对于有效管理数据库实例至关重要。本节将介绍管理数据库实例时涉及的主要概念，包括实例生命周期、规格管理、版本管理、调度配置和维护窗口等。

这些概念共同构成了 MySQL-MGR 数据库实例的管理基础，理解它们之间的关系和影响有助于您更好地规划和管理数据库实例，确保数据库服务的高可用性和性能。
