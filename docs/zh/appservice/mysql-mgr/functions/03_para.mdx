# 参数配置

## 核心参数
```yaml
group_replication_group_name: "acec66a9-1733-11ee-ae33-00163e0c9b29"
group_replication_start_on_boot: ON
group_replication_consistency: BEFORE
group_replication_flow_control_mode: QUOTA
```

## 操作步骤
<Tabs>
<Tab label="命令行操作">
1. 在线修改参数
```bash
kubectl patch mysqlcluster $name -n $namespace --type merge -p '
spec:
  config:
    mysql:
      group_replication_auto_increment_increment: 3
'
```

2. 滚动生效
```bash
kubectl rollout restart mysqlcluster $name -n $namespace
```
</Tab>

<Tab label="界面操作">
1. 进入实例 **参数配置**
2. 选择修改方式：
   - 基础参数：通过表单修改
   - 高级参数：直接编辑YAML
3. 验证语法：
   ```
   SELECT @@group_replication_consistency;
   ```
4. 提交变更并观察滚动更新状态
5. 回滚操作：
   - 选择历史版本
   - 确认回滚
</Tab>
</Tabs>
