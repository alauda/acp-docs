# 删除实例

## 删除保护
- 级联删除控制
- 最终一致性检查
- 备份保留策略

## 操作步骤
<Tabs>
<Tab label="命令行操作">
1. 删除实例
```bash
kubectl delete mysqlcluster $name -n $namespace
```

2. 清理持久化数据
```bash
kubectl get pvc -n $namespace -l app.kubernetes.io/instance=$name -o name | xargs kubectl delete
```

3. 保留备份
```bash
kubectl label backups -n $namespace -l cluster=$name protected=true
```
</Tab>

<Tab label="界面操作">
1. 进入实例 **概览**
2. 点击 **删除实例**
3. 选择删除选项：
   - 立即删除
   - 延迟删除（安全期）
4. 确认删除：
   - 关联资源检查
   - 最终确认
5. 删除后操作：
   - 查看操作审计
   - 清理监控数据
</Tab>
</Tabs>
