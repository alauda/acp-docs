# 用户管理

## 功能简介
提供完整的用户生命周期管理，包括创建、权限分配、密码修改和删除操作，支持RBAC权限模型。

## 安全特性
- 自动生成Kubernetes Secret存储凭证
- 支持TLS加密连接
- 审计日志记录

## 操作步骤
<Tabs>
<Tab label="命令行操作">
1. 创建用户
```bash
kubectl exec -n $namespace $name-primary -- mysql -uroot -p$password -e "CREATE USER 'webuser'@'%' IDENTIFIED BY 'password';"
```

2. 授权
```bash
kubectl exec -n $namespace $name-primary -- mysql -uroot -p$password -e "GRANT SELECT, INSERT ON appdb.* TO 'webuser'@'%';"
```

3. 生成Secret
```bash
kubectl create secret generic webuser-creds -n $namespace \
  --from-literal=username=webuser \
  --from-literal=password=password
```
</Tab>

<Tab label="界面操作">
1. 进入实例 **用户管理** 页签
2. 点击 **创建用户**
3. 配置参数：
   - 用户名：符合命名规范
   - 密码：符合复杂度要求
   - 权限：库/表级权限
4. 高级配置：
   - 连接限制
   - SSL/TLS要求
5. 生成访问凭证Secret
6. 验证用户连接：
   ```bash
   mysql -uwebuser -p -h<endpoint> --ssl-mode=REQUIRED
   ```
</Tab>
</Tabs>
