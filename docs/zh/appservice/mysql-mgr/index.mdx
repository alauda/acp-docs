# Alauda Database Service for MySQL-MGR

Alauda Database Service for MySQL-MGR 是一个基于 MySQL Group Replication 技术的高可用数据库解决方案，运行在 Kubernetes 上。它提供了自动化部署和管理 MySQL Group Replication 集群的能力。
