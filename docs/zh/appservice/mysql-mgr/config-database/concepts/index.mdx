## 概述

MySQL-MGR 配置数据库实例涉及多个核心概念，理解这些概念对于正确配置和管理数据库实例至关重要。本节将介绍配置数据库实例时涉及的主要概念，包括 Group Replication 技术、参数模板、实例规格、副本数、连接地址、存储类型和参数配置等。

这些概念共同构成了 MySQL-MGR 数据库实例的配置基础，理解它们之间的关系和影响有助于您做出更合理的配置决策，优化数据库性能和可靠性。
