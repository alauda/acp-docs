# 介绍

## 灵雀云应用服务概述

灵雀云应用服务是一个基于容器和Kubernetes技术的服务管理平台，专为开发人员和运维人员量身定制。它无缝集成了快速部署、精简管理和直观的可视化操作，旨在解决企业的部署和运营挑战，为业务应用开发提供强有力的支持。

## 灵雀云应用服务的优势

灵雀云应用服务的核心优势如下：

- 企业级服务可用性和数据安全性

灵雀云应用服务平台提供行业领先的企业级服务可用性和数据安全性，为企业核心运营提供坚定的支持。借助先进的高可用架构和强大的数据冗余机制，它确保在网络故障、停电、系统关闭、重启和重负载等五种关键高可用场景下，业务持续不间断。该平台结合了多层数据安全措施，如数据冗余、自动备份、加密传输、访问控制、实时监控和安全审计，以保护企业数据在存储、处理和传输阶段的安全。这使得企业能够自信地利用该平台来管理和访问其关键数据，确保万无一失。

- 云原生统一操作管理

一个强大的一体化平台，提供生命周期管理、可观察性、变更追踪、备份与恢复、审计和灾难恢复功能。通过无缝集成和先进的自动化，它为企业提供了一个精简、一致的界面，以轻松管理应用服务。

- 即插即用的简单性

通过直观的界面简化了部署和开发，使开发人员能够立即利用灵雀云应用服务。关键功能已准备好使用，支持快速、经济高效的服务实例启动，无需复杂的设置。

- 优化的资源利用

通过在Kubernetes（K8S）上进行智能调度，最大化资源效率，确保每个计算单元被有效利用。动态分配根据实时工作负载进行调整，而基于趋势的规划则最大限度地减少浪费和瓶颈，从而在云中提供经济高效的性能和敏捷响应能力。

- 高度可扩展

支持原生插件以实现实例旁无缝服务管理，同时支持定制插件集成以扩展功能，轻松适应不断变化的业务需求。

## 灵雀云应用服务的应用场景

灵雀云应用服务的主要应用场景如下：

- 统一IT运营管理

通过灵雀云应用服务平台，运维团队能够对多个集群和项目的服务进行集中控制。这个一站式接口提供了对所有企业应用服务的可视化，能够高效地进行部署、监控、告警、日志记录、备份和恢复。结果是优化了运营，提高了效率，降低了成本。

- 企业级应用部署与运营

这是灵雀云应用服务平台的核心场景，使企业能够以稳定、可靠和高性能的方式部署和管理生产中的应用。优化的资源利用确保了符合企业需求的强大服务体验。

- 开发和测试环境

该平台旨在提供敏捷性，使开发人员和测试人员能够通过简单的表单独立启动轻量级的灵雀云应用服务实例。这加速了开发和测试环境的创建，提升了效率，同时以最小的开销推进工作。

## 灵雀云应用服务的使用限制

在生产环境中使用灵雀云应用服务时，应注意以下限制：

- 存储类

  使用平台推荐的存储类以确保数据安全和可靠性。

- 部署依赖

  确保目标集群已部署灵雀云应用服务的依赖组件。
