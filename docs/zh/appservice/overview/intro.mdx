# 介绍

## 应用服务介绍

应用服务平台是基于容器与 Kubernetes 技术， 构建的面向开发者和运维人员的服务管理模块，集快速部署、便捷管理、可视化运维于一体，致力于帮助企业解决部署和运维难题，为企业开发业务应用提供有力支撑。

## 应用服务优势

应用服务的核心优势如下：

- 企业级服务可用性和数据安全性

  提供了业界领先的企业级服务可用性和数据安全性，旨在为企业核心业务运营提供坚如磐石的支持。通过采用先进的高可用架构和数据冗余机制，能够在断网、断电、关机、重启和高负载五大高可用挑战场景下，都可以保证业务的连续性。产品实施了多层次的数据安全措施，包括数据冗余、数据备份、加密传输、访问控制、实时监控和安全审计，来确保企业数据在存储、处理和传输过程中的安全性，让企业可以放心采用本产品去支撑他们最宝贵的资产————数据。

- 基于云原生的统一运维管理

  支持生命周期管理、可观测、变更管理、备份恢复、审计、容灾等全方位功能的统一运维管控平台。高集成度和自动化能力，为企业提供简洁、一致的界面来管理其应用服务。

- 开箱即用

  简化部署与开发过程，帮助开发人员自如的使用应用服务，为关键特性提供开箱即用的操作界面，用户可以低成本的快速启动服务实例。

- 充分利用底层资源

  基于 Kubernetes（K8S）平台实现精准的智能调度机制，最大化资源利用率，确保每一份计算资源都得到最有效的使用。企业可以根据实时负载动态调整资源分配，还能够根据业务趋势进行资源规划，从而避免不必要的资源浪费和性能瓶颈，在云计算环境中以最优的成本效益运行他们的应用，保持业务的敏捷性和响应速度。

- 易扩展

  支持各服务的原生插件，跟随实例进行管理，同时也支持自行添加插件，进一步扩展服务的能力。

## 应用服务应用场景

应用服务的主要应用场景如下：

- IT 运维统一管理

  在应用服务平台上，运维团队可以通过集中的界面，对各项目下进行跨集群的服务管理，最终实现一站式的运维管理。在视图中，可以查看企业内的所有应用服务，并支持对服务的部署、监控、告警、日志、备份、恢复等操作，提高运维效率，降低运维成本。

- 企业级应用部署与运维

  企业级应用部署与运维是应用服务平台的核心应用场景之一，企业可以在生产环境中部署相关的应用服务，已获得稳定可靠、高性能、高资源利用的服务体验。

- 开发测试环境

  支持开发与测试人员自助拉起应用服务，通过表单可以快速创建低开销的应用服务实例，提高开发测试效率。

## 应用服务使用限制

在生产环境中使用应用服务需要注意以下限制:

- 存储类

  使用平台推荐的存储类，以保证数据的安全性和可靠性。

- 部署依赖

  确保目标集群已经部署了应用服务的依赖组件。
