# 架构

![](../../../en/appservice/overview/assets/appservice_architecture_en.png)

灵雀云应用服务平台是一个基于 Kubernetes (K8S) 的强大开源服务管理解决方案，旨在无缝支持灵雀云应用服务的操作。它作为一个集中管理的中心，有效地编排各种流行的服务组件，包括 MySQL、Redis、Kafka 和 RabbitMQ。以下是其工作原理：

- 服务组件：该平台支持 MySQL、Redis、Kafka 和 RabbitMQ 等关键服务，确保所有组件的稳定、可靠运行。

- 常见服务：它提供参数模板、检查、告警管理、备份服务等，为用户提供一站式服务管理解决方案。

- 运维控制器：运维控制器是 Kubernetes 的扩展，旨在简化应用程序的部署和管理。它通过自定义资源定义（CRD）扩展 Kubernetes API，以管理应用程序的生命周期。

- 容器基础设施：灵雀云应用服务平台基于 Kubernetes 构建，利用其提供的基础设施，提供数据持久性和外部访问等基础功能。
