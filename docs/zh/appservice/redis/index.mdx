# Alauda Cache Service for Redis

Alauda Cache Service for Redis OSS 是一款开源的内存数据结构存储系统，支持多种数据结构，例如字符串、哈希、列表、集合和有序集合，并提供持久化、高可用性和分布式集群等功能。由于其高性能和低延迟特性，它被广泛应用于缓存、消息队列和实时数据分析等场景。
