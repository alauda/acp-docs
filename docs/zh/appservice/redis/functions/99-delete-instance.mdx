# 删除实例

您可以创建一个 Redis 实例，以获得更快的读写速度和可扩展性服务，满足业务的多样化和高可用性需求。

## 创建 Redis 实例

### 前提条件

请查阅 [支持的存储类]()，确保当前集群存在该类型的存储类，且可用空间充足。

### 操作步骤

1. 在左侧导航栏中，单击 **Redis**。

2. 单击 ***命名空间的名称***。

3. 单击 **创建 Redis 实例**。

4. 选择一个架构类型。

5. 参考以下说明完成相关配置。

6. 单击 **创建**。

    待实例状态变为 **运行中** 时，实例创建成功。

7. 访问实例，并检查实例连接状态。

**提示**：您可以在创建时配置实例支持 IPv6 访问，请参考 [如何配置中间件实例支持 IPv6 访问？]
