# Alauda 数据库服务 for MySQL-PXC

Alauda 数据库服务 for MySQL-PXC 是一个高可用性数据库解决方案，基于 Percona XtraDB Cluster 使用同步复制，运行在 Kubernetes 上。它提供了对 Percona XtraDB Cluster 中对象的自动部署和管理能力，并且还可以用于启动新的 Percona XtraDB Cluster 实例或扩展现有实例。
