Alauda Database Service for MySQL-PXC 是一种高可用性数据库解决方案，基于 Percona XtraDB Cluster 的同步复制，在 Kubernetes 环境中运行。它提供了对 Percona XtraDB Cluster 中对象的自动化部署和管理能力，同时也可以用于启动新的 Percona XtraDB Cluster 实例或扩展现有实例。
