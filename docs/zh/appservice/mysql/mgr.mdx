# Alauda 数据库服务 for MySQL-MGR

Alauda 数据库服务 for MySQL-MGR 是一个基于 MySQL Group Replication 技术的高可用数据库解决方案，运行在 Kubernetes 上。它提供了 MySQL Group Replication 集群的自动化部署和管理能力。
