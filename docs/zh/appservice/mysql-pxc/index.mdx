# Alauda Database Service for MySQL-PXC

Alauda Database Service for MySQL-PXC 是一种高可用性数据库解决方案，基于 Percona XtraDB Cluster 使用同步复制，运行在 Kubernetes 上。它提供了自动化部署和管理 Percona XtraDB Cluster 中对象的能力，也可以用来启动新的 Percona XtraDB Cluster 实例或扩展现有实例。
