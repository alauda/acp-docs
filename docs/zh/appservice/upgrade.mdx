# 升级

灵雀云应用服务基础组件旨在平台升级时自动升级，无需人工干预。系统在后台无缝处理所有必要的迁移和更新。

灵雀云应用服务 RDS 框架将根据配置的升级策略执行升级：

- **自动**：检测到新组件版本后立即触发自动升级。
- **手动**：在启动升级过程之前需要手动批准。

:::warning
当 ACP 平台从 v3 升级到 v4 时，建议您及时将 global 集群和业务集群同时升级至新版本，以确保 Alauda Application Services 视图显示正常。  
如果仅 global 集群升级到 v4，而业务集群仍是 v3，此期间请联系技术团队获取“未升级业务集群的 Alauda Application Services 视图显示“ 的解决方案。
:::
