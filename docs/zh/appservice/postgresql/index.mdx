# Alauda Database Service for PostgreSQL

Alauda Database for PostgreSQL 是基于 Zalando Postgres Operator 构建的 Kubernetes 原生 PostgreSQL 管理解决方案。它通过自定义资源定义（CRD）实现 PostgreSQL 集群的全生命周期管理，支持自动故障转移、滚动升 级、备份恢复等核心运维能力，帮助企业在 Kubernetes 上轻松运行生产级 PostgreSQL 数据库。
