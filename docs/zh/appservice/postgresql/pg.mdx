# Alauda 对 PostgreSQL 的支持

Alauda 对 PostgreSQL 的支持是一个基于 Zalando Postgres Operator 的云原生 PostgreSQL 管理解决方案。它通过自定义资源定义（CRD）实现 PostgreSQL 集群的全生命周期管理，支持自动故障转移、滚动升级和备份恢复等核心操作能力，使企业能够轻松在 Kubernetes 上运行生产级 PostgreSQL 数据库。
