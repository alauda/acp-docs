# Alauda Streaming Service for RabbitMQ

Alauda Streaming Service for RabbitMQ 是一个开源消息代理，实施先进的消息队列协议（AMQP），并在 Kubernetes 上运行。它充当应用程序之间的通信桥梁，允许不同系统异步地发送和接收消息。它为构建解耦、可扩展和弹性的分布式系统提供了坚实的基础。
