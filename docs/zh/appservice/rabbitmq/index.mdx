# Alauda Streaming Service for RabbitMQ

Alauda Streaming Service for RabbitMQ 是一款开源的消息代理，采用了高级消息队列协议 (AMQP)，运行于 Kubernetes 上。它充当应用程序之间的通信桥梁，允许不同的系统异步发送和接收消息。它为构建解耦、可扩展和弹性化的分布式系统提供了坚实的基础。
