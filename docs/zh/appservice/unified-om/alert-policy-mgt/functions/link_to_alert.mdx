# 与平台能力的关系

告警是平台的核心能力之一，提供了包括应用服务在内的六个资源类别的广泛告警功能。应用服务的告警中心是平台告警能力的一部分。有关创建告警策略和设置静默告警的详细信息，请参阅 [告警管理](/observability/monitor/functions/manage_alert.mdx)。

:::info

由于权限考虑，应用服务视图下的告警管理与平台管理视图下的告警管理在数据层面上是相互隔离的。这意味着，如果用户在应用服务视图中创建了告警策略，则该策略的管理（如更新、查看和删除）仍需在应用服务视图中进行，反之亦然。

:::
