# 与平台能力的关系

警报是平台的核心能力之一，为六个资源类别提供广泛的警报功能，包括应用服务。应用服务的警报中心是平台警报能力的一个子集。有关创建警报策略和设置静默警报的详细信息，请参阅 [警报管理](/observability/monitor/functions/manage_alert.mdx)。

::: info

由于权限考虑，应用服务视图下的警报管理与平台管理视图下的警报管理在数据层面上是相互隔离的。这意味着用户在应用服务视图中创建的警报策略，其管理（如更新、查看和删除）仍必须在应用服务视图内进行，反之亦然。

:::
