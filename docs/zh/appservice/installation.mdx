## 安装 Alauda 应用服务 Essentials 插件

::: tip 可选安装
该插件是 Alauda 应用服务网页控制台的基础组件。如果不需要网页控制台，则无需安装它。
:::

### 前提条件

1. 下载与平台相匹配的 Alauda 应用服务 Essentials 安装包。
2. 使用平台的应用发布能力将 Alauda 应用服务 Essentials 发布到 `global` 集群。

### 步骤

<Tabs>
  <Tab label="网页控制台">
    1. 在左侧导航栏中，转到 **应用商店** > **集群插件**。
    2. 选择 `global` 集群。
    3. 点击 **Alauda 应用服务 Essentials** 插件右侧的操作按钮 > **安装**。
  </Tab>
</Tabs>

## 安装 Alauda Container Platform Data Services RDS Framework插件

::: tip 可选安装
该插件是 Alauda 应用服务网页控制台的基础组件。如果不需要网页控制台，则无需安装它。
:::

### 前提条件

1. 下载与平台相匹配的 Alauda Container Platform Data Services RDS Framework安装包。
2. 使用平台的应用发布能力将 Alauda Container Platform Data Services RDS Framework发布到任何希望利用数据服务视图功能的集群。

### 步骤

<Tabs>
  <Tab label="网页控制台">
    1. 登录平台并转到 **Administrator** 页面。
    2. 在左侧导航栏中，选择 **应用商店管理** -> **OperatorHub**，进入 **OperatorHub** 页面。
    3. 查找 **Alauda Container Platform Data Services RDS Framework**，点击 **安装** 并进入安装页面。

    配置参数：

    | **参数**            | **推荐配置**                                                                                                                       |
    | :------------------- | :------------------------------------------------------------------------------------------------------------------------------- |
    | **渠道**            | 默认渠道为 **稳定**。                                                                                                              |
    | **版本**            | 选择所需的版本进行安装。                                                                                                          |
    | **安装模式**        | **集群**：集群下所有命名空间共享一个Operator来创建和管理实例，减少资源消耗。                                                            |
    | **安装位置**        | 选择 **推荐**：如果不存在，则将自动创建。                                                                                           |
    | **升级策略**        | **手动**：当 OperatorHub 有新版本时，需要手动确认以将Operator升级到最新版本。                                                             |

    4. 在 **安装Operator** 页面，选择 **默认配置**，点击 **安装**，完成 **Alauda Container Platform Data Services RDS Framework** 的安装。
  </Tab>
</Tabs>
