# 安装

## 安装 Alauda 应用服务 Essentials 插件

::: tip 可选安装
该插件是 Alauda 应用服务网页控制台的基础组件。如果不需要网页控制台，则无需安装它。
:::

### 前提条件

1. 下载与平台兼容的 Alauda 应用服务 Essentials 安装包。
2. 使用平台的应用发布能力将 Alauda 应用服务 Essentials 发布到 `global` 集群。

### 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 在左侧导航栏中，转到 **应用商店管理** > **集群插件**。
    2. 选择 `global` 集群。
    3. 点击 **Alauda 应用服务 Essentials** 插件右侧的操作按钮 > **安装**。
  </Tab>
</Tabs>

## 安装 Alauda 应用服务 RDS 框架插件

::: tip 可选安装
该插件是 Alauda 应用服务网页控制台的基础组件。如果不需要网页控制台，则无需安装它。
:::

### 前提条件

1. 下载与平台兼容的 Alauda 应用服务 RDS 框架安装包。
2. 使用平台的应用发布能力将 Alauda 应用服务 RDS 框架发布到任何希望利用应用服务图形功能的集群。

### 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 登录平台并转到 **平台管理** 页面。
    2. 在左侧导航栏中，选择 **应用商店管理** -> **操作员中心**，进入 **操作员中心** 页面。
    3. 查找 **Alauda 应用服务 RDS 框架**，点击 **安装** 并进入安装页面。

    配置参数：

    | **参数**            | **推荐配置**                                                                                                                       |
    | :------------------- | :------------------------------------------------------------------------------------------------------------------------------- |
    | **渠道**            | 默认渠道为 **稳定**。                                                                                                              |
    | **版本**            | 选择所需的版本进行安装。                                                                                                          |
    | **安装模式**        | **集群**：集群下所有命名空间共享一个操作员来创建和管理实例，减少资源消耗。                                                            |
    | **安装位置**        | 选择 **推荐**：如果不存在，则将自动创建。                                                                                           |
    | **升级策略**        | **手动**：当操作员中心有新版本时，需要手动确认以将操作员升级到最新版本。                                                             |

    4. 在 **安装操作员** 页面，选择 **默认配置**，点击 **安装**，完成 **Alauda 应用服务 RDS 框架** 的安装。
  </Tab>
</Tabs>
