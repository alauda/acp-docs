# 通过代码创建应用

> 使用强大的 Alauda 容器平台构建功能，实现从 Java 源代码到创建应用的整个过程，最终使应用能够在 Kubernetes 上以容器化的方式高效运行。

## 前置条件

使用此功能前，请确保：

- [安装 Alauda Container Platform Builds](../install_builds/install_builds_operator.mdx)
- 平台上有可访问的镜像仓库。如没有，请联系管理员以 [安装 Alauda Container Platform Registry](/developer/registry/install_registry.mdx)

## 操作步骤

1. 在 **容器平台** 中，点击 **应用** > **应用**。

2. 点击 **创建**。

3. 选择 **通过代码创建**。

4. 根据以下参数完成配置：
   | **参数**        | **推荐的配置**                                                                                                          |
   | :--------------- | :--------------------------------------------------------------------------------------------------------------------- |
   | **代码仓库**    | **类型**: `输入`<br /> **仓库地址**: `https://github.com/alauda/spring-boot-hello-world`                                |
   | **构建方式**     | `构建`                                                                                                                  |
   | **镜像仓库**     | 请联系管理员。                                                                                                           |
   | **应用**        | **应用**: `spring-boot-hello-world`<br /> **名称**: `spring-boot-hello-world` <br />**资源限制**: 使用默认值。 |
   | **网络**        | **目标端口**: `8080`                                                                                                  |

5. 填写参数后，点击 **创建**。

6. 可在 **详情** 页面查看相应的应用状态。
