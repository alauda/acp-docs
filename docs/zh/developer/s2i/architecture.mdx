# 架构

![源到镜像架构](../../../en/developer/s2i/assets/s2i-archtecture.png "源到镜像架构")

源到镜像 (S2I) 能力通过 **ACP Builds** 操作符实现，允许通过 Git 仓库源代码自动生成容器镜像并随后推送到指定的镜像注册表。核心组件包括：

- **ACP Builds** 操作符：管理端到端构建生命周期并协调 Tekton 管道。
- **Tekton** 管道：通过 Kubernetes 原生的 `TaskRun` 资源执行 S2I 工作流程。
