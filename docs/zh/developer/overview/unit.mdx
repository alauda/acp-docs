# 资源单位说明

* CPU：可选单位为：核（core）、m（millicore）。其中，1 核 = 1000 m。

* 内存：可选单位为：Mi（1 MiB = 2^20 bytes）、Gi（1 GiB = 2^30 bytes）。其中，1 Gi = 1024 Mi。

* 虚拟 GPU（非必填）：仅当集群下存在 GPU 资源时，该参数有效。虚拟 GPU 的核心数，100 个虚拟核心等于 1 个 GPU 物理核心。支持输入正整数。

* 显存（非必填）：仅当集群下存在 GPU 资源时，该参数有效。虚拟 GPU 显存，1 个单位的显存为 256 Mi。支持输入正整数。
