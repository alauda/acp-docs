# 架构

请增加一张或多张关于该功能的技术架构图，图中可以包含以下内容：

- 核心组件：例如 Operator、任务、中间件等，并在图中描绘出各个组件之间的关系及其功能。
- 数据流程：例如引入的资源、资源之间的依赖关系，以及资源消耗的过程等。
- 部署架构：例如多集群部署、高可用性设计等。

在架构图下方增加一个描述，说明图像的意义及各部分的作用。

对架构图中引入的概念进行简要介绍。如果某个概念需要更详细的说明，请链接到「核心概念」对应的章节。
