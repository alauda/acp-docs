# 监控面板

- 支持查看平台上计算组件过去7天的资源监控数据（可配置监控数据保留周期）。包括应用、工作负载、容器组的统计数据，以及CPU/内存使用的趋势和排名。
- 支持命名空间级别的监控。
- 支持的工作负载级别监控：**应用**、**部署**、**守护进程集**、**有状态副本集**和**容器组**。

## 命名空间级别监控面板

### 步骤

1. 在**容器平台**中，点击**可观测** > **监控面板**。

2. 查看命名空间下的监控数据。提供三个面板：**应用概览**、**工作负载概览**和**容器组概览**。

3. 在不同面板之间切换，以监控目标的**概览**。

### 创建命名空间级别监控面板

1. 在**平台管理**中，通过参考[创建监控面板](../../observability/monitor/functions/manage_dashboard.mdx#｛#create_dashboard｝)创建专属监控面板。
2. 配置如下标签，以在**容器平台**上显示命名空间级别监控面板：
   - `cpaas.io/dashboard.folder: container-platform`
   - `cpaas.io/dashboard.tag.overview: "true"`

## 工作负载级别监控

> 本步骤演示如何通过部署界面查看容器组的监控数据。

### 默认监控面板

#### 步骤

1. 在**容器平台**中，点击**工作负载** > **部署**。

2. 从列表中选择一个部署名称。

3. 导航至**监控**标签以查看默认监控指标。

#### 指标说明

| **监控资源** | **指标粒度**           | **技术定义**                                                                                                                                                                   |
| :----------- | :--------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **CPU**      | **使用率**/**使用量**   | **使用率** = 使用量/限制（limits）<br />评估容器的限制配置。高使用率表明限制不足。<br />**使用量**代表实际资源消耗。                                                |
| **内存**     | **使用率**/**使用量**   | **使用率** = 使用量/限制（limits）<br />评估方法与CPU相同。高使用率可能导致组件不稳定。                                                                                        |
| **网络流量** | **流入速率**/**流出速率** | 进出容器组的网络流量（字节/秒）。                                                                                                                                               |
| **网络报文** | **接收速率**/**发送速率** | 容器组接收/发送的网络报文（计数/秒）。                                                                                                                                           |
| **磁盘速率** | **读取**/**写入**       | 挂载卷每个工作负载的读写吞吐量（字节/秒）。                                                                                                                                     |
| **磁盘 IOPS** | **读取**/**写入**       | 挂载卷每个工作负载的每秒输入/输出操作数（IOPS）。                                                                                                                               |

### 自定义监控面板

4. 点击**切换图标**以切换到自定义面板。参考[在自定义面板中添加面板](../../observability/monitor/functions/manage_dashboard.mdx)创建专属的**工作负载级别**监控面板。

:::info
将光标悬停在图表曲线上以查看特定时间戳下每个容器组的指标和PromQL表达式。如果容器组超过15个，则仅显示按降序排列的前15个条目。
:::
