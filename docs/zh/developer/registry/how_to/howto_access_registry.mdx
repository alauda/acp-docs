# 获取 Docker 镜像仓库访问地址

> 如果启用了暴露服务开关，通过获取访问地址和管理员账户，您可以从集群外部管理此镜像仓库。

1. 登录，进入 **平台管理** 页面。
2. 点击 **市场** > **集群插件** 以进入 **集群插件** 列表页面。

## 暴露服务禁用

> 当暴露服务被禁用，但需要使用客户端管理镜像时，您需要手动配置 Docker Registry 服务的 ClusterIP 地址与当前集群节点的 /etc/hosts 文件中的镜像仓库 URL 之间的映射。要获取 ClusterIP 地址，请执行以下命令：`kubectl get service -n cpaas-system internal-docker-registry -o jsonpath='{.spec.clusterIP}'`

1. 找到 **ACP Registry** 集群插件并点击其名称，进入 **ACP Registry** 详情页面。
2. 点击 **注册地址** 的图标，复制 docker registry 的访问 URL。
3. 您可以从当前集群内的任何 Pod 访问 Docker Registry。

## 暴露服务启用

1. 找到 **ACP Registry** 集群插件并点击 **更新** 以编辑该插件。
2. 将 **暴露服务** 切换为 **启用**。
3. 然后点击 **ACP Registry** 名称，导航至 **ACP Registry** 详情页面。
4. 点击 **注册地址** 的图标，复制 docker registry 的访问 URL。
5. 您可以在当前集群外访问 Docker Registry。
