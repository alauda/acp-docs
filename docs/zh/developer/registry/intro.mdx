# 介绍

## ACP 注册表简介

即插即用的 Docker 注册表是云原生环境中不可或缺的组成部分，简化了容器镜像管理，提高了开发效率，并改善了系统稳定性。Docker 注册表提供了一种安全、可靠且高性能的私有镜像注册表解决方案，支持 Docker 镜像的集中管理和分发。该解决方案特别适合规模较小的镜像注册表部署场景，以及对基础设施资源控制要求更严格的环境。

## ACP 注册表优势

**ACP 注册表的核心优势如下：**

- **即用即部署：** 快速设置私有镜像库，无需复杂的配置。
- **灵活部署：** 支持集群内部和外部访问，以满足各种部署需求。
- **安全可靠：** 提供授权和身份验证，确保镜像安全性。
- **高可用性：** 确保持续服务。
- **生产就绪：** 经过严格测试和验证，以满足生产环境的需求。

## ACP 注册表应用场景

- 在低流量业务场景中，实施轻量级的即插即用注册表，有助于快速部署和简化业务应用的分发。
- 在边缘计算场景中，每个边缘集群可以利用其专用的镜像注册表，实现对操作集群的完全自主管理。
- 当客户的基础设施未得到充分利用时，该平台与源码到镜像（S2I）技术的集成，可以提供即插即用的演示场景，实现对业务工作流的全面端到端演示。
