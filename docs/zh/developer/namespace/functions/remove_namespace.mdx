# 删除/移除命名空间 

**命名空间操作**：

您可以永久删除一个命名空间，或者将其从当前项目中移除。

**删除命名空间**：永久删除命名空间及其所有关联资源（例如，工作负载、服务），释放已分配的配额。此操作不可逆转。

**移除命名空间**：从项目中移除命名空间，但不影响其集群级别的资源，包括应用程序、持久卷、配置等。

被移除的命名空间可以通过 **命名空间导入** 功能导入到其他与其目标集群相关联的项目中。具体操作请参考 [导入命名空间](./import_namespace.mdx)。
