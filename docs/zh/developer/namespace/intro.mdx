# 介绍

参考 Kubernetes 官方网站文档：[命名空间](https://kubernetes.io/zh-cn/docs/concepts/overview/working-with-objects/namespaces/)

> 在 Kubernetes 中，命名空间提供了一种在单一集群内对资源进行分组隔离的机制。资源的名称在命名空间内需要唯一，但在不同的命名空间之间不需要唯一。基于命名空间的范围仅适用于命名空间对象（例如：Deployments、Services 等），而不适用于全集群对象（例如：StorageClass、Nodes、PersistentVolumes 等）。
