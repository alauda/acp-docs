# 介绍

在 Alauda 容器平台上创建和管理应用。使用 <Term name="product" textCase="lower" />，您可以通过 Web 控制台或命令行界面 (CLI) 创建、编辑、删除和管理应用。

## 优势

### 创建应用

在 <Term name="product" textCase="lower" /> 中，应用可以通过多种方式创建。以下是一些常见方法：

- 从镜像创建：如果您已经拥有一个容器镜像，可以直接从中创建应用。
- 从目录创建：<Term name="product" textCase="lower" /> 提供应用目录，允许用户选择预定义的应用模板（Helm Charts 或 Operator-Backed）进行创建。
- 从 YAML 文件创建：通过导入 YAML 文件，一步创建包含所有资源的自定义应用。
- 从代码创建：通过选择或输入代码仓库地址导入代码，自动构建镜像并创建应用。

### 应用管理

需要持续的操作管理，包括日志、事件、监控等。

- 查看 Pod 日志。
- 查看命名空间中的事件。
- 通过容器平台查看监控仪表板（命名空间级监控仪表板）。

### 应用的全生命周期管理

<Term name="product" textCase="lower" /> 提供应用生命周期管理支持，从应用创建、运行时、更新到最终退役。可以使用应用及其他 Kubernetes 工作负载等资源执行滚动更新和回滚等操作。
