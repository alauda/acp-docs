# 通过 EXEC 进入容器

## 通过原生应用进入容器

通过 docker exec 命令方式可进入容器实例内部，以便在 Web 控制台窗口执行命令行操作。同时，您也可以使用文件传输功能方便地上传下载容器内文件。

### 前提条件

* 容器已正常运行中。

* 使用文件传输功能时，容器中必须有 tar 工具，且容器操作系统不能为 Windows。


### 操作步骤

1. 进入 **Container Platform**。

1. 在左侧导航栏中，单击 **应用管理** > **原生应用**。

2. 单击 ***应用的名称***。

3. 找到计算组件，单击 **EXEC** > ***容器组名称***。

4. 输入要执行的命令。

6. 单击 **确定**，进入 Web 控制台窗口执行命令行操作。

7. 单击 **文件传输**。输入 **上传路径**，向容器中上传文件进行测试；或输入 **下载路径**，下载容器内日志等文件至本地进行分析。
