# 创建 Job

定时任务每执行一次，会创建一个任务（Job），您可查看任务的相关信息，掌握任务的执行情况。

## 操作步骤

1. 进入 **Container Platform**。

1. 在左导航栏中，单击 **计算组件** > **任务**。

2. 找到任务，单击 ⋮ > **查看**。
    
    
2. 在 **容器组** 页签中，单击 ***容器组名称***，查看执行任务的容器组详情。
    
    您也可执行容器相关操作。        
    
    * EXEC 进入容器：单击 ⋮ > **EXEC** > ***容器名称***。
    
    * 删除容器组：单击 ⋮ > **删除**，并确定。

1. 在 **配置** 页签中，查看容器实例运行时使用的所有环境变量键值对。

    * **环境变量**：环境变量的键值对，或引用的配置字典或保密字典中的值的信息。
    
    * **配置引用**：引用的完整的配置文件（配置字典或保密字典）信息。

2.  在 **实时日志** 页签中，查看应用容器实例运行时输入的容器日志，可用于分析任务执行失败的原因。


    **说明**：如果容器组记录被删除，相关的日志也会被删除。 

9. 在 **事件** 页签中，可查看指定时间范围内，当前计算组件下的所有 Kubernetes 资源的事件信息。
