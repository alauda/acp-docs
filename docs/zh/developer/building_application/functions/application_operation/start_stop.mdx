# 原生应用的启动与停止

## 启动应用

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **应用管理** > **原生应用**。

3. 单击应用的名称。

4. 单击 **启动**。

## 停止应用

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **应用管理** > **原生应用**。

3. 单击应用的名称。

4. 单击 **停止**。

5. 阅读提示信息，确认无误后，单击 **停止**。
