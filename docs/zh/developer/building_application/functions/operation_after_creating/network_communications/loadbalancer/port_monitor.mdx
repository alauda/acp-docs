# 监控端口

当监听端口的协议为 HTTPS 或 HTTP 时，您可以查看流入/流出流量、请求和响应的监控数据。

**注意**：不支持监控 gRPC 协议的数据。

## 操作步骤

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **网络** > **负载均衡**。

3. 单击负载均衡器的名称。

4. 单击监听端口的名称。

5. 在 **监控** 中，查看该端口的监控数据。

   数据的计算公式为 `统计数据值 = 统计时间粒度内数据总量 / 统计时间粒度`。流量值将根据设置的单位转换为准确值；请求与响应的值若有小数，则会被四舍五入，忽略小数部分，单位为 `个/秒`。

   例如：如果在某个统计时间节点的统计时间粒度为 4 小时，并且在这 4 小时内，5xx 状态码的请求和响应总数为 10003，则该时间节点的 5xx 状态码统计值 = 10003 / 4*60*60，四舍五入后的统计值为 7 个/秒。

   | 时间范围          | 时间轴间隔      | 统计时间粒度            |
   | ------------------- | ------------------ | -------------------------- |
   | **过去 15 分钟**   | 2 分钟              | 30 秒                     |
   | **过去 1 小时**     | 5 分钟              | 1 分钟                     |
   | **过去 1 天**       | 3 小时              | 30 分钟                   |
   | **过去 7 天**       | 1 天                | 4 小时                     |
   | **过去 30 天**      | 2 天                | 12 小时                   |
