# 监控端口

当监听端口的协议为 HTTPS 或 HTTP 时，支持查看端口流入/流出流量、请求与响应的监控数据。

**提示**：不支持查看 gRPC 协议的监控数据。

## 操作步骤

1. 进入 **Container Platform**。

1. 在左侧导航栏中，单击 **网络** > **负载均衡**。
      
3. 单击负载均衡器的名称。
4. 单击监听端口的名称。


3. 在 **监控** 中，查看端口的监控数据。

    数据的计算公式为 `统计数据值 = 统计时间粒度内数据总量/统计时间粒度`。其中，流量的值会根据实际设定的单位换算为精确值；请求与响应的值若带有小数位时，会四舍五入忽略小数值，单位为 `个/秒`。
    
    例如：某个统计时间节点的统计时间粒度为 4 小时，4 小时内 5xx 状态码请求与响应的总量为 10003 个，该时间节点的 5xx 状态码的统计值 = 10003 / 4*60\*60，四舍五入后的统计值为 7 个/秒。
    
    |时间范围| 时间轴间隔    |   统计时间粒度    |
    | ------  |  ------  |  ------  |
    |  **过去 15 分钟**  |   2 分钟 |    30 秒  |
    |  **过去 1 小时**  |   5 分钟 |    1 分钟  |
    |  **过去 1 天**  |   3 小时 |    30 分钟  |
    |  **过去 7 天**  |   1 天 |    4 小时  |
    |  **过去 30 天**  |   2 天 |    12 小时  |
