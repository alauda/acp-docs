# 更新默认证书

通常，管理员会在证书有效期限结束前续订或替换证书。如果管理员替换了用于 HTTPS 或 gRPC 监听端口的证书，您必须更新监听端口以使用新证书作为默认证书。

## 操作步骤

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **网络** > **负载均衡**。

3. 单击负载均衡器的名称。

4. 单击监听端口的名称。

5. 单击 **操作** > **更新默认证书**。

6. 选择新证书，然后单击 **更新**。
