# 更新 Gateway

**注意**：更新入站网关会导致服务中断 3-5 分钟，请选择合适时间进行此操作。

**步骤**

1. 访问 **Container Platform**。

2. 在左侧导航栏中，点击 **网络** > **入站网关**。

3. 点击 ⋮ > **更新**。

4. 根据需要更新入站网关配置。

   **说明**：请根据业务要求合理设置规格。

5. 点击 **更新**。
