# 创建 Ingress

创建 Ingress 以管理对服务的外部 HTTP/HTTPS 访问。

:::warning
在同一命名空间下创建多个 Ingress 时，不同的 Ingress **必须** 不具有相同的 **域名**、**协议**和 **路径**（即不允许重复的访问点）。
:::

## 前提条件

- 当前命名空间中必须有一个可用的 **服务**。

- 请与管理员确认已为与当前命名空间关联的项目分配了可用的域名。

- 要通过 HTTPS 访问该域名，您需要首先将 HTTPS 证书保存为 TLS 密钥。

## 操作步骤

1. 访问 **Container Platform**。

2. 在左侧导航栏中，单击 **网络** > **Ingress**。

3. 单击 **创建 Ingress**。

4. 根据以下说明配置某些参数。

   | 参数                       | 说明                                                                                                                                                                                                                                                                                                 |
   | ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Ingress 类**            | Ingress 可以由不同控制器以不同的 `IngressClass` 名称实现。如果平台上可用多个 Ingress 控制器，用户可以通过此选项选择要使用的控制器。                                                                                                                                                                             |
   | **域名**                   | 主机可以是精确匹配（例如 `foo.bar.com`）或通配符（例如 `*.foo.com`）。可用的域名由平台管理员分配。                                                                                                                                                                                                      |
   | **证书**                   | 由平台管理员分配的 TLS 密钥或证书。                                                                                                                                                                                                                                                                         |
   | **匹配方式** 和 **路径**   | <ul><li>**前缀**：匹配路径前缀，例如 `/abcd` 可以匹配 `/abcd/efg` 或 `/abcde`。</li><li>**精确**：匹配精确路径，例如 `/abcd`。</li><li>**由实现决定**：如果您使用自定义 Ingress 控制器来管理 Ingress 规则，您可以选择由控制器来决定。</li></ul>                                        |
   | **服务**                   | 外部流量将被转发到此服务。                                                                                                                                                                                                                                                                                     |
   | **服务端口**               | 指定流量将转发到的服务端口。                                                                                                                                                                                                                                                                                   |

5. 单击 **创建**。

:::note
平台管理员可以为项目分配 ALB 实例。
如果 Ingress 没有 Ingress 类，则为该项目分配的所有 ALB 实例将处理此 Ingress。
:::
