# ALB 规则由 Ingress 同步

- 当 Ingress 规则未指定 IngressClass，或指定的 IngressClass 是由负载均衡器生成时。

  - 使用 **端口** 资源分配方法创建的负载均衡器，仅在命名空间具有 80 和 443 端口权限，并且创建了 HTTP 80 和 HTTPS 443 端口时，才会自动解析 Ingress 规则。

  - 使用 **实例** 资源分配方法创建的负载均衡器，在命名空间具有负载均衡器权限时，将自动解析该命名空间内的 Ingress 规则。

  **注意**：为确保命名空间内的 Ingress 规则正常工作，当仅仅使用平台负载均衡器时，请确保命名空间具有负载均衡器权限以解析 Ingress 规则。

- 当 Ingress 规则指定非负载均衡器生成的 IngressClass 时。

  平台负载均衡器将不再解析 Ingress 规则，您必须确保指定的 IngressClass 有可用的 Controller 来解析 Ingress 规则。
