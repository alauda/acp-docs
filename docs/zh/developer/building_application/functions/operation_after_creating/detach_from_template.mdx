# 通过 Chart 部署原生应用而不使用模板

为了将当前应用开发为可在未来分发到其他环境的形式，可以将应用从模板中脱离：导航至 **Container Platform** > **应用管理** > **原生应用**，进入应用详情页面，然后点击 **操作** > **脱离模板**。

在脱离模板之前，对底层资源（例如工作负载、内部路由等）进行的直接操作将仅是临时生效，并不会记录到模板中。一旦更新应用，模板中的配置将覆盖临时配置。脱离模板的应用与通过平台 [UI 表单](../create_applications/image_app.mdx) 创建的应用 indistinguishable，且不受上述限制。
