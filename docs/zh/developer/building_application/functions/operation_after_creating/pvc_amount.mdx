# 管理存储用量

查看已挂载的持久卷存储用量数据，供计算组件或应用时参考。如果可用容量不足，可相应扩展持久卷声明的容量。

## 查看持久卷存储用量

### 注意事项

* 存储用量数据仅供参考，仅支持采集已挂载的持久卷存储用量数据。

* HostPath、CephRBD、TopoLVM 等类型的 PVC 总容量和使用量数据信息可能与实际值存在一定差异。

### 操作步骤

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **存储 > 持久卷声明 (PVC)**。

3. 单击持久卷声明的名称。

4. 在 **详细信息** 页签中，滑动到 **存储用量** 区域。

5. 查看当前挂载的持久卷已用容量及其变化趋势。

## 扩增持久卷声明存储容量

1. 在左侧导航栏中，单击 **存储 > 持久卷声明 (PVC)**。

2. 找到持久卷声明，单击 ⋮ > **扩容**。

3. 填写新容量。

4. 单击 **扩容**。扩容过程可能需要一些时间，请耐心等待。
