# 管理存储用量

查看已挂载的持久卷存储用量数据，供计算组件或应用使用时参考。如果可用容量不足，可相应扩充持久卷声明容量。

## 查看持久卷存储用量

### 注意事项

* 存储用量数据仅供参考，仅支持采集已挂载的持久卷存储用量数据。

* Hostpath、CephRBD、Topolvm 等类型的 PVC 总容量和使用量数据信息可能与实际值有一定出入。

### 操作步骤

1. 进入 **Container Platform**。

1. 在左侧导航栏中，单击 **存储 > 持久卷声明(PVC)**。

2. 单击持久卷声明的名称。

3. 在 **详细信息** 页签中，滑动到 **存储用量** 区域。

4. 查看当前挂载的持久卷已用容量，及其变化曲线。

## 扩增持久卷声明存储容量

1. 在左侧导航栏中，单击 **存储 > 持久卷声明(PVC)**。

2. 找到持久卷声明，单击 ⋮ > **扩容**。

4. 填写新容量。
5. 单击 **扩容**。扩容过程需要一定耗时，请耐心等待。
