# 通过代码创建应用

通过代码创建应用是利用源代码到镜像(Source to Image, S2I)技术实现的。S2I是一个自动化框架，可以直接从源代码构建容器镜像。这种方法标准化并自动化了应用程序构建过程，使开发者可以专注于源代码开发，而无需担心容器化的细节。

## 先决条件

- 完成 [Alauda Container Platform Builds](../../../s2i/install_builds/install_builds_operator.mdx) 的安装

## 操作步骤

1. 进入 **容器平台**，然后导航到 **应用** > **应用**。

2. 点击 **创建**。

3. 选择 **从代码创建**。

4. 有关详细的参数描述，请参考 [管理从代码创建的应用](/developer/s2i/functions/s2i_application_management.mdx#s2iparameter)

5. 完成参数输入后，点击 **创建**。

6. 可在 **详细信息** 页面查看相应的部署。
