# 通过 Operator 创建组件应用


应用组件（Operator backed）为 Operator 提供的资源集合，基于应用组件（Operator backed）您可以快速部署一个组件应用，并借助 Operator 的能力自动化管理应用的整个生命周期。





## 操作步骤

1. 进入 **Container Platform**。

1. 在左侧导航栏中，单击 **Operator backed**。

2. 单击 Operator 卡片上的 **部署**。

4. 填写相关参数，例如 `资源配额`。


6. 单击 **部署**。平台将跳转至 **应用管理** > **组件应用** 页面。

    部署需要消耗一定的时间，请耐心等待。若操作失败，请根据界面提示信息完成操作。
