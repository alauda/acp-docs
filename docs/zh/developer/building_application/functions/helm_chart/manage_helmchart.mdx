# 管理应用模板

模板应用的更新依赖于应用模板（Helm Chart）的更新，若某些模板无法通过平台升级、公共仓库或自建私有仓库等方式自动更新，您可将其手动添加至具有 **管理** 权限的 Chart 或 OCI Chart 类型模板仓库中，以供部署该版本的应用。

## 前提条件

模板仓库由平台管理员添加，请联系平台管理员获取可用的具有 **管理** 权限的 Chart 或 OCI Chart类型模板仓库名称。


## 添加模板至可管理仓库

1. 进入 **Catalog**。

1. 在左侧导航栏中，单击 **Helm Charts**。

2. 单击页面右上角 **添加模板**，参考如下参数选择模板仓库。

    |参数|说明|
    |---|---|
    |**模板仓库**| 将模板直接同步到具有 **管理** 权限的 Chart 或 OCI Chart 类型模板仓库中。被分配了该 **模板仓库** 的项目所有者可直接使用该模板。|
    |**模板目录**|当选择的模板仓库类型为 OCI Chart 时，需选择或手动输入存放 Helm Chart 的模板目录。<br/>**注意**：手动输入新的模板目录时，平台会在模板仓库中创建此目录，但是会存在创建失败的风险。|


4. 单击 **上传模板**，并将本地模板上传至仓库。

 
5. 单击 **确定**。模板上传过程可能需要几分钟，请耐心等待。

    **说明**：待模板状态由 `上传中` 变为 `上传成功` 时，表示模板已上传成功。
7. 若上传失败，请根据下述提示排查。

    **说明**：文件格式不合法表示上传的压缩包中的文件存在问题，例如内容缺失，格式不正确等。


## 删除特定版本的模板

若某版本的模板已不再适用，可删除该模板。

### 操作步骤

1. 进入 **Catalog**。

1. 在左侧导航栏中，单击 **Helm Charts**。
    
1. 单击 Chart 卡片查看详情。

    
3. 单击 **管理版本**。

4. 找到不再适用的模板，单击 **删除**，并确认。

    删除版本后，对应的应用将无法更新。
