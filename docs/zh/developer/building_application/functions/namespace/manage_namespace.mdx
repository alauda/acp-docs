# 更新命名空间

## 更新配额

> [资源配额](./resource_quota.mdx)

### 使用 Web 控制台更新资源配额

1. 在 **项目管理** 中，导航到左侧边栏的 **命名空间** > **命名空间** 列表。

2. 点击目标 ***命名空间名称***。

3. 点击 **操作** > **更新配额**。

4. 调整资源配额（CPU、内存、Pods 等），然后点击 **更新**。

### 使用 CLI 更新资源配额

> [资源配额 YAML 文件示例](./create_namespace.mdx#namespace-yaml)

```bash
# 步骤 1：编辑命名空间配额  
kubectl edit resourcequota <quota-name> -n <namespace-name>  

# 步骤 2：验证更改  
kubectl get resourcequota <quota-name> -n <namespace-name> -o yaml  
```

## 更新容器 LimitRange

> [限制范围](./limit_range.mdx)

### 使用 Web 控制台更新 LimitRange

1. 在 **项目管理** 视图中，导航到左侧边栏的 **命名空间** > **命名空间** 列表。

2. 点击目标 ***命名空间名称***。

3. 点击 **操作** > **更新容器 LimitRange**。

4. 调整容器限制范围（`defaultRequest`、`default`、`max`），然后点击 **更新**。

### 使用 CLI 更新 LimitRange

> [限制范围 YAML 文件示例](./create_namespace.mdx#namespace-yaml)

```bash
# 步骤 1：编辑 LimitRange  
kubectl edit limitrange <limitrange-name> -n <namespace-name>  

# 步骤 2：验证更改  
kubectl get limitrange <limitrange-name> -n <namespace-name> -o yaml  
```
