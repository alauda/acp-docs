# 删除/移除命名空间

您可以永久删除命名空间或将其从当前项目中移除。

## 删除命名空间

**删除命名空间**：永久删除一个命名空间及其所有资源（例如，Pods、Services、ConfigMaps）。此操作无法撤销，并释放已分配的资源配额。

```bash
kubectl delete namespace <namespace-name>
```

## 移除命名空间

**从项目中移除命名空间**：从当前项目中移除一个命名空间，而不删除其资源。该命名空间仍然保留在集群中，可以通过 [导入命名空间](./import_namespace.mdx) 导入到其他项目中。

**注意**

- 此功能仅适用于 <Term name="product" />。
- Kubernetes 本身不支持从项目中“移除”命名空间。

```bash
kubectl label namespace <namespace-name> cpaas.io/project- --overwrite
```
