# 创建存储桶声明

创建存储桶声明，动态的使用对象存储类创建存储桶，并与之绑定。

## 前提条件

请联系管理员部署 **容器对象存储接口插件**。

## 操作步骤

1. 进入 **Container Platform**。

1. 在左侧导航栏中，单击 **存储 > 存储桶声明**。

2. 单击 **创建存储桶声明**。

2. 参考以下说明，配置相关参数。

    **说明**：下述内容以表单方式为例，您也可切换至 YAML 方式完成操作。

    |参数|说明                                                         |
    | -------- | ----------------  | 
    | **名称**   | 存储桶声明的名称。 |
    | **对象存储类**   |基于对象存储类动态生成存储桶并匹配绑定。|

5. 单击 **创建**。等待状态变为 `可用` 状态，则匹配成功。

## 相关操作

您可以在详情页面单击右上角的 **操作**，按需删除存储桶声明。删除存储桶声明后，存储桶中的数据将被清除，请在基于数据安全的前提下谨慎操作。
