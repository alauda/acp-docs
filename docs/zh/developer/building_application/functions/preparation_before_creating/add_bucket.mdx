# 创建存储桶声明

创建存储桶声明，动态生成存储桶并使用对象存储类与之关联。

## 前提条件

请联系管理员安装 **Object Storage Interface Plugin**。

## 操作步骤

1. **容器平台**。

2. 在左侧导航栏中，单击 **存储** > **存储桶声明**。

3. 单击 **创建存储桶声明**。

4. 参考以下说明配置相关参数。

   **说明**：下述内容以表单方式呈现，您也可以切换至 YAML 方式来完成操作。

   | 参数                     | 描述                                                                            |
   | ------------------------ | -------------------------------------------------------------------------------------- |
   | **名称**                 | 存储桶声明的名称。                                                    |
   | **对象存储类**           | 基于对象存储类动态生成存储桶并匹配绑定。 |

5. 单击 **创建**。等待状态变为 `可用`，这表示匹配成功。

## 相关操作

您可以在详情页面的右上角单击 **操作** 按钮，根据需要删除存储桶声明。删除存储桶声明后，存储桶中的数据将被清除， 请基于数据安全考虑谨慎操作。
