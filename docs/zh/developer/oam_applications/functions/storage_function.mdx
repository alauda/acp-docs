# 添加存储与配置

通常，在应用创建过程中，应用资源（存储与配置）的设置已经完成。本节将介绍如何添加新的存储和配置。

## 添加存储

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **应用管理** > **OAM 应用**。

3. 单击 ***应用名称***。

4. 在 **应用资源** 页签中，单击 **添加资源**。

5. 参考以下说明完成相关配置。

   | 参数             | 说明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
   | ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **存储类**       | <ul><li>存储类代表不同类型的存储特性，例如是否支持在线扩容、是否支持卷快照，以及是否提供高可用性。</li><li>每个存储类在底层存储级别支持数据的持久化存储，这意味着当应用停止或重启时，数据不会被删除。</li><li>存储类通常由管理员设置，您可以灵活选择符合预期的存储类。</li></ul>                                                                                                                                                                                                                                                      |
   | **容量**         | 这是指存储配额，配额大小取决于您的应用消耗。如果设置的值较大，请与管理员确认实际可用存储空间是否充足。                                                                                                                                                                                                                                                                                                                                                                                                                                |

6. 单击 **添加**。

7. 返回组件列表页，单击 ⋮ > **更新**，并将存储挂载为 [存储卷](../concepts/component.mdx)。

## 添加配置

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **应用管理** > **OAM 应用**。

3. 单击 ***应用名称***。

4. 在 **应用资源** 页签中，单击 **添加资源**。

5. 按需完成相关配置。

   - 对于一般配置，您可以选择 **配置字典** 或 **Opaque** 类型。

   - 对于敏感数据，务必根据数据性质选择类型，例如 **用户名和密码**、**SSH 密钥** 或 **镜像服务**。

6. 单击 **添加**。

7. 返回组件列表页，单击 ⋮ > **更新**，并将配置挂载为存储卷或引用为环境变量。
