# 监控与告警

## 监控

在组件的 **监控** 页签中，您可以查看近 7 天内应用的监控数据。

**说明**：仅当应用使用了 GPU 资源时，GPU 相关监控图表可见。

| 监控资源                     |数据统计粒度              | 指标说明                                                                                                                                                                                                                                                                                                                                                                         |
| ----------------------------- | ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **CPU**                       | 使用率/使用量            | **使用率** = 使用值 / 限制值（limits）<br />您可以通过使用率判断组件容器的限制值配置是否合理。当使用率过高时，表明容器的限制值设置过低，这可能会影响组件的正常运行。<br />**使用量** 是组件已使用的资源量。 |
| **内存**                     | 使用率/使用量            | **使用率** = 使用值 / 限制值（limits）<br />您可以通过使用率判断组件容器的限制值配置是否合理。当使用率过高时，表明容器的限制值设置过低，这可能会影响组件的正常运行。<br />**使用量** 是组件已使用的资源量。 |
| **网络流量**                 | 流入/流出                | 组件中所有容器组的网络流量速率总和。                                                                                                                                                                                                                                                                                                                   |
| **网络报文量**               | 接收/发送                | 组件中所有容器组的网络报文量速率总和。                                                                                                                                                                                                                                                                                                           |
| **磁盘速率**                 | 读/写                    | 组件中所有容器组挂载分区的读写速率总和。                                                                                                                                                                                                                                                                                              |
| **磁盘 IOPS**                 | 读/写                    | 组件中所有容器组挂载分区的 IOPS 总和。                                                                                                                                                                                                                                                                                                             |
| **虚拟/物理 GPU**            | 使用率/使用量            | **使用率** = 使用值 / 限制值（limits）<br />**使用量** 是组件已使用的资源量。                                                                                                                                                                                                                                                                |
| **虚拟/物理 GPU 显存**      | 使用率/使用量            | **使用率** = 使用值 / 限制值（limits）<br />**使用量** 是组件已使用的资源量。                                                                                                                                                                                                                                                                |

## 告警

1. **配置告警策略**：为启用告警，请首先参考组件的 **告警** 页签中的平台通用指导 [创建告警策略]()。告警策略描述了您希望监控的对象、在什么情况下希望收到告警，以及如何被通知相关告警。

2. **处理告警**：当界面提示有处于告警状态的策略时，请及时处理。

3. **绑定通知策略**：除了查看实时告警外，平台还支持通过邮件、短信等方式将告警信息发送给相关人员，以通知他们采取必要的措施来解决问题或避免故障的发生。通知策略需联系管理员设置后方可使用。
