# 版本对比与回滚

平台记录了通过界面和非界面操作（如 API 调用）所做的变更。每次变更都对应一个版本，这使得您可以轻松进行版本对比或回滚。

## 版本对比

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **应用管理** > **OAM 应用**。

3. 单击 ***应用名称***。

4. 单击 ***组件名称***。

5. 在 **变更日志** 页签中，单击 ***版本号***。

6. 单击 **与当前版本比较** 来查看变更。

## 版本回滚

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **应用管理** > **OAM 应用**。

3. 单击 ***应用名称***。

4. 单击 ***组件名称***。

5. 在 **变更日志** 页签中，单击 ***版本号***。

6. 单击 **回滚至该版本**，并确认。
