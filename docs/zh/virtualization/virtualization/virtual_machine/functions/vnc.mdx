# 使用 VNC 登录虚拟机

使用 Web 控制台（VNC）登录虚拟机，作为应急操作手段。

## 操作步骤

1. 进入 **Container Platform**。

2. 在左侧导航栏中，单击 **虚拟化** > **虚拟机**。

3. 单击 ⋮ > **VNC 登录**。

4. 控制台窗口将自动打开；您需要输入用户名和密码进行登录。

   <img src="./assets/vmi00.png" width="600" />

   **说明**：

   - 支持发送常用键盘命令。

   - 支持复制和粘贴命令和参数。
