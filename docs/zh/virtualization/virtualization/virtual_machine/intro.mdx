# 介绍

KubeVirt 提供了 CRD（自定义资源定义），如 **VirtualMachine** 和 **VirtualMachineInstance**，用于抽象虚拟机（VM）资源。基于这些 CRD，用户可以获得全面的虚拟机管理能力。在此基础上，**ACP Virtualization With KubeVirt** 进一步增强了可用性，提供了 **Web Console**，使用户能够更轻松地执行各种操作。

## 优势

- **全面的虚拟机管理**

  - 创建/删除虚拟机、重装操作系统、重置密码、更新 SSH 密钥、调整资源分配以及访问 VNC 控制台。
  - 导出/克隆虚拟机、执行在线迁移等。

- **用户友好的操作**

  - 大多数虚拟机操作可以通过直观的 Web UI 轻松完成。

- **可视化监控与告警**

  - 通过 Web UI 监控虚拟机关键指标（例如 CPU、内存、磁盘使用情况）。
  - 配置告警以主动检测潜在的运行时问题（例如资源耗尽、性能下降）。
