# 介绍

Kubevirt 提供了 **VirtualMachine**、**VirtualMachineInstance** 等 CRD 来抽象虚拟机资源，基于这些 CRD，用户可以获得较完整虚拟机管理能力，在此基础上 **ACP Virtualization With Kubevirt** 额外提供了 **Web Console** 让用户能更便捷的进行各种操作。

## 优势

- **完善的虚拟机管理能力**
  
  - 支持创建删除虚拟机、重装系统、重置密码、更新密钥、调整虚拟机资源、VNC 访问等基础能力。
  
  - 支持虚拟机导出、克隆、热迁移等高级功能。

- **便捷的操作**

  - 用户可以通过 Web UI 便捷的完成大部分虚拟机操作。

- **可视化监控与告警**

  用户可以通过 Web UI 来监控虚拟机的关键指标，也可以为虚拟机配置告警来及时发现虚拟机运行时存在潜在问题。
