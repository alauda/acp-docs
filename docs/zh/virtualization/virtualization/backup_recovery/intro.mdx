# 介绍

**ACP Virtualization With Kubevirt** 提供虚拟机快照能力，通过快照可以对虚拟机进行备份和恢复。

## 应用场景

- 灾难恢复与故障回退

  虚拟机因硬件故障、人为误操作（如误删文件）或恶意攻击（如勒索软件）导致数据丢失时，快照可作为最后一道防线恢复业务。

## 限制

- 创建快照需要先停止虚拟机。
- 虚拟机磁盘使用的 PVC 需要配置为多节点共享访问模式。
