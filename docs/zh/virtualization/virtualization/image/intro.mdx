# 介绍

Alauda Container Platform Virtualization with KubeVirt 利用 Kubernetes 扩展 API 能力将虚拟机镜像抽象为一个 CRD，并提供 UI 让用户可以便捷的将存放在远端的虚拟机镜像加入到 ACP 中使用。

## 优势

- **支持多种主流操作系统**

  支持各种常用的 Linux 发行版本以及 Windows 操作系统。

- **多架构支持**

  同时支持 X86_64 和 ARM64。

- **多来源支持**

  支持添加存放在镜像仓库、文件服务器以及 S3 对象存储中的虚拟机镜像。

- **多格式支持**
  
  支持 QCOW2 以及 RAW 格式的虚拟机镜像。
