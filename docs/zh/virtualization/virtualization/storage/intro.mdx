# 介绍

**ACP Virtualization With Kubevirt** 存储用于给虚拟机提供持久化能力，深度集成了 Kubernetes 原生的存储机制，通过 PersistentVolumeClaim（PVC） 来存储虚拟机磁盘数据，同时可利用 Container Storage Interface (CSI) 来集成各种存储系统来供虚拟机使用。使用 Container Data Importer （CDI）来为虚拟机磁盘初始化数据。在此基础上对虚拟机磁盘进行了进一步的功能扩展，可以更好的管理虚拟机磁盘的生命周期。

## 优势

- **操作便捷**

  可以通过 Web UI 便捷的完成大部分虚拟机磁盘操作。

- **虚拟机磁盘生命周期管理**

  可以设置虚拟机磁盘是否随虚拟机删除
