# 功能总览

## 虚拟机镜像

- 管理虚拟机镜像  
  
  平台支持创建、更新、删除虚拟机镜像。

- 管理镜像凭据

  支持创建更新删除镜像凭据。

## 虚拟机

- 创建虚拟机/虚拟机组
  
  支持快速创建单个虚拟机，或者创建多个配置相同的虚拟机（虚拟机组）。
  
- 管理虚拟机
  
  支持多种虚拟机的管理操作，包括重置密码、更新密钥、更新规格、更新标签及注解、重装系统等。

- 管理虚拟机磁盘

  支持虚拟机磁盘的管理操作，包括创建、删除、挂载、卸载、扩容等。

- 管理密钥对

  支持密钥对的创建、删除和更新。

- 虚拟机批量操作
  
  支持批量进行虚拟机的启动、停止、重启和删除操作。

- 使用 vnc 登录虚拟机

  支持 vnc 登录虚拟机。

- 快速定位虚拟机

  支持按集群展示虚拟机列表，方便平台管理员快速定位虚拟机所在命名空间。

- 监控与告警
  
  从 CPU、内存、存储及网络等方面对虚拟机进行监控和告警。

## 虚拟机网络

- 使用容器网络

  支持为虚拟机配置使用容器网络，支持 NAT、桥接等模式。

- 配置内部路由

  可以通过创建 Service 对象在集群内或集群外公开虚拟机。

- 使用SR-IOV

  将虚拟机接入 SR-IOV 网络。

## 备份与恢复

- 虚拟机快照
  
  支持为虚拟机添加快照，并基于快照恢复虚拟机。
