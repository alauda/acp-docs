# 虚拟机批量操作

批量进行虚拟机的启动、停止、重启和删除操作。

## 操作步骤

1. 进入 **Container Platform**。

1. 在左侧导航栏中，单击 **虚拟化** > **虚拟机**。

2. 找到目标虚拟机，单击 ⋮ 可对单个虚拟机进行操作，或参考下图，对虚拟机进行批量操作。

    **提示**：
    
    * 当虚拟机处于休眠或已停止状态时可执行 **启动/批量启动** 操作；当虚拟机处于准备中、启动中、运行中、休眠、未知、异常状态时可执行 **停止/批量停止** 操作；当虚拟机处于运行中状态时可执行 **重启/批量重启** 操作。

    * 对虚拟机进行强制 **重启/停止** 操作，相当于对虚拟机执行断电处理，可能会丢失未写入磁盘的数据。  

11. 根据界面提示完成操作。当虚拟机变为如下状态时，操作成功。
    
    |操作| 状态 |
    | --- | --- |
    |**启动虚拟机**| 运行中 |
    |**停止虚拟机**| 已停止 |
    |**重启虚拟机**| 运行中 |
