# Web 控制台概览

## 介绍

Alauda 容器平台的网页控制台提供了一个图形用户界面，用于可视化项目数据并执行管理、监控和故障排除任务。网页控制台旨在通过将平台功能组织成不同用户角色和工作流程所需的独特视角，来增强用户体验。

## 门户页面

登录网页控制台后，默认会进入门户页面。门户页面显示网页控制台支持的所有可用视角卡片。这些卡片作为平台不同功能区域的入口点。

## 视角切换

您可以使用网页控制台左上角的视角切换器在不同视角之间切换。这允许在平台内不同的功能上下文之间无缝导航。

## 主要视角

Alauda 容器平台的网页控制台提供了几个关键视角，每个视角都针对特定的用户角色和任务量身定制：

### 容器平台视角

容器平台视角专为开发者设计。在此视角中，开发者可以管理他们项目中的资源，例如：

- 应用工作负载
- 存储卷
- 配置
- 服务绑定
- 路由和网络资源

### 平台管理视角

平台管理视角面向管理员。此视角允许管理员：

- 创建或导入集群
- 安装和管理运营者
- 配置平台范围的设置
- 监控系统健康和指标
- 管理平台安全性

### 项目管理视角

项目管理视角侧重于项目层级的管理，使用户能够：

- 创建和配置项目
- 管理项目配额和资源限制
- 管理项目成员和角色绑定
- 配置项目级策略

## 概览页面

每个视角通常包括一个概览页面，提供相关信息的摘要。例如，平台管理概览页面显示平台范围的数据，包括：

- 项目总数
- 集群状态和健康
- 资源利用指标
- 最近事件和警报
- 平台健康指标

这些概览页面提供了对容器化基础设施状态和性能的一目了然的洞察，帮助用户快速评估系统健康并识别需要关注的领域。
