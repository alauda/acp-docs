# 自定义 Web 控制台

## 介绍

Alauda 容器平台的 Web 控制台可以自定义，以反映您组织的品牌和偏好。本文档描述了管理员可选的各种自定义选项。

## 用户偏好

### 主题和语言设置

用户可以通过位于控制台右上角的用户菜单自定义其 Web 控制台体验。

要更改主题和语言设置：

1. 点击右上角的用户名
2. 从下拉菜单中选择“用户偏好”
3. 导航到“主题”或“语言”部分
4. 选择您偏好的设置
5. 点击“保存”以应用更改

**注意：** 如果您首选的语言在语言选择器中不可用，您可能需要安装相应的语言插件。请联系您的平台管理员以获取安装其他语言插件的帮助。

## 管理员自定义

以下自定义需要管理员权限，可通过平台管理视角访问。

### 自定义登录页面

要自定义登录页面：

1. 导航到平台管理视角
2. 从导航菜单中选择“系统设置”
3. 点击“登录页面”
4. 自定义以下元素：
   - 更新 Logo：点击“更新 Logo”以上传自定义 logo 图像（推荐尺寸：200px × 60px）
   - 更新背景图像：点击“更新背景图像”以上传自定义背景图像（推荐尺寸：1920px × 1080px）
5. 点击“保存”以应用更改

### 自定义门户页面

要自定义门户页面：

1. 导航到平台管理视角
2. 从导航菜单中选择“系统设置”
3. 点击“门户”
4. 自定义以下元素：
   - 主标题：输入要显示的主标题文本
   - 副标题：输入要显示的副标题文本
   - 顶部图片：上传将在门户页面顶部显示的图像
   - 底部图片：上传将在门户页面底部显示的图像
5. 点击“保存”以应用更改

### 自定义平台品牌

要自定义平台品牌元素：

1. 导航到平台管理视角
2. 从导航菜单中选择“系统设置”
3. 点击“Logo”
4. 自定义以下元素：
   - 平台 Logo：上传自定义平台 logo（推荐尺寸：180px × 40px）
   - 平台名称：输入要显示的平台名称文本
   - Favicon：上传自定义 favicon（推荐尺寸：32px × 32px）
5. 点击“保存”以应用更改

### 自定义视角名称

要自定义视角名称：

1. 导航到平台管理视角
2. 从导航菜单中选择“系统设置”
3. 点击“平台参数”
4. 在“产品配置”下，找到您要重命名的视角
5. 使用您首选的名称更新“名称”字段
6. 点击“保存”以应用更改

重命名的视角将在视角切换器和门户页面中以新名称显示。

## 集成外部产品

您可以将外部产品 URL 集成到平台中，这些外部产品将在门户页面和视角切换器中显示为视角卡片。

要集成外部产品：

1. 导航到平台管理视角
2. 从导航菜单中选择“系统设置”
3. 点击“平台参数”
4. 在“产品配置”下，点击“集成产品”
5. 提供以下信息：
   - 名称：要显示的产品名称
   - 图标：上传该产品的图标
   - URL：输入外部产品的 URL
   - 描述：提供该产品的简要描述
6. 点击“提交”以完成集成

一旦集成，外部产品将在门户页面和视角切换器中作为卡片显示。点击卡片或选择视角将引导用户导航到指定的 URL。

## 自定义最佳实践

- 在所有自定义元素中保持一致的品牌形象
- 使用适当尺寸和文件大小的图像文件
- 确保自定义文本清晰、简明，且适合所有用户
- 在不同浏览器和屏幕尺寸中测试自定义
- 选择颜色和对比度时，考虑无障碍指导方针
