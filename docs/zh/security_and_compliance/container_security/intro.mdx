# 介绍

## 运行时安全介绍

运行时安全是一个专注于保护容器化应用在运行过程中的安全防护模块。它通过持续监控和保护容器环境，确保应用在运行时的安全性，包括漏洞检测、合规管理、网络安全、配置管理等多个维度的防护能力。平台提供了 NeuVector 和 StackRox（推荐）两种成熟的运行时安全引擎，为用户提供全方位的容器安全保障。

## 运行时安全优势

- **全生命周期防护**
  
  从 CI/CD 阶段到运行时，提供完整的容器安全生命周期保护，包括镜像扫描、漏洞检测、运行时防护等

- **多维度安全防护**
  
  集成漏洞管理、合规检查、网络安全、配置管理等多个安全维度，提供全面的安全防护能力

- **自动化安全运维**
  
  支持自动化的安全策略执行、异常检测和响应，降低安全运维成本

- **原生集成能力**
  
  与 Kubernetes 深度集成，支持原生网络策略和安全控制，保证安全策略的一致性和可移植性

- **可视化监控**
  
  提供直观的安全态势可视化，包括网络流量、系统事件、风险评估等多维度展示

## 运行时安全应用场景

- **金融行业安全合规**
  
  满足金融机构对容器环境的严格安全要求，支持 PCI、HIPAA 等多种合规标准

- **多云环境安全管理**
  
  通过统一的安全策略和管理平台，实现跨云环境的一致性安全管理

- **微服务应用防护**
  
  为微服务架构提供细粒度的安全控制和防护，确保服务间通信的安全性

- **DevSecOps 实践**
  
  将安全集成到 CI/CD 流程中，实现开发、运维和安全的深度融合

## 运行时安全使用限制

- 运行时安全引擎的部署需要额外的计算资源支持
- 部分高级功能可能需要额外的许可证
- 安全策略的制定需要对容器技术和业务场景有深入理解
- 某些深度防护功能可能会对应用性能产生轻微影响
