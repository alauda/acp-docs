# 介绍

## 审计功能介绍

审计功能是一个面向 Kubernetes 集群的安全审计模块，为平台提供全方位的操作记录追踪能力。通过审计功能，您可以全面掌握集群内的各类变更事件，实现问题快速定位与安全事件追溯，有效提升集群的安全性与可维护性。

## 审计功能优势

审计功能具有以下核心优势：

- 全面的操作记录
  
  支持记录用户操作和系统操作，覆盖集群内所有关键事件，确保操作可追溯。

- 精细的事件追踪
  
  提供详细的事件信息，包括操作者身份、操作类型、资源变更内容等，便于问题定位。

- 灵活的查询能力
  
  支持多维度筛选和时间范围查询，快速定位目标事件。

- 完整的数据导出
  
  支持将审计记录导出为 CSV 格式，便于后续分析和存档。

## 审计功能应用场景

审计功能的主要应用场景如下：

- 安全事件追踪
  
  当发生安全事件时，可通过审计记录追踪操作来源，确定事件影响范围。

- 问题诊断分析
  
  在集群或应用出现异常时，可通过审计记录快速定位问题原因。

- 合规性审查
  
  支持对集群操作进行合规性审查，满足企业安全管控要求。
