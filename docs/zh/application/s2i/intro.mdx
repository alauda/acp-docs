# 介绍

## 功能介绍

Source to Image（S2I）是一种用于从源代码构建可重复生成的容器镜像的工具和工作流。它通过将应用程序的源代码注入到一个预定义的构建器镜像中，并自动完成编译、打包等步骤，最终生成一个可直接运行的容器镜像，使得开发人员可以更专属于业务代码开发，无需关注容器化实现细节。
## 核心功能

打通从代码到应用的全链路流程，实现多语言构建、可视化发布，帮助开发者快速将代码转化为可运行的容器镜像，并可无缝集成到全栈云原生开放平台，最终实现业务应用的快速发布。

- 多语言构建： 支持 Java、Go、Python、Node.js 等多种编程语言，满足多样化的开发需求。
- 可视化操作： 提供直观的界面，让您轻松创建、配置和管理构建任务，无需深入了解底层技术。
- 全生命周期管理： 覆盖从代码提交到应用部署的整个过程，实现自动化构建、应用部署及运维管理。
- 深度集成： 与 Container Platform 产品深度集成，提供无缝的开发体验。
- 高度可扩展： 支持自定义插件和扩展，满足您的个性化需求。

## 核心优势

- 加速开发： 简化构建流程，加速应用交付。
- 增强灵活性： 支持多种编程语言构建。
- 提高效率： 自动化构建和部署，减少人工干预。
- 增强可靠性： 提供详细的构建日志和可视化监控，方便问题排查。

## 应用场景

Source to Image 的主要应用场景如下：

- Web 应用开发

  支持多种编程语言（如 Java、Go、Python、Node.js 等）等多种语言，通过平台应用管理能力，仅需输入代码仓库地址即可实现业务应用的快速构建和部署

- 持续集成/持续部署（CI/CD）

  通过平台应用管理能力，可以与 DevOps 无缝集成，实现自动化的镜像构建和部署流程。


## 使用限制

> 当前版本仅支持 Java、Go、Python、Node.js 语言。
