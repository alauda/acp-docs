# 通过代码创建原生应用

## 功能简介

通过代码创建原生应用是基于 S2I（Source to Image）技术实现。S2I 是一套自动化的镜像构建框架，能够将源代码直接转换为可运行的容器镜像，这种方式使得应用构建过程更加标准化、自动化，让开发人员可以专注于业务代码开发，无需关注容器化实现细节。


## 主要功能
- 输入代码仓库地址，即可将源代码构建成容器镜像，并发布成应用。
- 源代码更新后，可通过可视化界面，一键触发“重新构建”实现应用版本的更新升级。
