# 版本对比与回滚


平台不仅记录了通过界面操作的应用变更，还会记录调用 API 等非界面操作的变更。每次变更都对应一个版本，方便您进行版本对比或回滚。



## 版本对比


1. 在左侧导航栏中，单击 **应用管理** > **OAM 应用**。

3. 单击 ***应用名称***。
4. 单击 ***组件名称***。

3. 在 **变更记录** 页签中，单击 ***版本号***。

4. 单击 **与当前版本比较**，查看变更。



## 版本回滚


1. 在左侧导航栏中，单击 **应用管理** > **OAM 应用**。

3. 单击 ***应用名称***。
4. 单击 ***组件名称***。

3. 在 **变更记录** 页签中，单击 ***版本号***。

4. 单击 **回滚至该版本**，并确定。
