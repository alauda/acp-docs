## 介绍

### 适用场景

您是一位刚接触容器平台的新用户，希望能够创建应用。

### 预计阅读时间

预计完成本文档的阅读和操作时间，例如：5-10 分钟。

本文档不会对配置过程中的参数进行详细说明和配置，仅作为快速入门参考，具体说明及配置步骤请参考相关文档。
