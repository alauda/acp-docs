# 下载安装包

在安装之前，您需要下载安装包和签名文件，并使用 GPG 工具验证其完整性。

<Steps>
  ### 下载安装包

  登录 **客户门户** 下载安装包和签名文件。如果您尚未注册帐户，请联系技术支持。

  ### 安装 GPG 工具

  根据您的操作系统安装 GPG 工具（请参阅下面的命令）。

  ```bash
  # Ubuntu
  sudo apt-get install gnupg
  # CentOS
  yum install gnupg
  # Mac
  brew install gpg
  ```

  ### 下载公钥

  ```bash
  curl -O https://www.alauda.cn/download/verify-key.pub
  md5sum verify-key.pub
  ```

  输出的 MD5 值应为：`2eaddfab97d2951a8915f327acb53562`。

  ### 验证公钥指纹

  ```bash
  gpg --import verify-key.pub
  gpg --fingerprint BB097AE6
  ```

  指纹应完全匹配：

  ```
  09EE E7B9 A30C F4B3 5E31 A91B 2704 1C16 BB09 7AE6
  ```

  ### 验证安装包

  ```bash
  gpg --verify <签名文件> <安装包>
  ```

  例如：

  ```bash
  gpg --verify acp-4.0.0.tgz.sig acp-4.0.0.tgz
  ```

  正常输出应显示“Good signature”信息。
</Steps>
