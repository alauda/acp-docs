# 下载安装包 \{#download_installation_package}

在安装前，您需要下载安装包并解压。如果您需要使用扩展包，请在开始安装前按核心包、扩展包的顺序将他们解压到同一目录下。

<Steps>

### 下载安装包

请登录 **客户门户** 下载安装包和签名文件。如果您尚未注册账号，请联系技术支持。

安装包分为两类：
- **核心包**：所有安装都必须下载
- **扩展包**：可选，仅在需要额外功能时下载

### 核心包说明

请下载核心包文件 installer-core-v4.x.y-zzzz.tar，其中 "zzzz" 代表所选架构（如 x86、arm 或 hybrid，下文同理）。  
使用以下 tar 命令解压核心包：  
```shell
tar xvf installer-core-v4.x.y-zzzz.tar
```

### 扩展包说明

请下载扩展包文件 installer-extensions-v4.x.y-zzzz.tar，其中 "zzzz" 必须与核心包保持一致。扩展包中包含核心包未集成的插件组件。  
如果您下载了扩展包，请在安装前将其解压到与核心包相同的目录下：  
```bash
tar xvf installer-extensions-v4.x.y-zzzz.tar -C <the core package extraction directory>
```

示例：  
假设您已将核心包解压到 /root 目录下，解压后会在 /root 下生成 installer 目录。  
此时，请在 installer-extensions-v4.x.y-zzzz.tar 所在目录下执行如下命令，将扩展包解压到 /root 目录下：`tar xvf installer-extensions-v4.x.y-zzzz.tar -C /root/`  

</Steps>

安装指南将详细介绍核心包的安装流程，扩展包的使用和配置请参考扩展包内的相关文档。
