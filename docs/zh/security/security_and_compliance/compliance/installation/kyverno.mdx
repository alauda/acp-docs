# 灵雀云与 Kyverno 的合规性

:::tip

灵雀云与 Kyverno 的合规性是一个平台服务，集成了 Kyverno 用于管理 ACP 平台上的合规性政策。

:::

## 安装步骤

1. 导航到 **平台管理**
2. 在左侧导航栏，点击 **市场** > **集群插件**
3. 搜索 **灵雀云与 Kyverno 的合规性** 并点击查看其详情
4. 点击 **安装** 部署插件

## 卸载步骤

1. 按照安装过程中的步骤 1-3 定位插件
2. 点击 **卸载** 移除插件
