# 安装

ACP Telemetry 是一个平台监测服务，负责收集集群的监控指标数据，并将其上传到 Alauda 进行监控和分析。

## 先决条件

在安装之前，请确保：

- ACP 平台具有有效的许可证
- global 集群可以访问互联网

## 安装步骤

1. 转到 **平台管理**
2. 在左侧导航栏中，点击 **市场** > **集群插件**
3. 在顶部导航栏中选择 **global** 集群
4. 搜索 **ACP Telemetry** 并点击以查看其详细信息
5. 点击 **安装** 来部署该插件

## 启用在线操作

1. 在左侧导航栏中，点击 **系统设置** > **平台维护**
2. 点击 **在线操作** 右侧的 **启用** 按钮

## 卸载步骤

1. 按照安装过程中的步骤 1-4 找到该插件
2. 点击 **卸载** 来移除该插件
