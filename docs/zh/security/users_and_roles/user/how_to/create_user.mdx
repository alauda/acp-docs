# 创建用户

 :::tip

具备平台管理员角色的用户可以通过平台界面创建本地用户并为其分配角色。

:::

## 步骤

1. 在左侧导航栏中，点击 **用户** > **用户管理**
2. 点击 **创建用户**
3. 配置以下参数：

| 参数                 | 描述                                                                                                                                                                                                                                                                                                                                                     |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **密码类型**         | 选择密码生成方式：<br /><br />**随机**：系统生成一个安全的随机密码<br />**自定义**：用户手动输入密码                                                                                                                                                                                                                                                    |
| **密码**             | 根据选择的类型输入或生成一个密码。<br /><br />密码要求：<br />- 长度：8-32 个字符<br />- 必须包含字母和数字<br />- 必须包含特殊字符 (`~!@#$%^&*() -_=+?`)<br /><br />密码字段功能：<br />- 点击眼睛图标可显示/隐藏密码<br />- 点击复制图标可复制密码                                                                                            |
| **邮箱**             | 用户的电子邮件地址：<br />- 必须唯一<br />- 可用作登录用户名<br />- 与用户的姓名关联                                                                                                                                                                                                                                                                           |
| **有效期**           | 设置用户账户的有效期：<br /><br />选项：<br />- **永久**：无限制<br />- **自定义**：使用时间范围下拉菜单设置开始和结束时间                                                                                                                                                                                                                                   |
| **角色**             | 为用户分配一个或多个角色                                                                                                                                                                                                                                                                                                                                |
| **继续创建**         | 切换开关以控制创建后的行为：<br />- **开**：重定向到创建新用户页面<br />- **关**：显示用户详细信息页面                                                                                                                                                                                                                                                       |

4. 点击 **创建**

:::note

用户创建成功后：

- 如果启用了“继续创建”，您将被重定向到创建另一个用户
- 如果禁用，您将看到所创建用户的详细信息页面

:::
