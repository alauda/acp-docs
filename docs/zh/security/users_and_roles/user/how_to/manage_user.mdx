# 用户管理

:::tip

该平台提供灵活的用户管理功能，支持个人用户管理和批量操作，以提高特定场景（如现场或离线团队）的效率。

:::

:::warning

重要限制：

- 系统生成的账户无法管理（平台管理员角色，本地来源）
- 当前登录的用户不能管理自己的账户
- 如需修改个人账户信息（显示名称、密码），请使用个人信息页面

:::

## 重置本地用户密码

具有平台管理权限的用户可以重置其他本地用户的密码。

### 步骤

1. 在左侧导航栏，点击 **用户** > **用户管理**
2. 点击目标用户记录旁的图标
3. 点击 **重置密码**
4. 在对话框中选择密码类型：
   - **随机**：系统生成一个安全的随机密码
   - **自定义**：手动输入新密码

:::note

密码要求：

- 长度：8-32个字符
- 必须包含字母和数字
- 必须包含特殊字符（`~!@#$%^&*() -_=+?`）

密码字段功能：

- 点击眼睛图标以显示/隐藏密码
- 点击复制图标以复制密码

:::

5. 点击 **重置**

## 更新用户过期日期

您可以更新处于 **正常**、**禁用** 或 **锁定** 状态的用户的过期日期。超过过期日期的用户将自动禁用。

### 步骤

1. 在左侧导航栏，点击 **用户** > **用户管理**
2. 点击目标用户旁的 **更新过期日期**
3. 在对话框中选择过期日期选项：
   - **永久**：无时间限制
   - **自定义**：使用时间范围下拉菜单设置开始和结束时间
4. 点击 **更新**

## 激活用户

您可以激活处于 **禁用** 或 **锁定** 状态的用户。

:::note

激活行为：

- 如果用户在过期日期内：过期日期保持不变
- 如果用户已经过期：过期日期变为 **永久**

:::

### 步骤

1. 在左侧导航栏，点击 **用户** > **用户管理**
2. 点击目标用户旁的 **激活**
3. 在确认对话框中点击 **激活**
4. 用户状态将变为 **正常**

## 禁用用户

您可以在过期日期内禁用处于 **正常** 或 **锁定** 状态的用户。禁用的用户无法登录，但可以重新激活。

### 步骤

1. 在左侧导航栏，点击 **用户** > **用户管理**
2. 点击目标用户旁的图标
3. 点击 **禁用** 并确认

## 将用户添加到本地用户组

您可以将 **来源** 为 **本地** 或 `LDAP` 的用户添加到一个或多个本地用户组。

:::warning

组角色行为：

- 用户自动继承其组的角色
- 组角色仅在组的详细信息页面（配置角色选项卡）可见
- 个人用户角色列表仅显示直接分配的角色

:::

### 步骤

1. 在左侧导航栏，点击 **用户** > **用户管理**
2. 点击目标用户旁的图标
3. 点击 **添加到用户组**
4. 选择一个或多个本地用户组
5. 点击 **添加**

## 删除用户

平台管理员可以删除任何用户，排除当前登录的账户，包括：

- IDP 配置的用户
- 来源为 `-` 的用户
- 本地用户

### 步骤

1. 在左侧导航栏，点击 **用户** > **用户管理**
2. 点击目标用户旁的图标
3. 点击 **删除**
4. 点击 **确认**

## 批量操作

您可以对以下进行批量操作：

- 更新有效期
- 激活用户
- 禁用用户
- 删除用户

### 步骤

1. 在左侧导航栏，点击 **用户** > **用户管理**
2. 使用复选框选择一个或多个用户
3. 点击 **批量操作** 并选择一个动作：
   - **更新有效性**
   - **激活**
   - **禁用**
   - **删除**

:::note

批量操作详情：

- **更新有效性**：设置为永久或自定义时间范围
- **激活**：在对话框中确认激活
- **禁用**：在对话框中确认禁用
- **删除**：输入当前账户密码并确认

:::
