# 创建用户

具有平台管理员角色的用户可以通过平台界面创建本地用户并为其分配角色。

## 步骤

1. 在左侧导航栏中，单击 **用户** > **用户管理**
2. 单击 **创建用户**
3. 配置以下参数：

| 参数                     | 描述                                                                                                                                                                                                                                                                                                                                                       |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **密码类型**             | 选择一种密码生成方法：<br /><br />**随机**：系统生成一个安全的随机密码<br />**自定义**：用户手动输入密码                                                                                                                                                                                                                                              |
| **密码**                | 根据所选类型输入或生成密码。<br /><br />密码要求：<br />- 长度：8-32 个字符<br />- 必须包含字母和数字<br />- 必须包含特殊字符 (`~!@#$%^&*() -_=+?`)<br /><br />密码字段功能：<br />- 单击眼睛图标以显示/隐藏密码<br />- 单击复制图标以复制密码 |
| **邮箱**                | 用户的电子邮件地址：<br />- 必须是唯一的<br />- 可以用作登录用户名<br />- 与用户的名称关联                                                                                                                                                                                                                                                             |
| **有效期**              | 设置用户帐户的有效期：<br /><br />选项：<br />- **永久**：无限制<br />- **自定义**：使用时间范围下拉菜单设置开始和结束时间                                                                                                                                                                                                                                     |
| **角色**                | 为用户分配一个或多个角色                                                                                                                                                                                                                                                                                                                                   |
| **继续创建**            | 切换开关以控制创建后行为：<br />- **开启**：重定向到新用户创建页面<br />- **关闭**：显示用户详细信息页面                                                                                                                                                                                                                                             |

4. 单击 **创建**

:::note

用户创建成功后：

- 如果“继续创建”启用，将重定向到创建另一个用户的页面
- 如果禁用，则会显示创建用户的详细信息页面

:::
