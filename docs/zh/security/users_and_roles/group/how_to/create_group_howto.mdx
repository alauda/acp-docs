# 创建本地用户组

:::tip

本地用户组允许你为来自任何来源的多个用户实施基于角色的访问控制。

:::

## 创建用户组

### 步骤

1. 在左侧边栏，点击 **用户** > **用户组管理**
2. 点击 **创建用户组**
3. 输入以下信息：
   - **名称**：用户组的名称
   - **描述**：该组目的的描述
4. 点击 **创建**

## 管理用户组

你可以通过点击列表页面上的图标或在详情页面右上角点击 **操作** 来管理用户组。

| 操作                          | 描述                                                                                                                                                                                               |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **更新用户组**                | 根据组来源更新组信息：<br />- 对于 **来源** 为 `Local` 的组：可以更新名称和描述 <br />- 对于 **来源** 为 `IDP name` 的组：只能更新描述                                                               |
| **删除本地用户组**            | 删除 **来源** 为 `Local` 的用户组                                                                                                                                                                  |

:::warning

当你删除一个组时：

- 所有组成员将被移除
- 指派给该组的所有角色将被移除
- 此操作无法撤消

:::
