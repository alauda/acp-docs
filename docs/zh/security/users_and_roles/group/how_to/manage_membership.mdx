# 管理本地用户组成员

:::tip

只有具有平台管理权限的用户才能管理本地用户组成员。

:::

## 先决条件

:::warning

在管理组成员之前，请注意以下限制：

- 只有具有平台管理权限的用户才能管理组及其成员
- 系统帐户和当前登录的帐户无法管理（不能导入到组或从组中删除）
- 每个本地用户组最多可以有 5000 名成员
- 当一个组达到 5000 名成员的限制时，不允许再进行导入

:::

## 导入成员

您可以从平台导入用户到本地用户组，以实现统一的权限管理。

:::tip

导入到组中的用户将自动继承分配给该组的所有操作权限。

:::

### 步骤

1. 在左侧导航栏中，点击 **用户** > **用户组管理**
2. 点击您要添加成员的本地用户组的名称
3. 在 **组成员管理** 标签下，点击 **导入成员**
4. 通过勾选其用户名/显示名称旁边的框选择一个或多个用户
5. 点击 **导入**

:::note

- 您只能选择当前不是该组成员的用户
- 使用 **全部导入** 按钮一次性导入列表中的所有用户

:::

## 移除成员

当您将用户从组中移除时，通过该组授予该用户的所有操作权限将自动被撤销。

### 步骤

1. 在左侧导航栏中，点击 **用户** > **用户组管理**
2. 点击您要移除成员的本地用户组的名称
3. 在 **组成员管理** 标签下，您可以通过两种方式移除成员：
   - 点击成员名称旁边的 **移除** 并确认
   - 使用复选框选择一个或多个成员，然后点击 **批量移除** 并确认
