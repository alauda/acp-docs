# 管理用户组角色

:::tip

拥有平台管理权限的用户可以管理本地用户组和 IDP 同步用户组的角色。

:::

## 向组中添加角色

### 步骤

1. 在左侧导航栏中，点击 **用户** > **用户组管理**
2. 点击目标用户组的名称
3. 在 **配置角色** 标签下，点击 **添加角色**
4. 点击以添加角色

:::note

角色分配规则：

- 可以向一个组添加多个角色
- 每个角色只能添加一次到同一组

:::

5. 从下拉菜单中选择角色名称
6. 选择角色的权限范围（集群、项目或命名空间）
7. 点击 **添加**

## 从组中移除角色

:::warning

当您从组中移除角色时：

- 该角色授予组成员的所有权限将被撤销
- 此操作无法撤销

:::

### 步骤

1. 在左侧导航栏中，点击 **用户** > **用户组管理**
2. 点击目标用户组的名称
3. 在 **配置角色** 标签下，点击角色旁的 **移除**
4. 点击 **确认** 以移除该角色
