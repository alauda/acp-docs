# 关于灵雀云服务网格

服务网格是一个基于 Istio 的 PaaS 平台，为应用程序和微服务提供云原生解决方案。

该平台提供一键式基础设施部署和升级功能，可视化服务治理，高效的应用性能管理，以及服务网关管理的高可用性和高性能。它为正在进行数字化转型的企业提供全面、稳定、可靠和开放的微服务解决方案，帮助提高服务治理的效率，并减少微服务框架的维护成本，同时降低开发人员对框架的依赖，使企业能够专注于业务发展，并持续增强其核心竞争力。
