# Kyverno

Alauda Compliance with Kyverno is a service provided by ACP platform that integrates Kyverno to manage compliance policies.


## Installing

1. Enter **Platform Management**.

2. In the left navigation bar, click **Marketplace** > **Cluster Plugins**.

3. Search for **Alauda Compliance with Kyverno**, click it to enter the **Alauda Compliance with Kyverno** details page.

4. Click **Install** to install the plugin.

## Uninstalling

1. Refer to the first 4 steps of the installation method to search for plugin.

2. Click **Uninstall** to uninstall the plugin.
