# Installation

ACP API Refiner is a service provided by ACP platform that filters K8S API return data. It supports filtering K8S API return data by project, cluster, and namespace. It also supports excluding, including, and de-sensitizing the fields of the returned API content.

## Installing ACP API Refiner

1. Enter **Platform Management**.

2. In the left navigation bar, click **Marketplace** > **Cluster Plugins**.

3. Select the **global** cluster in the top navigation bar.

4. Search for **ACP API Refiner**, click it to enter the **ACP API Refiner** details page.

5. Click **Install** to install the plugin.

## Uninstalling ACP API Refiner

1. Refer to the first 4 steps of the installation method to search for plugin.

2. Click **Uninstall** to uninstall the plugin.

## Default Configuration Description

### Default Filtered Resource List

| Name | API Version |
|---------|----------|
| namespaces | v1 |
| projects | auth.alauda.io/v1 |
| clustermodules | cluster.alauda.io/v1alpha2 |
| clusters | clusterregistry.k8s.io/v1alpha1 |

### API Content Filtering

Defaultly, the annotation **metadata.annotations.cpaas.io/creator** is de-sensitized.
