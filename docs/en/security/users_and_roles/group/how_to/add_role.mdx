# Add a role

## Introduction

Support users with platform management permissions to manage the roles of user groups (local user groups and user groups synchronized from IDP).

### Procedure

1. In the left navigation bar, click **Users** > **User Group Management**.

2. Click the ***User Group Name*** to which you want to add a role.

3. On the **Configure Role** tab, click **Add Role**.

4. Click ![](/en/img/00add.png) to add a role for the user.



    **Note**: You can repeat this step to add multiple roles to the group.

5. Select the ***Role Name***.

    **Note**: The same role can only be added once to the same group.

6. After selecting the scope where the role permissions take effect (cluster, project, or namespace), click **Add**.

## Unbind

After removing a role from a user group, the permissions granted to the members of the group will be revoked.

### Procedure

1. In the left navigation bar, click **Users** > **User Group Management**.

2. Click the ***User Group Name*** from which you want to remove a role.

3. On the **Configure Role** tab, click **Remove** next to the role you want to remove, and confirm.
