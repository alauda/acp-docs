# Installation

ACP Telemetry is a service provided by ACP platform that collects telemetry data from clusters. Implemented the online O&M function of the product. Collected system metrics and uploaded them to alauda cloud for information display on alauda cloud.


## Prerequisites

1. ACP platform has already imported a valid license.

2. The global cluster of the platform can access the public network.


## Installing

1. Enter **Platform Management**.

2. In the left navigation bar, click **Marketplace** > **Cluster Plugins**.

3. Select the **global** cluster in the top navigation bar.

4. Search for **ACP Telemetry**, click it to enter the **ACP Telemetry** details page.

5. Click **Install** to install the plugin.

## Enable Online Operations

1. In the left navigation bar, click **System Settings** > **Platform Maintenace**.

2. Click the **On** button of **Online Operations**.

## Uninstalling

1. Refer to the first 4 steps of the installation method to search for plugin.

2. Click **Uninstall** to uninstall the plugin.
