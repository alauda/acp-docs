# Installing Alauda Container Platform Project Application Essential

## Prerequisites

1. **Download** the **Alauda Container Platform Project Application Essential** cluster plugin installation package corresponding to your platform architecture.
2. **Upload** the installation package using the Upload Packages mechanism.
3. **Install** the installation package to the `global` cluster using the cluster plugins mechanism.

:::info
Upload Packages:
**Administrator** > **Marketplace** > **Upload Packages** page.
Click **Help Document** on the right to get instructions on how to publish the cluster plugin to `global` cluster. For more details, please refer to [CLI](../cli_tools/index.mdx).
:::

## Installing Alauda Container Platform Project Application Essential cluster plugin

### Constraints and Limitations

- Only supports installation in the `global` cluster.

### Procedure

1. Login, go to the **Administrator** page.
2. Click **Marketplace** > **Cluster Plugins** to enter the **Cluster Plugins** list page.
3. Find the **Alauda Container Platform Project Application Essential** cluster plugin, click **Install**, and navigate to the **Install Alauda Container Platform Project Application Essential Plugin** page.
4. It is recommended to use the suggested default configuration; simply click **Install** to complete the **Alauda Container Platform Project Application Essential** cluster plugin installation.

This plugin does not require any specific configuration parameters during installation via the UI. It is recommended to use the default settings.
