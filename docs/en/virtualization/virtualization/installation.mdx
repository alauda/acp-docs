:::tip

If the module needs to be deployed independently, deployment documentation must be provided; otherwise, it can be omitted.

The deployment documentation should include the deployment process, deployment configuration, uninstallation process, etc. If the deployment documentation is extensive, it can be converted into a directory format and divided into multiple documents for introduction.

You can access [Deployment](https://product-doc-guide.alauda.cn/template/installation/example.html) to view examples of the corresponding documentation.

:::
