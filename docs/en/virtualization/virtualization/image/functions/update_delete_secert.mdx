# Update/Delete Image Credentials

> **New console**: image-pull credentials are Kubernetes `Secret`s referenced by a **[Bootable Volume](../../virtual_machine/bootable_volume.mdx)**'s registry or HTTP source (`spec.source.*.secretRef`). The steps below describe the classic console **Image Credentials** tab.

1. Go to **Administrator**.

2. In the left navigation bar, click **Virtualization Management** > **Virtual Machine Images**.

3. In the **Image Credentials** tab, click ⋮ > **Update/Delete**.

4. After confirmation, click **Update/Delete**.
