:::tip

If the module needs to be deployed separately, deployment documentation must be provided; otherwise, it can be omitted.

The deployment documentation should include the deployment process, deployment configuration, uninstallation process, etc. If the content of the deployment documentation is extensive, it can be transformed into a directory, dividing it into multiple documents for explanation.

You can visit [Deployment](https://product-doc-guide.alauda.cn/template/installation/example.html) to see examples of the corresponding documentation.

:::
