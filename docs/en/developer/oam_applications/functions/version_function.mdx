# Version Comparison and Rollback

The platform records changes made through the interface as well as those made through non-interface operations such as API calls. Each change corresponds to a version, making it easy for you to perform version comparisons or rollbacks.

## Version Comparison

1. Enter **Container Platform**.

2. In the left navigation bar, click **Application Management** > **OAM Application**.

3. Click ***Application Name***.

4. Click ***Component Name***.

5. In the **Change Log** tab, click ***Version Number***.

6. Click **Compare with Current Version** to view changes.

## Version Rollback

1. Enter **Container Platform**.

2. In the left navigation bar, click **Application Management** > **OAM Application**.

3. Click ***Application Name***.

4. Click ***Component Name***.

5. In the **Change Log** tab, click ***Version Number***.

6. Click **Rollback to this Version** and confirm.
