# Alauda Streaming Service for Kafka

Alauda Streaming Service for Kafka is a distributed stream processing platform based on Kafka, running on Kubernetes. As a distributed messaging queue, It efficiently transmits and stores large amounts of data across different systems. It supports multiple data formats and can handle real-time data streams, making it widely used in various fields such as log collection, event-driven architectures, real-time analytics, and data integration.
