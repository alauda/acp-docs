# Alauda Streaming Service for RabbitMQ

Alauda Streaming Service for RabbitMQ is an open-source message broker that implements the Advanced Message Queuing Protocol (AMQP), running on Kubernetes. It acts as a communication bridge between applications, allowing different systems to asynchronously send and receive messages. It provides a robust foundation for building decoupled, scalable, and resilient distributed systems.
