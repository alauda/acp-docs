:::warning Deprecation Notice
MySQL-PXC is deprecated and no longer managed by the MySQL operator. Existing PXC clusters will continue to run but will not receive operator updates. Please migrate to [MySQL-MGR](./mgr.mdx) for continued support and new deployments. See <ExternalSiteLink name="mysql-mgr" href="/how_to/35-migrate-from-pxc" children="Migrate MySQL-PXC to MySQL-MGR" /> for detailed migration instructions.
:::

Alauda Database Service for MySQL-PXC is a high availability database solution using synchronous replication based on Percona XtraDB Cluster, running on Kubernetes. It provides the capability for automated deployment and management of objects within the Percona XtraDB Cluster, and can also be used to start new Percona XtraDB Cluster instances or scale existing instance.
