title: About Alauda support for PostgreSQL
---

Alauda support for PostgreSQL is a cloud-native PostgreSQL management solution built on the Zalando Postgres Operator. It achieves full lifecycle management of PostgreSQL clusters through Custom Resource Definitions (CRD), supporting core operational capabilities such as automatic failover, rolling upgrades, and backup restoration, enabling enterprises to run production-grade PostgreSQL databases effortlessly on Kubernetes.
