# ServiceAccount [v1]

<OpenAPIPath
  path={[
    '/api/v1/namespaces/{namespace}/serviceaccounts',

  ]}

## API usage

### List ServiceAccounts

```http
GET /kubernetes/{cluster}/api/v1/namespaces/{namespace}/serviceaccounts
```

Common query parameters:

| Parameter | Type | Description |
| --- | --- | --- |
| `labelSelector` | `string` | Restricts returned resources by labels. |
| `fieldSelector` | `string` | Restricts returned resources by fields. |
| `limit` | `integer` | Limits the number of returned records. |
| `continue` | `string` | Continue token for paginated queries. |

### Get a specific ServiceAccount

```http
GET /kubernetes/{cluster}/api/v1/namespaces/{namespace}/serviceaccounts/{name}
```

### Create a ServiceAccount

```http
POST /kubernetes/{cluster}/api/v1/namespaces/{namespace}/serviceaccounts
```

### Request examples

List ServiceAccounts:

```bash
curl -H "Authorization: Bearer ${TOKEN}" \
  "${APISERVER}/kubernetes/${CLUSTER}/api/v1/namespaces/${NAMESPACE}/serviceaccounts?limit=20"
```

Get a specific ServiceAccount:

```bash
curl -H "Authorization: Bearer ${TOKEN}" \
  "${APISERVER}/kubernetes/${CLUSTER}/api/v1/namespaces/${NAMESPACE}/serviceaccounts/${SA_NAME}"
```

Create a ServiceAccount:

```bash
curl -X POST -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  "${APISERVER}/kubernetes/${CLUSTER}/api/v1/namespaces/${NAMESPACE}/serviceaccounts" \
  -d '{
    "apiVersion": "v1",
    "kind": "ServiceAccount",
    "metadata": {
      "name": "demo-sa"
    }
  }'
```

### Response examples

ServiceAccount list response snippet:

```json
{
  "kind": "ServiceAccountList",
  "items": [
    {
      "metadata": {
        "name": "default"
      }
    },
    {
      "metadata": {
        "name": "demo-sa"
      }
    }
  ]
}
```

ServiceAccount response snippet:

```json
{
  "kind": "ServiceAccount",
  "metadata": {
    "name": "demo-sa",
    "namespace": "default"
  }
}
```
