# Cluster Lifecycle APIs

These APIs expose the ACP aggregated Kubernetes API for cluster lifecycle operations. They cover the primary resources used to create, inspect, update, and delete clusters, cluster credentials, and machines.

This section documents only the main resources in `platform.tkestack.io/v1`. It does not cover subresources such as `clusters/exec` and `clusters/drain` in this first round.
