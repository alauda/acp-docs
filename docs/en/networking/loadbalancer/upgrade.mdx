# Upgrade

The upgrade of all ALBs in a cluster is controlled by the alb-operator in that cluster. When the alb-operator is upgraded, all ALBs in the cluster will be automatically upgraded.
The upgrade of the alb-operator is controlled by the cluster plugin.
